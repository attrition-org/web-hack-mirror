Archiving blind carbon copies

    You can automatically send a copy of all incoming mail to any registered user or group. You can use this feature to save all mail sent to your Mail Server.

    IMPORTANT: Users receiving blind carbon copies should periodically log on to the Mail Server to delete blind carbon copy mail messages.

    To automatically send a blind carbon copy of all incoming mail to the server, follow these steps:
    1 Log on to AppleShare IP Remote Admin.
    2 Make sure the Mail Server is running.
    3

    Click the Mail server button.


       
    4 Click the Server Settings button.
    5 Click Mail In.
    6 To turn on copies, select the "Blind Carbon Copies (BCC)" checkbox.
    When this box is checked, a copy of all incoming messages is sent to a specified user or group without the sender knowing.
    7 Type the name of the user or group to whom you want to send a copy.
    Type the name exactly as it appears in the Users & Groups lists.
    8 Click the Save button.
