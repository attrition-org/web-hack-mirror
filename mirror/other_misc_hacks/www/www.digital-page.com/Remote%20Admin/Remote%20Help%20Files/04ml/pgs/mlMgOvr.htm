Managing mail servers

    If you've installed mail servers on multiple computers, you need to set up mail forwarding between servers. If you're connected to an external network, such as the Internet, you may need to set up the Mail Server to relay mail to a server with a connection to servers outside your domain.

    You can also create aliases for your Mail Server.

    Note: If you need help deciding how to organize your mail service, or if you are running the Mail Server on more than one computer, see How mail service works before continuing with this section.
