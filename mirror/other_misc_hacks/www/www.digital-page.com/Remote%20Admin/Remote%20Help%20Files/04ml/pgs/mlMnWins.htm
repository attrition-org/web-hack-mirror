Viewing activity, info, and log information

    You monitor Mail Server activity by viewing the information, activity, or log page.

    This section tells you about

      Mail Activity
    Lists message status, server use, names and addresses of users connected via IMAP or POP, and SMTP server mail connections and status
    Host List and Host Info
    The Mail Activity page lists all the domains to which mail has been sent or from which mail has been received, and the "approved hosts" entered by the mail administrator; the Host Info window shows DNS and mail statistics for each host
    Mail Server Log and Error Log
    Contain information about Mail Server activity and mail service errors

 
Mail Activity
 

    The Mail Server Activity page displays summary information about Mail Server activity, storage, hosts, connected users, and SMTP activity.

    To view the Mail Server Activity page, follow these steps:
    1 Log on to AppleShare IP Remote Admin.
    2 Make sure the Mail Server is running.
    3

    Click the Mail server button.


       
    4 View the top portion of the Mail Activity page to see the mail storage and message information.
    The Mail Storage Volume is the volume on which the AppleShare IP Mail Database file resides.
    The Mail Database grows to accommodate new messages but does not contract when messages are deleted; space that was used by deleted messages is made available for new messages.
    5 Scroll down the Mail Activity page to see the Connected Users section. It shows
    • the user's name
    • the user's IP address
    • the type of network connection being used to connect
    • the length of time a user has been connected
    • the number of users connected via POP and IMAP
    6 Scroll further down the Mail Activity page to see the SMTP Activity section. It shows
    • hosts connected to the SMTP server to send or receive mail
    • IP addresses for each host
    • the length of time hosts have been connected
    • the status of the connection
    • the number of incoming and outgoing messages

 
Host List and Host Info
 

    You can monitor each domain to which mail has been sent or from which mail has been received on the Host List. You can also view the MX List, network connection, and message statistics.

    To view the Host List and outgoing mail information by host, follow these steps:
    1 Log on to AppleShare IP Remote Admin.
    2 Make sure the Mail Server is running.
    3

    Click the Mail server button.


       
    The Host List on the Mail Activity page shows each host to which mail has been sent or from which mail has been received, and it shows information about mail messages for each host.
    4 To view DNS or network statistics for a host, click a host in the Host List.
    5 To view DNS information about this host, click DNS Information on the host's page.
    6 To view network or other statistics about this host, click Statistics on the host's page.

 
Mail Server Log and Error Log
 

    The Mail Server maintains two logs:
    • The Mail Server Log contains information about normal operations, such as when the Mail Server started and stopped.
    • The Error Log contains information about error conditions, such as failure to deliver mail because mail is not enabled for a user.

    To view the Mail Server Log or the Error Log, follow these steps:
    1 Log on to AppleShare IP Remote Admin.
    2

    Click the Mail server button.


       
    3 Click the Logs button.
    4 Click the log you want to view.
