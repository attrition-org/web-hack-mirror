AppleShare IP 6.1 system requirements

    • Any Power Macintosh computer or Apple Workgroup Server with a G3, 604e, 604, or 601 microprocessor, or a 6500 series computer with a 603e microprocessor
    • To use the AppleTalk multihoming feature, your computer must have a PCI expansion slot. Computers with a 601 processor and NuBus slots cannot use AppleTalk multihoming.
    • Mac OS system software 8.5 is recommended. Mac OS 8.1 or later is required for all features except TCP Filter; Mac OS 8.5 is required to use TCP Filter software.
    • 48 MB of RAM minimum (with virtual memory turned on), 64 MB or more recommended (with virtual memory turned off)
    • A minimum of 75 MB of disk space

    For more detailed information about system requirements, see the "Getting Started" manual or "Upgrading" document that came with your software.
