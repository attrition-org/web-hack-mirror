Mail Server specifications

    • Supported protocols and the corresponding port numbers they use: SMTP (port 25), POP3 (port 110), IMAP (port 143), APOP (port 110), PASS (port 106), Finger (port 79), and Notify Mail (port 79)
    • Maximum message size: 512 MB
    • Maximum recipients per message (before expansion of group names): 512
    • Supported character size: Eight-bit transparent for interoperability with two-byte character sets
    • Shared folder support conforms to Internet standard RFC 2086
    • Maximum number of transfers per POP connection: 2048
    • Maximum number of incoming SMTP connections: 1024
    • Maximum number of outgoing SMTP connections: 64
    • Maximum number of POP and IMAP connections: 2048
    • Maximum number of IMAP folders per user: 1000
    • Maximum number of messages per IMAP folder: 1000
    • Maximum folder name size: 128 bytes
    • Maximum IMAP folder path depth: 32 folders or 256 characters
    • Maximum number of unique messages stored on the Mail Server: approximately one million
    • Maximum size of the AppleShare IP Mail 6.0 Data file: 2 gigabytes

    Note: You may need to increase the amount of memory allocated to the Mail Server program (in the Extensions folder in the System Folder on your server's startup disk) in order to achieve these maximum settings.
