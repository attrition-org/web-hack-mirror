Logging on to AppleShare IP Remote Admin

    To use AppleShare IP Remote Admin, the Web & File Server must be running. Once the Web & File Server is running, you can log on to AppleShare IP Remote Admin by doing the following:
    1 On any computer connected to your AppleShare IP network, open one of the recommended Industry-standard Web browser programs.
    Apple recommends you use Internet Explorer 4.0 or later or Netscape Navigator 3.0 or later.
    2 Type your server's URL in the browser's Address field as follows:
    http://serverDNSname/webadmin/webadmin.acgi
    Where "serverDNSname" is the DNS name for your AppleShare IP Web & File Server.
    The AppleShare IP Remote Server Administration (AppleShare IP Remote Admin) log on screen appears.
    Note: To find your server's DNS name, open the AppleShare IP Manager program on your server and choose Show Network Info from the Manager menu. For help finding this information, see Visiting your Web site.
    3 Type your name and password in the Name and Password text fields, then click "Log on."
    The AppleShare IP Remote Server Administration home page appears and shows the current status of the AppleShare IP servers. Click one of the buttons to the right of the Status button to administer a different area.

    To return to the task you were in, click the Back arrow in your browser program.
