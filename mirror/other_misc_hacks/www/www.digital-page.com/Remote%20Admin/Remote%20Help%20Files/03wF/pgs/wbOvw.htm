Web service

    The AppleShare IP Web Server allows your computer to host a Web site and provide related Web services.

    This section tells you how to

      Access your Web site
    Turn Web service on and off, view your Web site, or control access to your Web folder
    Change Web settings
    Change the number of Web connections, turn on and off Web logging and directory listing, and change your Web folder, home page, and error messages
    Monitor Web Server activity
    Understand the log files that contain information about Web activity
    Upload files to the Web Server
    Copy files to the server using your Web browser and administer Web uploading
    Manage Web plug-ins
    Install and use Web plug-ins, change plug-in settings, view the plug-in log file

    For instructions on setting up multiple Web sites, go to your server and see AppleShare IP Help.
