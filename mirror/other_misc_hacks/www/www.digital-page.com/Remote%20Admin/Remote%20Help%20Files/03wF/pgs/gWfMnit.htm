Monitoring Web & File activity

    You can remotely monitor Web & File Server use by viewing the Status and Web & File Server Activity page.

    To display the Web & File Activity page, follow these steps:
    1 Log on to AppleShare IP Remote Admin.
      The Status page that appears shows which servers are starting up, running, shutting down, or not running.
    2 Click the File server button.
    On the File Activity page, you can see the Web & File Server activity, usage history, and a list of the connected users.
