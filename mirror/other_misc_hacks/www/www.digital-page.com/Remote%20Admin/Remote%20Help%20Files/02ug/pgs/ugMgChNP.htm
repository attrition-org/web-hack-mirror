Changing a user's attributes

    You can change a user's name, Internet alias, password, or administrative privileges at any time. You'll typically need to change a password because a user has forgotten it.

    To change a user's attributes, follow these steps:
    1 Log on to AppleShare IP Remote Admin.
    2

    Click the Users button.


       
    3 In the User list, click the user you want to edit.
    4 Select the type of change you want to make by clicking an item in the edit list, then make your changes in the window that appears.
    For more information about each setting, see Creating new users.
    5 Click the Save button. (Or, if you're changing Group Memberships, click Done.)
    6 Be sure to tell the user if you've changed his or her general information, password, mail settings, or group membership.
