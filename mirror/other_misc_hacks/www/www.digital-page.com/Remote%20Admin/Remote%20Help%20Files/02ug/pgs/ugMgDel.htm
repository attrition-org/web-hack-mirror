Deleting users or groups

    You can delete users or groups from the Users or Groups list.

    Note: Deleting a group does not delete the individual users that are members of the group.

    To delete a user or group, follow these steps:
    1 If you've installed the Mail Server, make sure it's running.
    The Mail Server needs to be running for mail settings to be updated when you delete a user. To start the Mail Server, log on to AppleShare IP Remote Admin and on the Status page, click the Start Mail Server button.
    2

    Click the Users button.


       
    3 To delete a user, click the user in the User list. (If you only want to delete a group, skip to step 6.)
    4 In the window that appears, click Delete User.
    5 In the list that appears, click a user that will inherit the current user's folder privileges. Then click Reassign & Delete.
    WARNING: When you delete a user, the Mail Server removes all of the user's mail from the Mail Database.
    Note: You cannot delete a user who is logged on.
    6 To delete a group, click the Group List button.
    7 Click the name of the group you wish to delete.
    8 In the window that appears, click Delete Group.
    9 In the list that appears, click a user or group that will inherit the current group's folder privileges. Then click Reassign & Delete.
