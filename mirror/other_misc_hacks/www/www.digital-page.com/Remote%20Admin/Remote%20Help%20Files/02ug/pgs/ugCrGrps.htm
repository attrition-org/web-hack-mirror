Creating groups

    You can create groups of registered users so that you can easily assign multiple users the same access privileges to shared files, your Web site, or mail services. Usually groups reflect the divisions or working relationships of an organization, but you can choose any arrangement, such as geographic location.

    To create a group, follow these steps:
    1 Log on to AppleShare IP Remote Admin.
    2

    Click the Users button.


       
    3 Make sure the users you want to add to your group are listed.
    If you need to register more users, follow the instructions in Creating users.
    4 Click the New Group button.
    5 In the Name box, enter the name of the group you want to create.
    The name can be up to 31 characters long and is not case-sensitive. If you want users to be able to send mail to this group, don't use the space character or the Option key.
    6 Click the Save & Continue button.
    7 Select a user to add to the current group, then click the Add button.
    The user appears in the list labeled "Is A Member Of".
    8 When you're done adding users to this group, click Done.
