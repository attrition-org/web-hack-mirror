Web & File Server troubleshooting

    This section helps you address problems that may occur when you are running the AppleShare IP Web & File Server to provide file sharing, FTP, and Web services.

    IMPORTANT: Some troubleshooting cannot be performed remotely. If the answers in this section don't solve the problem, go to the server and follow the troubleshooting instructions in AppleShare IP Help.
