General AppleShare IP troubleshooting

    This section describes how to solve general problems that may arise while using AppleShare IP.

    IMPORTANT: Some troubleshooting cannot be performed remotely. If the answers in AppleShare IP Remote Help don't solve the problem, be sure to go to the server and follow the Troubleshooting instructions in AppleShare IP Help.

    This section tells you how to solve this problem:

      You can't log on to Remote Admin

    Note: If you're having trouble finding the troubleshooting information that you're looking for, be sure to check the Web & File Server, Mail Server, and Print Server troubleshooting sections or the Index.
