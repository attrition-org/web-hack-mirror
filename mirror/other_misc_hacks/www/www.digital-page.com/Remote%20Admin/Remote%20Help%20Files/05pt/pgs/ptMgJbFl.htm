Changing the print jobs folder 

    By default, the Print Server stores jobs in a folder inside the Preferences folder on your startup disk. You may want to choose a folder on a different disk. The disk must be a local hard disk that cannot be removed.

    Note: If you want to store print jobs in a new folder, you should create the new folder before you start this task.

    To change the print jobs folder, follow these steps:
    1 Log on to AppleShare IP Remote Admin.
    2 Make sure the Print Server is running.
    3

    Click the Print server button.


       
    4 Click the Server Settings button.
    5 Click General.
    6 Under Print Jobs Storage Location, type a new location for all incoming print jobs. The path you type should start with the name of the hard disk, followed by the name of the print jobs folder, separated by colons. For example:
    <hard disk name>: <new print jobs folder name>
    or
    <hard disk name>: <folder name>: <new print jobs folder name>
    7 Click the Save button.
    Your changes take effect after you stop and start the Print Server. You should do this when you're sure no one is using it. See Starting and stopping the Print Server.
      As the Print Server restarts, it copies jobs from the old folder to the new folder.
