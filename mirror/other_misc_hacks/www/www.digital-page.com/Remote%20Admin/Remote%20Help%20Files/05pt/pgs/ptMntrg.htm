Monitoring print service 

    You can monitor Print Server activity on the Print Server Activity and Server Log pages.

    The Print Server Activity page is the central place for viewing the status of print queues and jobs. The Print Server logs record significant events, such as when the Print Server was stopped and started and when queues were created.

    This section tells you about viewing the

      Print Server Activity page
    Print Server logs

    Note: The Print Server Activity page is also the central place for managing the Print Server–editing queues, putting queues on hold, and so on. To learn about managing print jobs, queues, and printers, see Managing the Print Server.

Viewing the Print Server Activity page

    You can view the Print Server Activity page to see a list of the print queues available on your Print Server, a list of the attached printers, the number of jobs printing, and the status of print jobs.

    To view the Print Server Activity page, follow these steps:
    1 Log on to AppleShare IP Remote Admin.
    2 Make sure the Print Server is running.
    3

    Click the Print server button.


       
    The Print Server Activity page displays the following information:
    • Queues: A list of the server's print queues.
    • Jobs: The number of jobs currently being printed on each queue ("in queue")
    • Printers: The number of printers attached to each queue
    • Status (Queue): Whether the queue is operating normally. When there's a problem with a queue or a printer that's attached to it, an alert icon appears near the queue's name and the Status column indicates the cause of the problem.
    • Status (Printer): Whether the printer is idle, has an error, and so on.
    4 To see greater detail about the jobs being printed, click a queue in the list.
    The Queue window shows each print job in this queue, the name of the person sending the job, the status of the job, and the printer to which the job is being sent.

Viewing the Print Server logs

    The Print Server logs record significant events, such as when the Print Server was stopped and started and when queues were created. The All Queues log also contains information about jobs that have been printed or deleted.

    The maximum size of the Print Server log is 64K. When this limit is reached, old log entries are deleted so that new log entries can be added. You can also clear the log manually.

    To view the Print Server logs, follow these steps:
    1 Log on to AppleShare IP Remote Admin.
    2 Make sure the Print Server is running.
    3

    Click the Print server button.


       
    4 Click the Logs button.
    5 To see the contents of a log, click the name of the log in the Logs list.
    6 To see print activity, click Print Server log. This log lists events such as when the Print Server started and stopped, when queues were created, and when printers were attached to queues.
    7 To see information about each print job sent to a queue, click All Queues log.
