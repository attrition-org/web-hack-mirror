Managing print queues

    You can manage print queues by limiting who can print on them, editing settings, enabling or disabling queue protocols, and deleting queues.

    This section tells you how to

      Enable or disable secure printing
    Specify whether all local users or only a specified group of users can print on a queue
    Change print queue settings
    Cover pages, color printing advertisement, binary data printing, and printing while queuing
    Editing the attached printers list
    Attaching printers to and detaching printers from a queue, or enabling and disabling printers
    Enable or disable AppleTalk or TCP/IP queuing
    Change the name of a queue
    Delete a queue

    Note: The tasks in this section assume you've already created a print queue. If you haven't, see Creating a print queue.

 
Enabling and disabling secure printing
 

    You can secure an existing print queue by limiting use of the queue to users or groups you specify. You can temporarily turn off secure printing.

    Note: If you reenable secure printing after disabling it, the user list for whom you originally secured the queue is still intact.

    To make a print queue secure, follow these steps:
    1 Log on to AppleShare IP Remote Admin.
    2 Make sure the Print Server is running.
    3

    Click the Print server button.


       
    4 In the Print Server Activity queue list, click the name of the print queue that you want to make secure.
    5 Click Edit Queue.
    6 Click Security.
    7 To limit the users and groups who can print to this queue, select the Enable Security checkbox.
    When security is enabled, only the users and groups you specify will be able to print to this queue. When security is disabled, any user in your Users or Groups list can print to this queue.
    8 If you've installed the 8.6.1 or later LaserWriter driver on client computers, you can select the "Require Password For Printing" checkbox if you want users to enter a password before printing.
    9 Click the Save button.
    10 To add a user or group to the secure print list for this queue, click Users & Groups.
    11 Select a user or group to add to the secure queue list, then click Add.
    To remove a user or group, select the user in the "Can print to Queue" list, and click Remove.
    12 When you're done adding users or groups, click the Done button at the bottom of the page.
    The selected print queue is now secure. Only users and groups listed in the secure queue user list can print on the print queue. (Other users who try to print on this queue will receive an error message.)

Editing the attached printers list

    You can edit the attached printers list for a queue by attaching or detaching printers, or by putting printers in the list on or off hold. (Putting a printer on hold temporarily prevents the Print Server from using it.)

    You may want to attach more printers to improve printing performance. You may want to detach a printer from a queue, for example, to attach it to another queue or to change the PostScript printer description (PPD) file for the selected queue. Detaching a printer removes it from the queue.

    For more on attaching printers or PPDs, see Guidelines for attaching multiple printers to a queue.

    To edit the attached printers list for a queue, follow these steps:
    1 Log on to AppleShare IP Remote Admin.
    2 Make sure the Print Server is running.
    3

    Click the Print server button.


       
    4 To place a printer on hold or detach it from a queue, click a printer in the queue's printer list. To attach printers to a queue, skip to step 6.
    5 In the queue window, select the "Hold printer" checkbox, then click Save. Or click the Remove Printer button.
      If the printer you are going to detach is the only printer attached to the queue, tell users that the queue will be unavailable.
    6 Click the queue to which you want to attach a printer.
    7 In the queue window, click Edit Queue.
    8 In the Edit Queue list, click Attached Printers.
    9 In the Attached Printers list, click Attach AppleTalk Printer or Attach TCP/IP Printer, then enter printer information in the screens that follow. For instructions, start on step 7 of Creating a print queue.

Changing print queue settings

    You can change print queue settings for cover pages, printing while queuing, advertising color printing, and binary data.

    To edit print queue settings, follow these steps:
    1 Log on to AppleShare IP Remote Admin.
    2 Make sure the Print Server is running.
    3

    Click the Print server button.


       
    4 In the Print Server Activity queue list, click the name of the print queue that you want to edit.
    5 In the Queue window, click Edit Queue.
    6 Click General.
    7 Edit the General settings for this queue.
    If you want a cover page to be printed (includes the user and file name) with jobs sent to this queue, scroll to the Print Cover Page section, and click the "Before each job" or "After each job" button.
    If you don't want a color printer icon to appear next to the queue's name in the Chooser (when printing over AppleTalk), deselect the "Advertise color printing" checkbox. (The icon appears only when the first printer that you attach to a queue supports color.)
    If you want to allow only ASCII data to be printed, deselect the "Allow binary data" checkbox. Older laser printers can't print documents that contain binary data.
    8 Click the Save button.

Enabling and disabling AppleTalk or TCP/IP queuing

    You can disable queuing to AppleTalk or TCP/IP queues.

    Note: Enabling or disabling AppleTalk or TCP/IP queuing is a queue-specific setting. For more information, see Enabling and disabling TCP/IP printing.

    To disable queuing by network type, follow these steps:
    1 Log on to AppleShare IP Remote Admin.
    2 Make sure the Print Server is running.
    3

    Click the Print server button.


       
    4 In the Print Server Activity queue list, click the name of the print queue that you want to edit.
    5 In the queue window, click Edit Queue.
    6 Click General.
    7 To prevent users from printing over AppleTalk, deselect the "Enable AppleTalk" checkbox.
    To allow users to print on queues over AppleTalk, click the "Enable AppleTalk" checkbox.
    8 To prevent users from printing over TCP/IP, deselect the "Enable TCP/IP" checkbox.
    To enable users to print on queues over TCP/IP, click the "Enable TCP/IP" checkbox so that a checkmark appears.
    9 Click the Save button.

Changing the name of a queue

    You can change the name of a queue as it appears in the Chooser or as users need to type it in their Desktop Printer Utility window.

    Note: After you change the name, users who selected the queue's old name in the Chooser have to select the queue's new name. Users printing over TCP/IP have to type the new name in their Desktop Printer Utility window.

    To change the name of a queue, follow these steps:
    1 Log on to AppleShare IP Remote Admin.
    2 If the Print Server is not already running, click the Start Print Server button in the Status window.
    3

    Click the Print server button.


       
    4 In the Print Server Activity queue list, click the name of the print queue that you want to edit.
    5 In the Queue window, click Edit Queue.
    6 Click General.
    7 Type a new name for the queue.
    The name can be up to 31 characters and cannot contain special characters. For AppleTalk printers, the name must also be unique within its zone.
    If you do not want users to be able to print directly on this printer, the queue name can be the same as the name of the attached printer. See Creating print queues.
    8 Click the Save button.
    Be sure to tell users the new print queue name.

Deleting a queue
 

    You may want to delete a queue if you are reorganizing your printers.

    To delete a print queue, follow these steps:
    1 Log on to AppleShare IP Remote Admin.
    2 Make sure the Print Server is running.
    3

    Click the Print server button.


       
    4 In the Print Server Activity queue list, click the queue that you want to delete.
    5 Click Delete Queue.
    The Print Server stops sending jobs to printers that are attached to the queue, deletes all jobs that are waiting to be printed, and stops accepting jobs for this queue.
