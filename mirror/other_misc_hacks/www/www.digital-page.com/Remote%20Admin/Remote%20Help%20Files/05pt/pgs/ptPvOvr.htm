    Note: The instructions in this section assume you've configured the AppleTalk and TCP/IP control panels on your server following the instructions in your "Getting Started" manual.

      Start and stop the Print Server

    Create print queues

    Set up TCP/IP printing
    Enable or disable TCP/IP printing, and set the default print queue for TCP/IP connections
