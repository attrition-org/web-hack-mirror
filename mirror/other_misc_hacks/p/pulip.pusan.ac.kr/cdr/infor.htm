CD-ROM 검색프로그램 설치 안내

 CD-ROM을 검색하기 위해서는 우선 아래의 프로그램을 다운로드 하십시요.

 

CD-ROM 검색용 클라이언트 프로그램 (원도우95/98)

Nettra 2000 프로그램 설치 방법

1. 위의 프로그램을 다운로드 받은 후에 해당 파일(nettra2000.exe)을 더블클릭하면 <화면1>과 같은 윈도우가 나타납니다.

 

화면1. Nettra2000 Client 프로그램의 설치시작

 

 

2. 여기서 “Setup” 버튼을 누르면 도스 창이 나오면서 자동으로 압축이 풀립니다. 이렇게 압축이 모든 풀리고 마지막으로 아래 <화면2> 연결관리자 가 나타납니다.

 

 

화면 2. 가져오기를 위한 연결관리자

 

 

3. 화면의 '파일메뉴' 에서 '가져오기'를 누르면 아래 <화면 3>이 나옵니다. 여기서 'connect.cns 파일' 을 선택하시고 “열기” 버튼을 누르면 됩니다.

 

화면 3. 다음에서 가져오기 화면

 

  이후에 나오는 모든 다이얼로그창에서 '예(Yes)'를 선택하시면 됩니다.

  그리고 마지막으로 위의 <화면 2> 에서의 '파일메뉴' 에서의 '끝내기' 버튼을 누르면 창을 닫으면 프로그램 설치가 모두 끝나게 됩니다.

 

 

Nettra 서버의 접속과 CD-ROM의 검색 방법 안내

접속 방법

1. 자신의 컴퓨터 C:\nettra\nettra.sc를 실행시키면 됩니다.

2. CD-NET 서버에 접속되면 아래화면처럼 CD-Rom의 리스트가 나타납니다. 원하시는 CD를 더블클릭하셔서 검색하시면 됩니다.
