학술 CD 검색

※ CD-NET 검색은 웹 브라우저를 이용한 인터넷 방식이 아니라 CD-NET 전용프로그램을 이용하는 검색입니다.

 

CD-NET 검색을 재개하였습니다만, 학교내 컴퓨터에 있는 Funlove바이러스(네트워크 공유를 통해 침투)가 아직 돌아다니기 때문에 완전히 정상적인 서비스를 할 수 없습니다. 약간의 변경사항을 아래와 같이 알려 드립니다.

 

CD-NET실행파일
(도서관 홈페이지에서 CD-NET 프로그램을 설치하시면 컴퓨터의 C 드라이브 밑에 NETTRA폴더가 자동 생성됩니다.)
       이전 - C:\NETTRA\CD_RUNCLIENT.EXE 실행

       신규 - C:\NETTRA\NETTRA.SC 실행

 

CD-NET 서버 접속방법
         NETTRA.SC 실행 → 사용자이름/암호를 묻는 과정에서 그냥 확인버튼을 클릭
         → 자신의 컴퓨터 IP주소와 컴퓨터 이름을 선택하여 접속합니다.

 

검색결과 저장 및 출력 방식

       이전 - 하드디스크 저장 + 프린트

       신규 - A드라이브에만 저장

 

이 바이러스가 학내에서 다 잡히면 검색결과 프린트 및 하드디스크 저장이 가능한 새버젼으로 업그레이드 하겠습니다. 

 

CD-ROM 검색프로그램을 다운로드하여 PC에 설치하신 후에 검색하십시오.

도서관에 오실 필요없이 Network을 통해 LAN이 설치된 교내 어디서나 검색할 수 있습니다.(일부는 제외)

검색결과를 A드라이브에 저장하실 수 있습니다.(바이러스 문제가 정리되면 곧 새 버젼으로 업그레이드 하겠습니다.)


프로그램 다운로드는 여기를 클릭하십시요.
