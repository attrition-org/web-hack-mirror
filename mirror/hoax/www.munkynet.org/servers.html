tamarin.munkynet.org

Tamarin is a 333 MHz Cyrix desktop computer with 88 MB RAM and 4 GB local storage. It was once a free-pc, but now runs Debian GNU/Linux 2.2.

Tamarin runs most of the services for the munkynet.org domain, including running the domain's primary domain name server, a MySQL server, and the primary web server, which hosts search.resnet.rochester.edu and UR Peace.

In concert with the search.resnet.rochester.edu web site, tamarin runs the UR network neighborhood indexer, which is a rewrite of Jeff Feasel's UB network neighborhood search.

In addition to providing the above services, tamarin also holds the user accounts of any "full" munkynet.org account holders, and provides remote telnet, ssh, pop3, ftp, smb, and afp, and local X access to these accounts.

Reverse DNS lookups will report this host as search.resnet.rochester.edu. It is also known as www.munkynet.org, ftp.munkynet.org, and mail.munkynet.org. Chances are *.munkynet.org points to this host.

marmoset.munkynet.org

Marmoset is the direct descendant of paddies.resnet.rochester.edu, the file server that I set up in my first year of college. It is also a Debian GNU/Linux 2.2 system, running on a 200 MHz MMX Pentium with 64 MB RAM and 75 GB local storage. Marmoset's tower case is painted black, with red and yellow flames on the sides.

Marmoset provides private access to my file archive via ftp, smb, and afp. Passwords can be changed via telnet or ssh, though users on this system do not have a full shell environment. Please note that marmoset is a private server and does not allow any anonymous connections. All unauthorized access is prohibited.

Marmoset currently acts as the munkynet.org domain's secondary domain name server. However, this is only an interim solution, until I find an offsite name server.

Reverse DNS lookups will report this host as pad.resnet.rochester.edu. It is also known as files.munkynet.org.
