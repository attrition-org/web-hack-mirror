Transfering a Domain Name to MartNet

Moving an existing domain name to MartNet is quite simple, but changing registration information is the responsibility of the owner of the domain (usually the customer. The Administrative contact of the Domain is the only person or entity that can usually modify Domain Name Registrations. If you are not registered as the Administrative Contact for the Domain you wish to transfer, then you MUST get whoever is the Administrative contact to submit the necessary changes to the Internic, or whichever registration service you used to obtain the Domain Name. To view contact information for a given domain, use the Internic Whois search engine. If you have any questions or concerns, you should contact whoever is currently hosting the Domain Name or The Internic.)
 
Click Here to go to the Internic's Help Page about transfering an existing Domain Name. A copy of the form you will need to submit along with detailed instructions are available here.
 
When changing the registration record of an existing Domain, the following must be changed:
 
Technical Contact should be changed to NIC Handle MH394-ORG
(a NIC handle just a database record for an existing Internic entity, in this case, the MartNet Hostmaster. It saves you the trouble of typing in a name, address, etc. every time you register or modify a Domain Name registration.)
 
Primary Nameserver must be changed to:
NS.MARTNET.COM 207.19.97.101
 
Secondary Nameserver must be changed to:
NS2.MARTNET.COM 216.203.14.133
 
These 3 options are the only things that need to be changed so that your Domain can be hosted properly at MartNet. Once the change is submitted to the Internic, it can take up to three days for the change to take effect, and for all Name Servers throughout the Internet to become aware of the change, so please plan accordingly if moving an existing Web Site along with the Domain to MartNet (ie: arrange with us to host your Domain, get your site moved and tested to be sure everything works properly, then submit the registration change to the Internic.)
 

Your Updated Domain Name Registration should look something like this. Red Text is the stuff that has to be EXACTLY AS SHOWN for your Domain to work properly when hosted at MartNet. The Administrative and Billing contact should be someone in your organization who will be responsible for managing and paying for the Domain.
