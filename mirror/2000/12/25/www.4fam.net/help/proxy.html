Proxy settings

The proxy settings must be set properly for your browser to be able to access anything off the 4FAM home page.

The proxy does the filtering for our users and proper configuration is required to gain access to the net.

Use of the proxy not only makes your browsing safer (porn and violence free) but also faster!

 
Browser Configuration
Internet Explorer 5
  • Open Internet Explorer 5
  • Click on Tools
  • Click on Internet options...
  • Click on Connections tab
  • Highlight the dialer you use to access 4FAM
  • Click on the Settings button
  • Check the box for Use a proxy server
  • Set the address to speedy.mics.net
  • Set the port to 80
  • Click on OK
  • Click on Apply
  • Click on OK
You are set to use the proxy now.
Netscape 4.6 or later
  • Open Netscape Navigator
  • Click on Edit
  • Click on Preferences...
  • On the left, in the window, at the bottom click on the + in front of Advanced
  • Click on Proxies
  • Select Manual proxy configuration
  • Click on the View button
  • Set HTTP, Security, FTP, and Gopher to use speedy.mics.net on port 80
  • Don not change anything else
  • Click OK and then click OK again.
You are set to use the proxy now.
