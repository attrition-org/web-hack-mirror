Password problems

Your password and your user name are used as a pair to determine if you can gain access to the system.

If either is incorrectly spelled the system will deny access and reject the password as invalid.

On our UNIX systems, capitalization is significant, therefore; all user names must be spelled using lower case characters.

User names and passwords never contain a "space" character (not even at the end)

 
Problem Description Solution
Won't take my password Retype both user name and password, making sure to follow the hints above and that the caps lock is not active
I get a second log on screen

Uncheck "Log onto Network" in Dial-up Networking properties for the dialer you are using:

From the desktop-

  • Click on My Computer
  • Click on Dial-Up Networking
  • Right click on the dialer you use to call 4fam
  • Go all the way down to the bottom of the menu and left click on Properties
  • Click on the Server Types tab
  • Uncheck "Log onto Network" if selected by clicking on the check box.
  • Click on OK
  • Close all open windows

Try to login now.
