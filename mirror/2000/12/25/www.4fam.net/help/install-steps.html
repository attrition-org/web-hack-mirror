Customer Installation Steps

This process uses Internet Explorer and Outlook Express which must already be installed on your machine.... it they are not please use the MIC&S installation CD.

  1. Check modem settings
  2. Create MIC&S dialer
  3. Setup Internet Explorer
  4. Setup Outlook Express
  5. Test the setup, if you have problems call for support at 297-6212 or 1-877-get4fam toll-free from outside the Dayton local calling area

Check modem settings

From the desktop:

  • Click on My Computer
  • Click on Control Panel
  • Click on Modems
  • Highlight the the modem you use
  • Click on the Properties button
  • Set the max speed 57600
  • Click on Connection tab
  • It should read (from top to bottom) 8, none, 1
  • Verify the check in the box for Wait for dial tone before dialing
  • Uncheck any other boxes that are checked
  • Click on OK
  • Click on OK or Close
  • Close the Control Panel window
  • Close any open widows, go back to the desktop

back to installation steps

Create a MIC&S Dialer

  • Click on My Computer
  • Click on Control Panel
  • Click on Dial-up Networking
  • What do you see? If you see a MIC&S icon click here otherwise go to next step
  • Click on Make New Connection icon
  • Type MIC&S, then click on Next
  • Type phone number... select appropriate number from list, then click on next
  • Click on Finish
  • You should now see the MIC&S icon in the Dial-up Networking window
  • Using your mouse to point at the MIC&S icon, do a RIGHT button click on the icon
  • From the menu, select Properties with a left button click
  • Verify the phone number is correct
  • Take the check out of the box labeled Use area code and dialing properties
  • Click on the Configure...button at the bottom of the page.
  • There pages look like the ones we visited while doing the modem configuration stuff, but they can and often do contain different settings. In order to work correctly, they must be the same so....
  • Set the max speed 57600
  • Click on Connection tab
  • It should read (from top to bottom) 8, none, 1
  • Check in the box for Wait for dial tone before dialing
  • Uncheck any other boxes that are checked
  • Click on OK
  • You should now be back at the General tab, select the Server Types tab by clicking on it
  • In the box label Advanced options remove all check, nothing should be checked
  • In the box at the bottom labeled Allowed protocols remove the top two checks leaving only TCP/IP checked on the entire page
  • Click on OK
  • Close all open windows

back to installation steps

Setup Internet Explorer

  • From the desktop, do a right button click on the Internet Explorer icon
  • Go all the way to the bottom of the menu and select Properties with a left button click
  • You should see a 6 tab window now, if you do not then do the following:
    • Close the current widow, and double click on Internet Explorer
    • Click on Tools on the menu
    • Go all the way to the bottom and click on Internet Options
    • You should now be on the six tab window
  • On the General tab, in the box at the top label Home Page in the address window insert the address: http://www.4fam.net
  • Go to the Connections tab look for the dialer created earlier, highlight the MIC&S dialer in the box, click on the settings button
  • Check the box for Use a proxy server
  • Set the address to speedy.mics.net
  • Set the port to 80
  • Click on OK
  • With the MIC&S dialer still highlighted, click on the Set Default button
  • Put a dot in the selection to Always dial my default connection
  • Click on the Programs tab
  • Select Outlook Express as the e-mail program
  • Click on OK
  • You should now be back at the desktop

back to installation steps

Setup Outlook Express

  • Double click on Outlook Express from the desk top
  • Work off-line or cancel if it offers to dial
  • Click on Tools on the menu
  • Go down and click on Accounts
  • Select the Mail tab
  • If you have any other mail account other than the MIC&S which is inactive you should highlight this account and click on Remove
  • Click on Add, then select mail.
  • Answer the questions according to your account setup and the values below
  • Your name should be entered according to what you like to see in the From section of the e-mails sent and click Next
  • Your e-mail address is your <your account name>@mics.net    Please note that the terminology: account name, user name, user, or account can for the most part be used interchangeable
  • Your e-mail account is a POP3
  • Enter the incoming mail server as: mail.mics.net
  • Enter the outgoing mail server as: mail.mics.net
  • Click Next
  • Your account name should already be filled in, if not enter it now
  • Enter your password, click Next
  • Select: Connect using my phone line, Click Next
  • If asked select the MIC&S dialer, Click Next
  • When congratulated click on Finish
  • Click Close
  • Close Outlook Express

back to installation steps
