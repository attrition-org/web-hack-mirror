Access to the Online Catalog and Research & Reference Databases components of RIDER requires the Citrix Winframe ICA plug-in for Netscape Navigator and Microsoft Internet Explorer. Follow the below instructions to install the plug-in on your terminal.

1. Download the self-extracting EXE for Netscape Navigator:

(Click on above link and choose "Save Program to Disk or Right-click on the link and select 'Save Link As' or 'Save Target As').  Select a directory to download the WFPLUG32.EXE to.

or

  Dowload the self-extracting EXE for Internet Explorer:

(Click on above link and choose "Save Program to Disk or Right-click on the link and select 'Save Link As' or 'Save Target As').  Select a directory to download the WFICA32.EXE to.

 

2. Registering the plug-in to the browser:

From your Start menu click:  Start -->Run and then type the following:

drive:\downloadpath\WFPLUG32.EXE /setup 
for Netscape, replacing drive and downloadpath with the drive letter and full path to where WFPLUG32.EXE was downloaded.

or

drive:\downloadpath\WFICA32.EXE /setup 
for Internet Explorer, replacing drive and downloadpath with the drive letter and full path to where WFICA32.EXE was downloaded.
