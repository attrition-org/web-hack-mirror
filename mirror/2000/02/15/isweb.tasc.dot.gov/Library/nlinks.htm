Search tips and search engines

Introduction to Internet search tools

There are two basic types of search tools: search engines and directories. Search engines are tools like AltaVista and Excite, which allow you to enter terms related to the topic you are searching and then return any number of web pages, depending on the number of sites that match your search criteria indexed by that particular search engine. Yahoo is another type of search tool, called a directory, which provides a more structured approach to finding information on the web. Sites are organized by subject. These types of tools are useful for getting a general idea of what information is out there, but they do not index as many sites as many of the search engines.

Some general tips:

  • If you plan to use one or more search engines on a regular basis, spend some time reading some of the help screens on the search site. These often give useful information to get the most out of the search engine.
  • Many search engines offer an advanced search option which allows you to utilize searching techniques not available with the basic search.
  • Some search engines will allow you to search not only web documents, but Usenet newsgroups as well. Deja News is a service that searches newsgroups only.
  • For more in depth search tips and comparisons of the various search tools, take a look at one of the following sites:
