Herramientas y utilerías

Adobe Acrobat Reader

Reader/Lector Función
Adobe Acrobat Reader Permite a los usuarios de Windows® 95, Windows 3.x, Windows NT® 3.51 o posterior, Macintosh, Linux y otras plataformas, ver e imprimir archivos de tipo PDF (Portable Document Format o Formato Portátil de Documento).

Microsoft Internet Explorer

Browser/Explorador Función
Microsoft Internet Explorer Permite a los usuarios de Windows® 95, Windows 3.1x, Windows NT® 3.51 o posterior, Macintosh y UNIX, ver y navegar por documentos de Internet.

Netscape Communicator

Browser/Explorador Función
Netscape Communicator Permite a los usuarios de Windows® 95, Windows 3.1x, Windows NT® 3.51 o posterior, Macintosh, UNIX y otras plataformas, ver y navegar por documentos de Internet.

Actualización y complementos de Microsoft Office

Microsoft Word Viewers

Viewer/Lector Función
Word 97 Viewer (Windows 95/98 or Windows NT) Permite a los usuarios de Windows® 95/98 o de Windows NT® 3.51 o posterior, abrir documentos de Word 97 para su lectura e impresión.
Word 97 Viewer (Windows 3.x) Permite a los usuarios de Windows® 3.1 o posterior abrir documentos de Word 97 para su lectura e impresión.

Microsoft PowerPoint Viewers

Viewer/Lector Función
PowerPoint Viewer 97 (Windows 95/98 or Windows NT) Permite a los usuarios de Windows® 95/98 o de Windows NT® 3.51 o posterior abrir presentaciones creadas con PowerPoint para Windows versión 2.0 o posterior y PowerPoint para Macintosh versión 3.0 o posterior para su lectura e impresión.
PowerPoint Viewer 97 (Windows 3.x) Permite a los usuarios de Windows® 3.1 o posterior abrir presentaciones creadas con PowerPoint para Windows versiones 2.0 a 4.0, PowerPoint 95 o PowerPoint para Macintosh versiones 3.0 a 4.0 para su lectura e impresión.

Microsoft Excel Viewer

Viewer/Lector Función
Excel 97 Viewer Permite a la gente que (1) está usando el sistema operativo Windows® 95/98 o el sistema operativo Microsoft Windows NT® Workstation 3.51 o posterior y que (2) no tienen Excel, ver e imprimir archivos de hoja de cálculo creadas en Microsoft Excel para Windows versión 2.0 o posterior y Microsoft Excel para Macintosh versión 2.2a o posterior.

Descompactador PKzip de PKware

Programa Función
PKWARE- PKZIP Permite a los usuarios de Windows® 95, Windows NT®, Windows 3.1x, DOS, UNIX y otras plataformas, abrir archivos comprimidos con el programa PKZIP y descomprimir sus contenidos.
