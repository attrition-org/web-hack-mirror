PLEASE NOTE -

THE FOLLOWING SECTIONS OF INFORMATION WILL HELP SET UP EXPLORER 5.0 & 5.01 FOR USE WITH THE GAINSKEEPER SITE.

SECTION 1 - Adjusting Default Settings:

There are several new features in the IE5.01 Browser that Microsoft has set as "DEFAULT" settings. They are designed to make web browsing easier but they create many problems with e-commerce web sites.

The following instructions will help you set up your browser:
(They only need to be performed once)

Turn off "Auto Complete Form"
This function saves information you entered into forms including passwords and usernames this is not advised for security reasons. This function attempts to fill-in forms for you, creating many problems while accessing your account information.

1. Go to the TOOLS menu > then INTERNET OPTIONS item > then CONTENT tab.

  • Under "Personal Information" Click the "AutoComplete" button.
  • "Uncheck" the following: Forms, User names and passwords.
  • The click the "Clear Forms" and "Clear Passwords" buttons.
  • Click "OK".

2. Go to the TOOLS menu > then INTERNET OPTIONS item > then ADVANCED tab.

  • In the "Settings" scrolling window go to the BROWSER header.
  • "Uncheck" - "Use inline AutoComplete for Web Addresses".
  • Go to the SECURITY header.
  • "Check" - "Do not save encrypted pages to disk.
  • "Check" - " Empty Temporary Internet File folders when browser is closed".
  • "Uncheck" - "Warn if changing between secure and not secure mode".

Clear out old web pages
This will remove any old pages that were giving you problems from your previous browser.

1. Go to the TOOLS menu > then INTERNET OPTIONS item > then GENERAL tab.

  • Under "Temporary Internet files" click "Delete Files".
  • Check" - "Delete all offline content" - Click "OK".
  • Under "Temporary Internet files" click "Settings".
  • "Check" - "Every visit to the page".
  • Click "OK".

Clear your History
This will remove any old links that are cuasing current problems.

1. Go to the TOOLS menu > then INTERNET OPTIONS item > then GENERAL tab.

  • Under "History" click "Clear History".
  • At the dialog window click "OK".
  • Close the "Internet Options" window by clicking "APPLY then "OK" at the bottom of the window.
