ADJUSTING DEFAULT SETTINGS:

There are several new features in the Navigator Browser that have been set as "DEFAULT" settings. They are designed to make web browsing easier but they create many problems with e-commerce web sites.

The following instructions will help you set up your browser:
(They only need to be performed once)

Turn off "Auto Complete Form"
This function saves information you entered into forms including passwords and usernames. This is not advised for security reasons. This function attempts to fill-in forms for you, creating many problems while accessing your account information.

Clear out old web pages
This will remove any old pages that were giving you problems from your previous browser and assure that the data you are viewing is the most recent data available.

1. Go to the EDIT menu > then PREFERENCES item.

  • Under "+ Advanced" click "Cache".
  • On the page to the right under the section "Document in cache is compared to document in Network:" click the radio button for "Every Time".

    Clear your History
    This will remove any old links that are causing current problems.

  • Click the "Clear Memory Cache" and "Clear Disk Cache" buttons on the same page.
  • Click "OK" to close and save changes.

RESTART YOUR BROWSER!
