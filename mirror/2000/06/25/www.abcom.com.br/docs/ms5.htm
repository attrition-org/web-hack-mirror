Microsoft Certified Professional + Internet

Requisitos: Passar nos três exames abaixo, que garantirão o seu nível de conhecimento e expertise na Internet.

Benefícios:

  • Reconhecimento de sua proficiência nos produtos e tecnologias Microsoft, através do certificado.
  • Acesso à informação sobre os produtos e tecnologias Microsoft através do site privado para MCPs.
  • Logotipos do programa de MCP.
  • Assinatura grátis da revista Microsoft Certified Professional Magazine.



MCP+Internet

O candidato deve ser aprovado em três exames

  • Exame 70-059: Internetworking Microsoft® TCP/IP on Microsoft® Windows NT 4.0
  • Exame 70-067: Implementing and Supporting Microsoft® Windows NT Server 4.0
  • Exame 70-077: Implementing and Supporting Internet Information Server 3.0 and Microsoft® Index Server 1.1
    Ou
  • Exame 70-087: Implementing and Supporting Microsoft® Internet Information Server 4.0
