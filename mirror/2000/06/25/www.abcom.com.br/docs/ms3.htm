O que é um Microsoft Certified Professional

O mercado não para de acessar os avanços tecnológicos da informática. . E exige, cada vez mais, profissionais que possam projetar, desenvolver e dar suporte a essa tecnologia. O Programa de Treinamento e Certificação da Microsoft foi criado para profissionais da área de informática que desejam se tornar especialistas nos produtos e no ambiente Microsoft.

Para o profissional, o Programa representa um método rápido e seguro de acesso a novos conhecimentos e atualização sobre os produtos Microsoft utilizados no seu dia-a-dia e sobre aqueles que acabam de ser lançados. O que resulta em maior competência técnica, produtividade e eficiência. E essa qualificação irá trazer reconhecimento profissional, facilidade de desenvolvimento de carreira e os benefícios pessoais decorrentes disso.

Dessa forma, Microsoft Certified Professional é o título que o profissional recebe ao ser aprovado nos exames de uma das categorias de certificação Microsoft. Existem atualmente 8 categorias para o profissional técnico que deseja se especializar. São elas:

1) Microsoft Certified Professional - 1 exame
2) Microsoft Certified Professional + Internet - 3 exames
3) Microsoft Certified Professional + Site Building - 2 exames
4) Microsoft Certified Database Administrator - 5 exames
5) Microsoft Certified Solution Developer - 4 exames
6) Microsoft Certified Systems Engineer - 6 exames
7) Microsoft Certified Systems Engineer + Internet - 9 exames
8) Microsoft Certified Trainer

Para ganhar qualquer um dos títulos acima, o profissional deverá, como primeiro passo, ser aprovado em um exame de sistema operacional e, a partir disso, ele poderá se especializar, percorrendo o caminho mais atraente, de acordo com cada uma das outras categorias.
