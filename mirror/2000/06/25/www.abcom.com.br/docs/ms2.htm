Oito categorias de certificação para atender a diferentes necessidades profissionais.

Os exames possibilitam oito categorias diferentes de certificação, à escolha do candidato, que irão torná-lo um profissional reconhecido internacionalmente como um Microsoft Certified Professional - MCP. Um verdadeiro expert na tecnologia Microsoft.

Microsoft Certified Professional
Oferece ao profissional conhecimentos avançados sobre um produto específico da Microsoft, tornando-o um especialista. Para alcançar esse título o profissional deverá certificar-se em 1 exame (exceto Networking Essentials).

Microsoft Certified Professional + Internet
Proporciona uma qualificação específica em Internet, para planejar a segurança, instalação e configuração dos servidores, gerenciar seus recursos, estender os servidores para rodar "scripts" CGI ou ISAPI, monitorar e analisar a performance e diagnosticar problemas. Para alcançar esse título o profissional deverá certificar-se em 3 exames.

Microsoft Certified Professional + Site Building
Proporciona uma qualificação para desenvolvedores de web sites. O profissional com essa certificação é qualificado para planejar, desenvolver, manter e gerenciar sites da web utilizando os produtos e tecnologia Microsoft. Para alcançar esse título o profissional deverá certificar-se em 2 exames.

Microsoft Certified Database Administrator
Microsoft Certified Database Administrator é a primeira certificação para profissionais que implementam e administram bancos de dados em Microsoft SQL Server. Essa certificação é apropriada para profissionais que desenvolvem modelos de bancos de dados, criam serviços de dados utilizando Transact-SQL, gerenciam e mantém bancos de dados, configuram e gerenciam a segurança, monitoram e otimizam bancos de dados e instalam e configuram o Microsoft SQL Server. Para alcançar esse título o profissional deverá certificar-se em 5 exames.
Os cursos e exames para essa nova certificação estarão disponíveis no 1º trimestre de 1999.

Microsoft Certified Solution Developer
Capacita o profissional para projetos, especificações e desenvolvimento de soluções customizadas de negócios utilizando as tecnologias e ferramentas de desenvolvimento Microsoft Tools. Para alcançar este título, o profissional deverá certificar-se em 4 exames.

Microsoft Certified Systems Engineer
Qualifica o profissional para o planejamento, implantação, manutenção e suporte técnico de soluções de negócios baseadas na tecnologia Microsoft Windows NT Server e na família de produtos servidores BackOffice. Para alcançar este título o profissional deverá certificar-se em 6 exames.

Microsoft Systems Engineer + Internet
Além das habilidades que um Systems Engineer possui, o MCSE + Internet estará capacitado a otimizar, implementar e gerenciar soluções sofisticadas de Intranet e Internet que incluem o browser, o servidor proxy, servidores host, bancos de dados e componentes de comércio e messaging. Além disto, um profissional certificado MCSE + Internet será capaz de gerenciar e analisar sites da Web. Para alcançar esse título o profissional deverá certificar-se em 9 exames.

Microsoft Certified Trainer
Específico para profissionais que desejam se tornar instrutores, munindo-os com qualificação técnica e didática para ministrar cursos do Microsoft Official Curriculum nos Microsoft Certified Technical Education Centers - Microsoft CTEC.
