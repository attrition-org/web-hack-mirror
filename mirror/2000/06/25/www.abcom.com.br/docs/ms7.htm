Microsoft Certified Database Administrator

Requisitos:

Passar em 4 exames obrigatórios e 1 exame eletivo.

Benefícios:

  • Reconhecimento da sua proficiência nos produtos e tecnologias Microsoft, através do certificado.
  • Um ano complementar de assinatura grátis do Microsoft TechNet Technical Information Network, provendo informações valiosas em CDs mensais
  • Um ano de assinatura grátis dos CDs beta em inglês dos novos produtos.
  • Um ano de assinatura grátis da revista SQL Server Magazine
  • Acesso à informação sobre os produtos e tecnologias Microsoft através do site privado para MCPs.
  • Logotipos do programa de MCP, e logotipo de Profissional Certificado Microsoft Certified Database Administrator.
  • Assinatura grátis da revista Microsoft Certified Professional Magazine.



Passar em 4 exames obrigatórios:

Exam 70-028: Administering Microsoft® SQL Server 7.0
Exam 70-029 Designing and Implementing Databases with Microsoft® SQL Server 7.0
Exam 70-067 Implementing and Supporting Microsoft® Windows NT® Server 4.0
Exam 70-068 Implementing and Supporting Microsoft® Windows NT® Server 4.0 in the Enterprise

Passar em 1 exame eletivo:

Exam 70-015: Designing and Implementing Distributed Applications with Microsoft® Visual C++® 6.0
Exam 70-019: Designing and Implementing Data Warehouses with Microsoft® SQL Server 7.0 and Microsoft® Decision Support Services 1.0
Exam 70-059 Internetworking with Microsoft® TCP/IP on Microsoft® Windows NT® 4.0
Exam 70-087 Implementing and Supporting Microsoft® Internet Information Server 4.0
Exam 70-175 Designing and Implementing Distributed Applications with Microsoft® Visual Basic® 6.0
