Microsoft Certified Professional + Site Building

Requisitos: Passar em dois dos exames abaixo, que garantirão o seu nível de conhecimento e expertise técnico.

Benefícios:

  • Reconhecimento de sua proficiência nos produtos e tecnologias Microsoft, através do certificado.
  • Acesso à informação sobre os produtos e tecnologias Microsoft através do site privado para MCPs.
  • Logotipos do programa de MCP, e logotipo do profissional certificado Microsoft Certified Professional + Site Building.
  • Assinatura grátis da revista Microsoft Certified Professional Magazine.



Passar em 2 dos exames abaixo:

Exam 70-055: Designing and Implementing Web Sites with Microsoft® FrontPage® 98
Exam 70-057: Designing and Implementing Commerce Solutions with Microsoft® Site Server 3.0, Commerce Edition
Exam 70-152: Designing and Implementing Web Solutions with Microsoft® Visual InterDev 6.0
