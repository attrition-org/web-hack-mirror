CIP, Novell Internet Professional

O Que É?

A Internet e a World Wide Web estão em todos os lugares! Elas afetam diariamente nossa vida pessoal e profissional. E as oportunidades que cercam essa rede de redes estão crescendo proporcionalmente ao número de novos usuários que se conectam a ela a cada dia. As companhias estão percebendo o potencial dessa tecnologia emergente e estão usando cada vez mais a Internet para marketing, comunicação e transferência de informações. Para o Novell Education, isto significa a necessidade de funcionários com qualificações certificadas para gerenciar os Web sites da companhia.

A certificação Novell Certified Internet Professional prova que você tem capacidade para projetar e gerenciar Web sites. As duas opções da certificação Novell Certified Internet Professional lhe fornecem as seguintes qualificações:

Gerenciador de Web Site

  • Planejar comercialmente a distribuição de Web sites e tecnologia Internet
  • Copiar um Web site
  • Agrupar os recursos necessários para construir um Web site
  • Medir o retorno dos Web sites de sua organização
  • Gerenciar efetivamente a manutenção de implementação de Web sites

Opção de Projetista da Web

  • Conhecer os princípios de criação de um Web site eficiente
  • Projetar e criar páginas Web eficientes
  • Projetar diferenças tecnológicas
  • Incorporar tecnologias da Internet em Web sites
