Curso 656
Web Server Management		  

Mostra as características e recursos do Intranetware v4.11 Web Server. Aborda a instalação e configuração do Web Server para utilização na Internet ou Intranet. Mostra ainda como acessar o NDS (Netware Directory Services) via Internet, além da configuração de serviços tais como FTP e DNS.








Carga Horária: 2 dias
Pré-requisito: Curso 605
Teste: 50-710 (2 créditos)
Idioma do material: Inglês
