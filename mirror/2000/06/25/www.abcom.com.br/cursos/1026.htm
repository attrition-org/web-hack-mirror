Microsoft Exchange Server 5.5 Series - Concepts and Administration

Este curso é destinado a formação de administradores para o Microsoft® Exchange Server v5.5, ensinando fornecendo os conhecimentos necessários para as tarefas de gerenciamento e configuração. Ao completar o curso o aluno, entre outras atividades, estará apto a: • Desenvolver toda a infra-estrutura do sistema
• Escolher as estratégias de configuração e instalação de clientes
• Desenvolver as estratégias de administração de longo prazo.
• Configurar todo os recursos de segurança do sistema
• Configurar a integração de segurança com o Microsoft Windows NT®
• Gerenciar usuários, listas de distribuição e o sistema de diretório
