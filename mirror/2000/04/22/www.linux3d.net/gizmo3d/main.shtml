Gizmo3D Version 0.97 (soon) Posted Wednesday, March 15, 2000 - 03:01 by anders
We will release version 0.97 soon with support for Performer Database loader (.pfb) We need to do some extensive testing before we release it and we need help with samples etc.

If you are interested in trying it and helping us to verify it, you can email us with a request.

Gizmo3D version 0.96 Posted Thursday, February 24, 2000 - 17:17 by anders

We have now isolated the LOD memory structure from the actual height field data in the gzParametricGeometry. This has lowered the memory requirements down to 10 bytes per height value. We have also speeded the IRMA algo even further !!!!

We are also in the progress of adding more loaders now. VRML97, Performer and MultiGen OpenFlight are currently beeing developed.

3DNow support Posted Thursday, January 13, 2000 - 16:42 by anders
Gizmo version 0.95b added support for 3DNow. There were also some bugs removed in the blending render stage!

Version 0.95b shall be up in a couple of days...

Gizmo3D Version 0.95 Posted Thursday, December 16, 1999 - 08:33 by anders
Version 0.95 is here with new features like reflective recursive mirrors, portals, refractions, magnifiers , multipass textures etc. We have also improved the IRMA CAT technique even further...

We need help with demos so please contact anders@linux3d.net
