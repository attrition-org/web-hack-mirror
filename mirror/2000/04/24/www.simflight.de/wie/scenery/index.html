Szenerien Installieren im FS2000
von Matthias Brückner

Nachdem Sie die Szenerie heruntergeladen haben, müssen Sie im Windows-Explorer oder einfach im "Arbeitsplatz" zu dieser Datei gehen. In diesem Beispiel heisst die Datei "XY.zip".

Klicken Sie mit der rechten Maustaste auf die Datei und dann sollte der Menüpunkt "Extract to folder ..." erscheinen. Natürlich müssen Sie zuvor WinZip installiert haben.

Jetzt sollte das Verzeichnis "xy" erstellt worden sein. Als nächstes sollten Sie hier die Datei "Readme.txt" lesen. Finden Sie keine solche Datei, so hat es bestimmt eine andere Textdatei oder ein Word-Document das Sie über die Szenerie informiert. In dieser Datei sollte erklärt sein, wie Sie die Szenerie installieren müssen. Ist dies nicht der Fall, müssen Sie hier weiterlesen.

Hat das Verzeichnis XY zwei Unterverzeichnisse namens "Scenery" und "Texture"? Wenn ja, müssen Sie das Verzeichnis XY einfach in das "Scenery"-Verzeichnis von FS2000 hineinkopieren. Dies sollte dann im FS2000 Verzeichnis wie auf dem Bild rechts aussehen.

Starten Sie jetzt den FS2000

Gehen Sie im Menü "Umwelt" auf "Szeneriebibliothek"

Klicken Sie dort auf "Gebiet hinzufügen"

Gehen Sie im erscheinenden Dateidialog ins Verzeichnis "Scenery" und klicken Sie einmal auf das Verzeichnis "XY"

Im Feld Szeneriegebietstitel können Sie einen passenden Titel für die Szenerie eingeben.

Klicken Sie auf OK und Sie sollten in der Liste ganz oben die Szenerie XY sehen. Klicken Sie anschließend auf das grüne Häkchen um wieder zum FS2000 zurückzukehren.

Falls Sie eine Startbahn der Szenerie XY in der Flughafenauswahl suchen, ist diese meistens unter "Szenerien für FS95 und ältere Versionen" zu finden.

Matthias Brückner
mbrueckner@datacomm.ch 

Was wird benötigt?

Normalerweise ist die Szenerie in einer ZIP-Datei verpackt. Dafür braucht man WinZip (oder ein ähnliches Programm) um diese ZIP-Datei zu entpacken.
