Flugzeuge installieren im FS2000
von Matthias Brückner

ZIP-Datei Entpacken

Nachdem Sie das Flugzeug heruntergeladen haben, müssen Sie im Windows-Explorer oder einfach im "Arbeitsplatz" zu dieser Datei gehen. In diesem Beispiel heisst die Datei "XY.zip".

Klicken Sie mit der rechten Maustaste auf die Datei und dann sollte der Menüpunkt "Extract to folder ..." erscheinen. Natürlich müssen Sie zuvor WinZip installiert haben.

Flugzeug installieren

Jetzt sollte das Verzeichnis "xy" erstellt worden sein. Als nächstes sollten Sie hier die Datei "Readme.txt" lesen. In dieser Datei sollte erklärt sein, wie Sie das Flugzeug installieren müssen. Ist dies nicht der Fall, müssen Sie hier weiterlesen.

Gibt es eine ZIP-Datei oder ein Unterverzeichnis namens "Gauges"? In diesem Fall müssen Sie die enthaltenen GAU-Dateien in das Verzeichnis "Gauges" von FS2000 tun. Eine GAU-Datei ist ein Instrument das dann auf einem Instrumentenbrett von FlightSimulator erscheint. Die Instrumentenbretter sind für jedes Flugzeug einzeln im Unterverzeichnis "panel" gespeichert, die Instrumente selbst sind aber im Gauges-Verzeichnis von FlightSimulator abgelegt.

Sofern das Verzeichnis XY die Unterverzeichnisse "Model", "Panel", "Sound" und "Texture" enthält kopieren Sie das Verzeichnis "XY" ins "Aircraft"-Verzeichnis von FlightSimulator.

Starten Sie jetzt den FS2000 und schauen Sie ob Sie das Flugzeug XY in der Luftfahrzeugauswahl finden.

Viel Erfolg!

Für den interessierten Leser :-)

Damit im FlightSimulator die Flugzeuge angezeigt werden, müssen folgende Dateien unbedingt vorhanden sein:

Dateiname Zweck
Aircraft.cfg Grundinformationen zum Flugzeug. (z.B. Titel)
xy.air Flugeigenschaften vom Flugzeug. (Aerodynamik, ...)
model\model.cfg Verweist normalerweise auf xy.mdl
model\xy.mdl Flugzeugmodell (Form vom Flugzeugmodell)
panel\panel.cfg Instrumentenbrett oder Verweis auf ein Instrumentenbrett von einem Standartflugzeug.
panel\*.bmp Hintergrundbilder von den Instrumentenbrettern, nur nötig, wenn das Flugzeug ein eigenes Instrumentenbrett hat.
sound\sound.cfg Soundinformationen vom Flugzeug oder Verweis auf die Soundinformationen von einem anderen Flugzeug.
sound\*.wav Sounddateien, nur nötig, falls das Flugzeug eigene Soundinformationen besitzt.
texture\*.?af
texture\*.bmp		 Texturen vom Flugzeug, sie sind für die richtige Bemalung zuständig. Um Repaints zu machen, muss man diese Dateien verändern.

Was wird benötigt?

Normalerweise ist ein Flugzeug in einer ZIP-Datei verpackt. Dafür braucht man WinZip (oder ein ähnliches Programm) um diese ZIP-Datei zu entpacken.
