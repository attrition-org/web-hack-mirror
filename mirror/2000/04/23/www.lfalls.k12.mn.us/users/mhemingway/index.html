New and Notable:

Music Performance Package
Posted Monday, December 20, 1999
The Minnesota State Graduation High Standards Performance Package for the Artistic Performance can be fulfilled in the Mixed and Concert Choir classes. These package tasks can be veiwed in a link to the right. This package is a state mandate for graduation of students in 10th Grade and lower. Many of the tasks our covered by curriculum activities which all choir members participate in on a daily basis. The solo performance portion of the package can be fulfilled by participation in Solo/Ensemble Contest in early February, or presenting a solo, and completing a solo performance worksheet at several student recitals to be presented in the spring.

If you have any questions concerning the performance packages please feel free to contact me at the address on this page.
