If you are using AOL to view this site, you may notice some distortion in the graphics, caused by AOL's default image compression. To eliminate this problem please follow these steps:

  1. Select "Members" in the toolbar
  2. Select "Preferences"
  3. Select "WWW"
  4. Select "Uncompressed Graphics"

If the graphics are still distorted, clear the cache as follows:

  1. Return to this page (so that you don't "cache" the page again)
  2. Repeat steps 1 to 3 above
  3. Select "Advanced"
  4. Select "Purge Cache"
  5. Return to this site to view the undistorted graphics.
