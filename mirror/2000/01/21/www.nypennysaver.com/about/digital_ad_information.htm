The Pennysaver now accepts display ads digitally!
These ads can be either E-mailed or submitted on Zip Disks or Floppy Disks.

 

The following criteria should be followed when submitting these documents:

  • Ads made to Pennysaver sizes and correct if output at 100%.
  • Contact names and phone numbers on E-mail and disk ads.
  • 100% Laser proofs are required for ads that come in on disk.

DEADLINE for digital ads is 12 NOON, the FRIDAY PRIOR TO PUBLICATION DATE

File Information for Macintosh

  • Include Quark Xpress's "Collect for Output" Versions 3x and 4
  • Multi Ad Creator & Creator2 Documents
  • Adobe Acrobat PDF Files
  • Files Using "Stuffit Compression Utility"

File Information for PC

  • MS Word
  • MS Publisher
  • Adobe Acrobat PDF
  • Quark Xpress 4.0

Linked Files and Fonts

  • All placed images used in layout are supplied
  • Include both printer and screen fonts
  • Include and indicate fonts embedded in EPS files.
