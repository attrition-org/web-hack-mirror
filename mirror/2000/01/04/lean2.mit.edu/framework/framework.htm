Production System Design and Deployment Framework


Welcome to the PSDD Framework.  The following web pages explain the design system and provide examples of its implementation in industry.  You can access these pages by either clicking on the figures or using the tool bar on the bottom.   The Design component of the system is comprised of the Design Decomposition and the Design Matrix.  The Design Decomposition section relates Functional Requirements (FR's) to Design Parameters (DP's) and provides concrete examples.  The Examples section details the ideology behind these concepts.  The Design Evaluation Tool section shows how to assess the success level of a proposed design system.  The Deployment component of the system is based upon the System Design Flowchart which shows the implementation precedence of the Design Parameters.  The steps associated with the deployment of the system are explained in the Deployment Steps section.
