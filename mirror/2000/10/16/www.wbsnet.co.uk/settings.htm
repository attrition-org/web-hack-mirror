Step by Step Settings Guide

  • Ensure that the correct Windows 95/98 components have been installed.

     

    • Dial-Up Networking

       

      • Double click on My Computer
      • Double click on Control Panel
      • Double click on Add/Remove Programs
      • Select the Windows Setup Tab
      • Select Communications
      • Click on the Details buttons
      • Check that the Dial-Up Networking option has been selected, if not make sure that you tick it.
      • Click OK
     
    • Network Settings

       

      • Double click on My Computer
      • Double click on Control Panel
      • Double click on Networking
      • Make sure that you have a Dial-Up adapter, Client for Microsoft Networking and the TCP/IP protocol
      • If any on these are missing then you should add them by clicking on the Add button and then selecting the correct option according to the type.
      • Client -> Microsoft -> Client for Microsoft Networks
      • Adapter -> Microsoft -> Dial-Up Adapter
      • Protocol -> Microsoft -> TCP/IP
      • You may be required to reboot and/or insert your original Windows 95 media

       

  • Setting up a Dial-Up connection

     

    • Double click on My Computer
    • Double click on Dial-Up Networking
    • Follow the screens and provide the information that is requested, including name of connection and the phone number to be dialled. These settings may be obtained from our settings page.
    • When you have finished you must make sure that the TCP/IP settings are correct. Do this by right clicking on the new connection that you have created and select properties.
    • Click Configure and then General
      • Set the maximum speed to 115,200
    • Click on Connection and then Advanced
      • Make sure that 'Use Error Control', 'Compress Data' and 'Use Flow Control and Hardware' are all ticked
      • Click on Server Type and select 'PPP, Windows 95, Windows NT and Internet'
    • In the advanced options
      • Uncheck 'Log on to Network'
      • Tick 'Enable Software compression'
      • Uncheck 'Require Encrypted Password'
    • In 'Allowed network protocols'
      • Tick TCP/IP
      • Uncheck NETBEUI
      • Uncheck IPX/SPX
    • Click on 'TCP/IP Settings'
      • Tick 'Server assigned IP addresses'
      • Tick 'Specify Name Server addresses'
        • Use the settings page to confirm the DNS settings
      • Tick 'Use IP header compression'
      • Tick 'Use Default gateway on remote network'
    • Click 'OK'
    • Click 'OK' again to confirm all changes
  • Using your Dial-Up connection
    • Double click on the icon in the Dial-Up Networking folder that you just created
    • Enter the correct Username and password when prompted
