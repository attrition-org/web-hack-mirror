DEF CON FTP Archives
All files are located on the DEF CON FTP server at 207.173.52.18.  Log in as anonymous.  If you are having problems using your browser to log in, try an ftp client program like wsftp, etc. Some FTP sites are only partially mirrored.  Please visit the original FTP or web site for their complete files.
Enough sites have come and gone over the years that I wanted to create a  FTP repository to preserve the better sites out there, as well as help distribute their content.  If you want your site mirrored here, please email the FTP-Archives account.
