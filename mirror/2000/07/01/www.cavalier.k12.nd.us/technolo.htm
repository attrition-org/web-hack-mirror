April 11, 2000
Intel Announces Software Giveaway By Chris Hager
    Computer maker Intel Corporation on Monday will announce plans to give away advanced security software on the Internet. The Intel software uses a set of industry-standard security functions known as the Common Data Security Architecture. The software includes encryption features and tools to manage digital certificates, which guarantee the identity of Internet users.
    Intel will begin offering the security software, as well as the program code for the package, for download by May 15.
