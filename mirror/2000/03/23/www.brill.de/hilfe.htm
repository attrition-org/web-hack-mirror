Die folgenden Symbole sollen das Wechseln zwischen den Seiten erleichtern:

Zurück zur Homepage

Zurück zum Kapitelanfang

Zurück zur vorherigen Seite

Weiter zur nächsten Seite

Diese Symbole finden Sie immer in der rechten, oberen Ecke der Seiten, auf denen sie sinnvoll erscheinen. Nur in diesem Beispiel verzeigen alle Links dieser Symbole zurück auf die Homepage.

Weitere Symbole weisen Sie auf vertiefende Informationen oder ergänzende Dokumente hin:

  • separates Bild (Ladezeit!) zu diesem Thema
  • Dateien zum Laden (z. B.: Software)
  • Filmsequenz (ofmals besonderer Viewer erforderlich!)
  • Hilfestellung finden Sie hier
  • Menü: wählen Sie das gewünschte Thema
  • Telnet-Session
  • Textdatei (.TXT oder .DOC)
  • Ton zur aktuellen Seite

Im Fuß jeder Seite finden Sie einige nützliche Adreßinformationen. Zunächst einmal den Dateinamen der Seite (URL). Sollten Sie Probleme haben diese Seite erneut zu laden, so verwenden Sie ausschließlich Kleinbuchstaben und lassen Sie das "l" von html weg. Darüber hinaus enthält diese Zeile das Datum der letzten Änderung dieser Seite. Hinweis: Lösen Sie gelegentlich die Funktion RELOAD Ihres Browsers aus, um sicher zu sein, daß Sie wirklich die aktuelle Version unserer Seite sehen. Da unser Forum sehr zeitnah gepflegt wird, kann es Ihnen anderenfalls passieren, daß Sie eine "alte" Seite aus Ihrem lokalen cache oder einem Proxy erhalten.

Eine weitere hilfreiche Information am Fuße jeder Seite ist der Name des Verfassers. Klicken Sie auf die E-Mail-Adresse, um schnell und einfach eine Nachricht an den Verfasser zu senden. Alle Nachrichten sind uns willkommen. Wir sind stets um prompte Antwort bemüht. Hinweise auf eventuelle Fehler oder sonstige Handicaps unserer Seiten an die Adresse des jeweiligen Autors sind uns ebenso immer willkommen.

Auf der Titelseite finden Sie das Datum der letzten Änderung über einen Link mit der Seite "Neues im Forum" verknüpft. An dieser Stelle sehen Sie, was sich an dem genannten Tag im Forum alles geändert hat.
