Understanding the live data

The live data has 6 columns.  Not every entry uses all six columns. 

Column 1:
month/day/year  (not Y2K complient!)

Column 2:
hours:minutes

Column 3:
event:
   Opening - Connecting with Speak Freely
   Opening_RA - Connecting with RealAudio
   Busy - Failure to connect because the receiver is in the middle of a frequency change
   Closing - Closing a Speak Freely connection
   Locked_frequency - This frequency is not allowed. It is either outside the range of the radio or it has been specially disabled.
   Denied - This IP address is not allowed to use the Listening Post

Column 4:
frequency in megahertz

Column 5:
mode:
   FMn - narrowband frequency modulation
   FMw - wideband frequency modulation
   USB - upper sideband
   AM - amplitude modulation

Column 6:
IP address of computer making request.
