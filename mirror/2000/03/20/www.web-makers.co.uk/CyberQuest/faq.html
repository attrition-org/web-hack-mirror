What is a domain name ?

The Domain Name System (DNS) was introduced to associate easily remembered and convinient names, such as your company name, with the corresponding internet protocal addresses. Computers actually identify internet services by numeric internet protocol addresses. These comprise four numbers, each in the range 0 - 255, separated by dots.

  • for example: 195.188.165.50

Domain names are often confused with e-mail addresses, or references on the internet known as Universal Resource Locators (URLs). A domain name is a key part of both of these.
The domain name on which your e-mail or web site relies may be registered to either yourself or to CyberQuest.

How can I check if a domain is available ?

You can use our online search form below to check if the domain that you want is still unregistered. If the domain that you were looking for is no longer available try altering it slightly.
for example:

  • if "car.co.uk" isn't available try "cars.co.uk"
  • similarly you could alter the suffix using "car.net" or "car.com" instead of "car.co.uk".

Why is it nesessary to renew domain name registrations ?
Domain names need to be re-registered every two years to ensure that old names or names that are no longer being used can become available to someone else.
Currently 'Nominet' charges &pound;20 for the re-registration fee.
