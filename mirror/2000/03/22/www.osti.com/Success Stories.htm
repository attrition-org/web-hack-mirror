Support Sentinel Suite of Tools & Services

Analyst Incident Manager (AIM)

A Microsoft Windows and Oracle RDBMS based application, providing a GUI to a product and customer exception tracking system that centralizes and automates the task of information gathering and rapid response, for NCR Customer Services support sites around the world.

The AIM application features an on-line database of support activity, automatic exception notification, customer service entitlement authorization, customer history information, and step-by-step tracking of exception activity.

Developed in an object oriented fashion using a Model/View/Controller paradigm, Microsoft C++ 8.0 and Microsoft MFC class library for Windows; AIM also provides a home-base for interacting with NCR Support Sentinel's many tools and services. A Client/Server model is incorporated for interaction with UNIX based servers providing Sentinel's services.

The Human Interface (HI) component fulfills the client side of the model through DDE connections to NCR Windows. NCR Windows, in turn, provides a transport layer with a companion NCR Window’s process on the UNIX server.

 

  InfoServ

A worldwide distributed document storage and retrieval system featuring Fulcrum’s Ful/Text document server. The InfoServ application stores product documentation and customer support case histories on-line, facilitating sophisticated high-performance information retrieval for a broad base of NCR Customer Services personnel, substantially increasing their productivity and adding tremendous value for NCR customers.

While providing its users with a vast amount of information at their fingertips, the InfoServ Ful/Text retrieval engine simplifies the process of locating needed information by allowing documents to be rapidly searched using keywords, keyword phrases, wildcards and proximity queries.

To accommodate a diverse installed base, the application has been written to operate on NCR's Tower and newer 3000 products under the UNIX operating system. Connections are provided via high-speed TCP/IP and RS-232 dial-up communications.

 

  InfoServ Authoring Tool

An InfoServ document editing utility using Microsoft's Word for Windows 2.0 macro and template capabilities. This tool reduces effort and improves accuracy by assisting the user in creating and formatting documents to be loaded into the InfoServ Ful/Text document collection.

 

Automated Information Update (AIU)

A facility used in conjunction with the NCR InfoServ product, this product provides the distribution mechanism, collecting all newly added or updated information from individual sites around the world. The information is subsequently redistributed to all other sites during the next 'cycle'.

Using the AIU facilities, all Customer Support sites are kept updated with new information from all other sites worldwide within a 24-hour turnaround, substantially reducing problem rediscovery and increasing the accuracy and currency of information available to Customer Service personnel.
