OSTI is not primarily a Product-Oriented company. But we have worked with a number of organizations looking to move (or gain access to) their applications/data located on older NCR legacy mainframes to newer (open) systems.  Thereby affording themselves to take advantage of the competitively-valuable features available on the newer systems.

During this process, as with any true Object-Oriented process, we created some key common tools which could be applied to each of these sites.

From these tools the OpenLegacy Suite of Products were created, specifically for the NCR 9800/8000 mainframe user.  Those products are listed below:

NetworkConnect

NetworkConnect is a reliable, high-speed messenger working between a VRX/E system and any node on a network. Using NetworkConnect you have the opportunity to:

  • Develop new (Microsoft Windows & DOS-based) applications.

  • Interface-in your current applications.

to interact with applications on your mainframe system.

DataConnect

DataConnect was designed to establish a reliable, high speed communications link with Open Systems and the VRX/E TOTAL Database & RMS Files. Using DataConnect you have the opportunity to build new (MS Windows & DOS-based) applications (using our ODE API) to access(Read-Write) your legacy data.

File Transfer Utility

The File Transfer Utility provides the ability to copy files from a VRX host to a UNIX server, and to copy files from a UNIX server to a VRX host.
