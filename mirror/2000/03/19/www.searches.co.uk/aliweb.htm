Aliweb is a unique robot because it uses a very detailed description of the website provided by the website owner in a file called site.idx. This file contains a precise definition of the website. It may describe only one or two of the principal pages or it may contain descriptions of every one of hundreds of pages on the website. Aliweb is the only robot which updates all of its sites on a nightly basis. Because the submission process is rather precise and complicated, the Aliweb database is relatively small, but for the same reasons it usually contains sites which are professionally maintained and virtually none of the amateur "home pages". Since the website pages are also categorized by keywords provided by the site owner, it can be searched with a very high degree of precision and reliability. Aliweb is so popular that, in spite of having several mirror sites, it is often slow or too busy to respond to a request. Highly recommended for accuracy. Best used in off hours or quiet periods.
