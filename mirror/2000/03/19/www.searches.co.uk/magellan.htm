Magellan is one of the few major search engines which provides ratings and reviews of websites. This is a tremendous value to the web surfer because it helps one to focus on the websites which might be the most useful. Magellan has a simple one line entry that allows users to add the search capability to their own page. Magellan has a hierarchial directory which is similar in structure and form to Yahoo. Updates to this directory from submissions seems to take a very long time. Reviews take an even longer time. Date of entry are not published. Magellan's frequently Asked Questions.
