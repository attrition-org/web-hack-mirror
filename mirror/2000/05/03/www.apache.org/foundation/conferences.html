Apache Conferences

The Apache Conferences Project is happy to announce ApacheCon 2000. Scheduled for March 8-10, 2000 in Orlando Florida, AC2K is the first official Apache Software Foundation Conference. Check out the site for more information.

Third-Party Conferences

The O'Reilly Open Source Software Convention

O'Reilly and Associates is hosting an Open Source Convention in Monterey, California, August 21st-24th, 1999, which includes a set of conference tracks focused on Apache and Apache-related software. Sessions and tutorials on Apache 2.0, Apache JServ, mod_perl, PHP, security, load balancing, security, and much much more are included, and many of the Apache core developers will be presenting.
