NEXTLINK Will Use Equinox

Billing World
April 2000

Equinox, using its ShowCDR Deluxe system, plans to modernize Nextlink Communications Inc.'s 30-switch network. ShowCDR is a menu-driven system designed to let carriers extract data from CDRs using search techniques on any field configuration or combination of values. A report generator lets users create customized call reports by selecting and sorting CDRs, which can also be exported to any spreadsheet or database application.
