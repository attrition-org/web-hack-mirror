At Corel, we believe in value, compatibility and performance and this is enshrined in our corporate mission statement. We have always believed consumers should be able to use their favorite application on the operating system of their choice. This philosophy has led Corel to become one of the industry’s top supporters of the Linux® operating system.

Corel® LINUX® OS
Experience Linux® power designed specifically for the desktop with Corel® LINUX® OS, the Debian-based operating system that combines renowned Linux performance and stability with intelligent simplicity. Featuring an incredible four-step graphical installer that automatically detects most PCI hardware, Corel LINUX OS is easy to install and easy to configure. Enjoy a drag-and-drop environment and easy-to-use graphical Control Center with the enhanced KDE-based Corel Desktop. Access local and network drives and the Internet with an innovative browser-style file manager, and easily obtain system updates over the Web. With outstanding file compatibility and network integration, Corel LINUX OS is an exciting development in operating system evolution. For more information on Corel LINUX OS and other Linux products and development initiatives from Corel, please visit Corel’s Linux community Web site.
