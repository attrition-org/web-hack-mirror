What is Impact :

Impact 1.x works with CRIS (The County Road Information System).

Impact 1.x builds your collision diagrams in Autocad or Intellicad

 Impact diagrams are geometrically similar to your actual intersection, since they are based on your Reference Point information. 

 Impact diagrams use symbols based on WSP collision coding standards 

 Impact diagrams can be modified, saved, or plotted using familiar Autocad (Intellicad) commands.

Improvements in Version 1.1 include :

  • Better support for saving and managing diagrams (Diagram Save and Diagram Open).
  • Better control over the default seed drawing (Diagram Initialize).
  • More reporting options, including detailed data theme quick reports on a diagram by diagram basis.
  • Runs in Autocad 13, Autocad 14, and Intellicad 98.

Improvements in Version 1.2 include :

  • All enhancements described in Version 1.1
  • Fix Version 1.1 bug where collision records are repeated in the summary report
  • Fix Version 1.1. bug where Reference Point only has one road name entered
  • Provide options about symbol and text scaling and text positioning
  • Select from Reference Points by road names (alphabetically) or by Reference Point (Numerically)
  • Collision index numbers are now plotted at the symbol when collisions/symbol is set to one
  • Option to group collision symbols by severity (property, injury, fatality)
  • New midblock reporting feature in CRIS can synthesize Reference Points for Impact plots between intersections (road segment analysis)

Looking ahead to Version 2.0

  • Manual input for intersection and collision records

  • Compatible with Mobility

The Impact Interface

Start using Impact today here's how :

  1. Download Impact Version 1.2 using this link.
  2. Unzip the downloaded file into its own folder on your computer (ie C:\IMPACT).
  3. Follow the instructions in the README.TXT file found in the ZIP.
  4. Send your comments, suggestions, and bugs to John Donahue at the County Road Administration Board.
  5. *Intellicad Warning : Impact 1.1 is written in Autolisp and Intellicad's present Autolisp interpreter is VERRRRY SLOWWWW. We're calling it a "prototype" version for this reason, and look forward to improvements in Intellicad's lisp support in the near future.
  6. *Intellicad Warning : Impact 1.1 for Intellicad does not support Diagram Save and Diagram Open at this time. All collision graphing and reporting features are available. Use Intellicad Save and Open commands instead.
