Programming In Arabic under Windows

  1. Visual Basic 4.0 Programming In Arabic On Win3.1/Win95 Arabic Editions
  2. Arabic Programming in C/C++. This is just an introduction to some basic techniques, which you might find useful .
  3. Optima++ from Powersoft. This compiler is based on the Watcom C/C++ compiler. This product is worth some attention. It also includes the Sybase SQL Anywhere engine.
  4. Borland C++ Builder. This product is impressive. Borland also offer Delphi 2.0 , and its companion Arabic Enablement which offers completely arabised controls for database, menus, labels and more. Borland also offer Free Downloads .
  5. Open Architecture Contextual Analysis (OAC). This is a new technology that promises Arabic text of calligraphic quality with cross-platform compatibility. Currently works under Win 95, Win 3.x and OS/2 WARP. In addition, the system is compatible with current technologies, and is independent of the actual codepage used. A demo editor to illustrate its capabilities is currently under development.
