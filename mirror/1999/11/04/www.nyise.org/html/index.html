Interested in developing a website?. This page lists what I consider to be the best tools, utilitites, and software for web site development. Most of these programs are available as shareware so you can test them for free. I've also added links to places for locating graphics for your pages and sites that will inspect your html code for errors. Site development is fun. Many Internet service providers provide free web page space or make it available very cheaply. Go ahead.express your self!!!

     
 

 

HTML Assistant Pro 97

My HTML editor of choice is
Pro 97 for Windows95/NT by Brooklyn North. This version has dozens of advanced features such as:

Long File Name Support, Word Processor File Converter, Image Mapping, Site Assistant for uploading, Spell Checking, Quick Frames, Background Assistance and much more.

There is a Windows 3.1 version.

GIF Construction Set

A quick way to create transparent, interlaced and animated GIF files for your web page.

If you are a Mac user the program of choice is GifBuilder.

GIF Animation on the WWW
Can be a lot of fun and is easy to learn. Just remember to keep the files as small as possible by using small components and optimize your palettes.

Another excellent program is PhotoImpact Gif Animator.
