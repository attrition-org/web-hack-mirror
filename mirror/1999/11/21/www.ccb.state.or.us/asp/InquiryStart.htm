CONSTRUCTION CONTRACTORS BOARD

CONTRACTOR INQUIRY

The CCB's database may be accessed to obtain general information on  construction contractors.

A query may be done by:
Registration Number (format: 1234567)
Telephone Number (format: 3784621) DO NOT INCLUDE AREA CODE or PUNCTUATION.

To search for a contractor, enter the criteria (registration number or telephone number) in the text box, then press the button for the Type of Search you want to perform.   If you use a registration number for the criteria, use the "Search by CCB Number" button.  If you use a telephone number for the criteria, use the "Search by Phone Number" button.

*If you are a first time user, PLEASE  View the Inquiry ReadMe file for more details.
