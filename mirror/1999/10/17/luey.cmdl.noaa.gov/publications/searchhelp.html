Search Type Instructions Examples
Author: At least part of last name is required.  The first and middle initial is optional. Harr

Harris

Harris, J.

Harris, J. M.
Title of Publication: Type in words found in publications title. Typing the word "Atmosphere" will return any publications that contain that word in the title.

Typing the word "Atmos" will return any publications containing that character string in the title.  For example, atmosphere or atmospheric.
Abstract: Type in words found in the publication's abstract. Typing the word "Atmosphere" will return any publications that contain that word anywhere in the abstract.
Source of Publication: Type in words found the publication's journal, book, etc. Typing in the word "Geophysical" would return all publications from journals containing the word "Geophysical."
Content: Type in keywords pertaining to the publication's content. Typing in "ozone" might return publications concerning the South Pole ozone hole.
CMDL Group: Select the group primarily responsible for the publication. Publications are listed under the first CMDL author.
Year of Publication: Select the year that the publication was printed. Publications are available for 1992 onward.
