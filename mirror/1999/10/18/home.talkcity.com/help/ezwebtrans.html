  EZ WebTransfer Help
 
  Need help with copying files from another web site to your Talk City site? You've come to the right place! If you'd like to go straight to EZ WebTransfer, click here.
  Q How does EZ WebTransfer work?
  A You give EZ WebTransfer a starting point, either a directory or file somewhere on the web (for example, your home page on another site like Geocities or Tripod). EZ WebTransfer then does a "spider crawl" through all the links it finds. It will copy all the files it discovers (web pages, images, sound files, etc.) during this "crawl," as long as they are within the source directory and its subdirectories.

EZ WebTransfer will also make any necessary changes to your files so all the URLs will still work after they reside in Talk City.

  Q What are all those fields in the EZ WebTransfer form?
  A Nickname:  This is your Talk City nickname.
Password:  This is the password that you chose for yourself during enrollment.
Source URL:  This is the starting point of the files to be copied from your old web site.
Destination subdirectory:  (Optional) This is the destination in your Talk City web site where you want the files copied to. If you leave this field blank, then the files will be copied into your main home page directory.

For example. Imagine that your Talk City nickname is "blue_lizard" and you want to copy your old web site from Tripod to Talk City. You would fill out the form like this:

Nickname:  blue_lizard
Password:  my-super-secret-password
Source URL:  http://members.tripod.com/~blue_lizard/
Destination subdirectory:  (leave this blank.)

A note on the field "Source URL":

In the above example, you could have used http://members.tripod.com/~blue_lizard/index.html as the starting point and the transfer would still work Ð see (Default File Names) below.

A note on the field "Destination subdirectory":

The Talk City web site "blue_lizard" in the example above is home.talkcity.com/EaselSt/blue_lizard/. If you fill out the EZ WebTransfer form with an empty destination subdirectory, then everything will be copied to home.talkcity.com/EaselSt/blue_lizard/.

If you want to copy everything from my old blue_lizard Tripod site into it's own folder called "old_stuff" in your Talk City site, then you would type old_stuff into the Destination subdirectory field above. EZ WebTransfer will then copy all of the files to home.talkcity.com/EaselSt/blue_lizard/old_stuff/.

Warning! If a file on your Talk City site has the same name as a file being copied from your Source URL, that file will overwrite your Talk City file. So be sure you rename the Talk City file before you start the transfer!

  Q What gets translated?
  A When EZ WebTransfer copies your files to Talk City, it tries its best to update the HTML links in your .htm and .html documents to point to your new document locations within your Talk City site. It does this by searching through and fixing up the following tags: <a> <area> <body> <frame> <form> <img> <link>. EZ WebTransfer pretty much ignores the contents of HTML comments. It will skim through some comments to remove the JavaScript associated with advertisements and watermarks. It will not touch other JavaScript or Java.
  Q What gets added?
  A A Talk City masthead will get added to each .htm or .html file. This will contain the street name, some navigation tools, and a banner ad (to keep your Talk City home page available at no cost to you).
  Q How do I avoid overwriting files?
  A If you there's a file in your Source directory that has the same name as a file in your Talk City destination directory, the Talk City file will be overwritten! For this reason, make sure you any Talk City file you want to keep has been renamed before you do the transfer.
  Q How much will be transferred?
  A EZ WebTransfer will only transfer the material that it finds through HTML links in your pages. If you have content that is not entirely linked together, you may have to transfer it in parts.

Currently we limit each Talk City web site to 12MB. At most, EZ WebTransfer will copy 12Mb of content.

Warning: If your reach your 12MB limit, EZ WebTransfer will stop transferring, even if it is in the middle of copying a large file.

EZ WebTransfer has a session time limit. You may find that if you are copying from a very slow site, that the EZ WebTransfer session will time out. The timeout limit is currently set to 20 minutes.

  Q I was copying a bunch of files, and the transfer stopped. I'm sure I didn't reach the 12MB limit. What happened?
  A EZ WebTransfer has a session time limit. You may find that if you are copying from a very slow site, that the EZ WebTransfer session will time out. The timeout limit is currently set to 20 minutes.
  Q What if I have special characters in file names?
  A Different web sites have different schemes for naming files. Not all web sites allow the same set of characters in their file names. As EZ WebTransfer copies files from your old site, you may find that some files are renamed to be consistent with our web site scheme. Any character that our site does not accept will be replaced by a hyphen ( - ). For example if your old site had a document with a space in the name, like "my page.html", it would be renamed "my-page.html" on Talk City.

Default file names: The default file name at Talk City is index.html. That means if you point to someone's Talk City home page directory, like http://home.talkcity.com/EaselSt/blue_lizard/, the file named index.html will be displayed.

On some servers, the default file name is default.htm or home.html. Whatever the default file is in your source directory, we will copy it over and name it index.html.

Be aware that since we will copy over any file within your source directory that your web pages explicitly point to, it's possible we'd transfer a second copy of your default file but with its original name (if you directly link to that file in your pages).

  Q What if I don't want to transfer my whole site, or not all of my site is linked together?
  A You can run EZ WebTransfer multiple times with different source URLs and destination subdirectories. If you run EZ WebTransfer many times in the same directory, you may find that it will overwrite files with the same name. This is not a problem if, for example, you have many links to a home page file like index.html.

In a given session, you should find that EZ WebTransfer copies each file exactly once.

  Q What kind of content is allowed?
  A Talk City is a family place. We tolerate and encourage all kinds of points of view. We expect tolerance and respect from everyone that is a member of Talk City. Generally, try to be nice to each other and don't go out of your way to make someone mad. It just isn't worth it.

Keep the content in line with our Code of Conduct. Only copy material from sites that you own or have the rights to. Have fun and keep it light.

Note: If you believe that someone has illegally copied material from your site, please send email to homepages@corp.talkcity.com. Include the appropriate URLs in your correspondence. If one of our members is found to be in violation of our Home Page Standards, their account and web site may be immediately removed.

  If you have problems with this tool, please send email to homepages@corp.talkcity.com.

If EZ WebTransfer is having trouble transferring your web pages and images from your previous web site, please be sure to include the source URL in your email so that we can test (and possibly fix) the EZ WebTransfer tool.
