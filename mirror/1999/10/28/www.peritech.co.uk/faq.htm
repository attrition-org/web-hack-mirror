The main job of the name server is to provide information about a Domain or a Host. Every Host needs to know how to get to one place only.
At the top level is root. Underneath are net, org, com, edu, mil, int and gov; beneath these are the various Hosts. These name servers exchange data and initially new zone files were built on a daily basis to reflect the growing nature of the Internet. Soon more top level Domains were added including all country codes. The DNS is now a truly distributed system with servers spread throughout Europe and the rest of the World.

A Proxy Server is a dedicated machine which is set up to provide the connection between a company's LAN and the Iinternet.
It is set up so that when each client on the LAN wants to browse the Internet or pick up mail etc. a request is sent to the Proxy Server which carries out the task and sends the results back to the client making the request.
This is an excellent way to minimize configuration of each computer on the LAN and it serves as a very good security measure allowing the corporate LAN to keep its IP addresses invisible to the outside world.

Answer:

With a hardware RAID configuration, all the computer sees is another single device. As far as the computer and its operating system is concerned it just has one big hard disk connected, this is because the hardware raid unit is self contained with disk management software, the disk controllers and the disks enclosed in one box. This box plugs into an available port or slot on the computer which is fooled into thinking that it is just a single device.

With a software RAID configuration the computer will see each disk as an individual device so for example if a computer has 6 disks plugged in, then the computer will see 6 disks.
To set these up as a single RAID device, special RAID management software has to be purchased and installed and then configured to instruct the computer's operating system to treat these disks as a single RAID device.
