How to Navigate the Product Documentation

Welcome to the Internet Information Server Product Documentation.

Each page of this documentation provides its own navigation bar separate from your browser’s toolbar. You can use the navigation bar at the top and bottom of each page to browse the documentation:

  • Click Contents to see a list of chapters, with links to each chapter.

  • Click Index to see a list of terms, with links to information about each term.

  • Click the left and right arrows to jump to the previous or next chapter.

  • Click the up arrow to return to the top of the current page.

No matter where you are in the documentation, you can use the navigation bar to return quickly to the Contents page or the Index.

A note about your browser   The browser navigation tools work on any Web page including these pages, but the documentation navigation bar works only for the Internet Information Server Product Documentation.
