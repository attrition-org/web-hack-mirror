PC Software - WINDOWS® 95/98/NT

SPOTCHECK's state-of-the-art Visual Basic 32 bit operating system software programs are powerful yet easy to use. Available for both single-user PCs and network systems, they operate on computers running WINDOWS 95/98 and NT Workstations.

SPOTCHECK-VB PC Software includes:

  • Immediate electronic access to SPOTCHECK's multiple payors
  • Online eligibility information responses are dependent upon the information made available by each payor

    Future enhancements to the SPOTCHECK-VB PC Software includes:

  • Credit card functions
  • End-of-day credit reporting
  • Storage of patient demographic for easy recall for future SPOTCHECK use
  • Import and export of data from Spotcheck to other programs utilizing common WINDOWS routines

SPOTCHECK-VB for WINDOWS NT requires version 4.0 using service pack 3 or higher.

There are no special requirement for Windows 95/98 systems. The system must have a modem attached and a printer is recommended.
