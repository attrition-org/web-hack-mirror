Linux Certification untuk para Profesional

Artikel terkait:
Linux Certification Jalan Terus.

4/30/99; 5:00 PM EST
By Kevin Merrill ,
Computer Reseller News

Sebuah perusahaan pelayanan berbasis di San Fransisco www.linuxcare.com yang khusus menangani Linux menargetkan sebuah program sertifikasi dalam bentuk kursus. Para technical professionals Linux dapat memperoleh sertifikasi tersebut.

Program dalam bentuk courseware itu akan ditawarkan melalui para penyelenggara komputer training.

Program yang disajikan adalah murni Linux dan tidak menitik beratkan kesalah satu versi distributor yang ada dipasaran.

Linuxcare merencanakan untuk bekerja sama dengan Linux Professional Institute (LPI) sebuah lembaga yang baru-baru ini dibentuk untuk menyusun standar dan kurikulum.

LPI didirikan dari gabungan berbagai distributors terkemuka seperti RedHat, SuSE, Caldera dsb.-nya.

Oracle menyentuh
e-business

By Timothy Dyck
April 27, 1999
PC Week

Oracle8 bukan hanya database saja -- melainkan juga server untuk file dan applikasi. Program ini yang cenderung didominasi oleh Java dan Internet ketimbang data-base belaka.

Beberapa hal yang menarik dalam versi 8i adalah kemampuan yang dijanjikan sebagai network file server berikut native-code Java compiler.

Oracle8i tersedia untuk berbagai platform termasuk Windows NT, Solaris dan LiNUX.

Pemogram Java atau yang harus menggunakan teknologi Java akan bergembira akan Oracle8i yang langsung mendukung itu.

Produk ini telah diluncurkan pada bulan April 99 [at Shangri-la Hotel] secara besar- besaran.

Harga lisensi mulai dari sekitar US$ 1.500 sampai puluhan ribu dollars sesuai dengan besarnya jaringan.

NSI merencanakan  Direktori baru untuk bertahan

4/30/99; 3:00 PM EST
By Madeleine Acey, TechWeb

LONDON -- Pemegang registrasi top level domain Network Solution Inc. (NSI) harus memulai hidup baru karena monopoli registrasi nama seperti .com, .org, dan .net yang disandangnya itu berakhir sampai April 99. Mulai Senin awal Mei 99 lima pesaing lainnya termasuk AOL, ISP terbesar di dunia akan aktif. Banjir pesaing lainnya akan masuk mulai bulan Juli mendatang.

NSI merencanakan untuk meluncurkan jasa direktori baru dalam bulan Juni dengan nama dot com dan mengklaim jauh lebih efisien dari search-engine yang sudah ada.

Pencarian data lewat dot com dapat dilakukan berdasarkan nama perusahaan, produk atau nomor telepon. Data yang akan dipakai diduga berbasis pada database NSI yang kini sudah berisi 4,2 juta data pelanggan.
