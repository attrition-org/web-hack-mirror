CHAT: Linus Torvalds, 9 Peb. 1999

Subject: Linus Torvalds 2-9-99
From: Chris Donohue (MSNBC)
Host: MSNBC
Date: Tue Feb 9 17:37:29

MSNBC Chat Auditorium: http://chat.msnbc.com

Host Vickie-MSNBC says:
Malam ini nge-Chat bersama – Linus Torvalds –  seseorang dengan ranking sekelas bintang seperti Michael (Jordan), atau Bill (Gates) – adalah orang dibelakang Linux, sebuah ‘open source operating system’ yang sensasional itu.

http://www.msnbc.com/news/236289.asp

Host Linus_Torvalds- says:
Saya siap untuk mulai.

[1] Host Chris_MSNBC says:
Robert Hoopes: Apakah ciptaan Linux itu direncanakan sebelumnya?

Host Linus_Torvalds- says:
Tidak, sudah pasti tidak. Dimulai sebagai proyek pribadi kemudian berkembang luar biasa.

[2] Host Chris_MSNBC says:
Reed H. Petty: Apakah ‘top 3 goals’ untuk Linux selama 12 atau 24 bulan mendatang?

Host Linus_Torvalds- says:
Saya sebenarnya tidak memiliki ‘goals’ jangka panjang. Saya ingin ber-reaksi terhadap pasar dan melihat apa yang ingin dilakukan oleh pengembang lainnya.

[3] Host Chris_MSNBC says:
Terence C. Haddock: Langkah apa yang dapat dilakukan sebuah pengembang untuk membuat software mereka ‘open source’ atau ‘free’ tapi masih mungkin menghasilkan uang untuk keperluan itu?

Host Linus_Torvalds- says:
Sudah tentu sangat sulit untuk menghasilkan uang. Ia dapat membuka peluang untuk bisnis dalam bentuk lain. Dengan menggunakan infrastruktur ‘open source’ yang ada kemudian menambahkan ‘propierty modules’ diatasnya.

[4]Host Chris_MSNBC says:
Jeff says: Menurut Anda berapa lama dibutuhkan untuk ‘open source programs’ agar dipakai secara luas oleh masyarakat umumnya [mainstream public]?

Host Linus_Torvalds- says:
Itu telah terjadi, hanya kebanyakan ‘mainstream people’ itu tidak menyadarinya.

[5]Host Chris_MSNBC says:

me - Pernahkah anda bertemu dengan Bill Gates? Apa yang anda katakan? Apa yang ingin anda katakan kepadanya?

Host Linus_Torvalds- says
Saya tidak pernah bertemu dengan dia dan saya kira kami tidak akan bicara banyak tentang itu. Kami sama-sama memiliki ‘operating systems’ tapi dia adalah orang marketing dan saya orang teknik.

[6]Host Chris_MSNBC says:
Jeff Moser: Saya seorang programer berumur 16 tahun, pertanyaan saya kepada anda adalah: Apabila anda sekarang ini umur 16 lagi, apa yang akan anda program.

Host Linus_Torvalds- says:
Kembali ketika saya berumur 16 saya membuat program permainan [games].

[7]Host Chris_MSNBC says:

Jesus Couto: Apakah anda takut dengan ‘code fork’? Perpecahan seperti Net/Open/Free BSD?

Host Linus_Torvalds- says:

Nope. Saya kira lisensi itu tidak akan kondusiv untuk memisahkan kode basis.

[8[Host Chris_MSNBC says:

Ricdude: Bagaimana caranya meyakinkan kualitas dari software menggunakan model pengembangan ‘open source’?

Host Linus_Torvalds- says:

Kualitas cenderung menjadi penting bila developer merasa yakin dengan kode-nya, itu akan menjadi sesuatu yang dapat dibanggakannya.

[9]Host Chris_MSNBC says:

CyberYoda: Berapa persen dari ‘kernel’ adalah masih sama seperti yang semula anda release?

Host Linus_Torvalds- says:

Pada dasarnya tidak ada lagi. Original release terdiri dari kira-kira 10.000 [10 ribu] baris dan hampir semua sudah ditulis ulang. Sekarang jumlahnya sudah mencapai 15.000.000 [15 juta].

[10]Host Chris_MSNBC says:

Worshiper: Apa yang dapat anda sampaikan tepatnya yang anda kerjakan di Transmeta?

Host Linus_Torvalds- says:

Saya tidak dapat mengatakan apa-apa.

[11]Host Chris_MSNBC says:

Worshiper: Apakah semuanya ini bermanfaat? Apakah anda tidak pernah berkeinginan untuk menjadi orang biasa saja?

Host Linus_Torvalds- says:

Itu sangat bermanfaat. Kadang-kadang saya sangat ‘stress’, tapi itu dapat saya pastikan sangat bermanfaat.

[12]Host Chris_MSNBC says:

Gdog GdogGd: Apakah Bill Gate takut dengan anda dan perusahaan anda? Apa yang dapat diharapkan oleh konsumen dari sistim anda dibandingkan dengan Windows 98?

Host Linus_Torvalds- says:

Saya kira Bill Gate tidak benar takut, ia hanya mengikuti perkembangan dari itu. Dia sekarang tidak berada dimana dia harus melek setiap malam.
Untuk apa yang diharapkan dari Linux, itu tergantung kepada apa yang anda ingin lakukan. Sebuah sistim yang tak bergantung dan lebih stabil.

[13]Host Chris_MSNBC says:

Josh Bell: Apa pendapat anda mengenai proyek BSD dimana sekarang mulai mendapatkan dukungan, apakah anda pikir aka mengambil jejak Linux? Satu persatu orang mengetahui tentang ‘free distribution of BSD?’

Host Linus_Torvalds- says:

Saya sebenarnya berpendapat bahwa mereka telah menyimpang ketimbang ‘comeback’. Saya pikir proyek mereka cukup menarik.

[14]Host Chris_MSNBC says:

JAF: Berapa orang yang pada saat ini bekerja bersama anda mengembangkan Linux?

Host Linus_Torvalds- says:

Itu tergantung dari bagaimana anda menghitungnya. Bila dihitung inti-nya saja, kira-kira 50 orang yang aktif dan barangkali beberapa ratus yang tidak tetap.

Bila anda melihat secara keseluruhannya maka itu jauh lebih luas dibandingkan dengan saya.

Kira-kira 50 orang bekerja untuk ‘operating system’, dan ribuan yang membuat aplikasi.

[15]Host Chris_MSNBC says:

Gervase Markham: Bagaimana anda dapat membuat ke-seimbangan antara komitmen anda terhadap Linux dengan perkerjaan anda di Transmeta dan keluarga anda?

Host Linus_Torvalds- says:

It just happens.

Saya tidak banyak memikirkan tentang itu. Tergantung dari apa yang sedang berjalan – prioritasnya berganti-ganti.

Kadang saya bekerja untuk peruasahaan saya, kadang untuk Linux, kadang saya ambil cuti satu hari.

[16]Host Chris_MSNBC says:

ElvenKnight: Bagaimanpun indahnya Linux, ia masih harus bertempur dengan kenyataan bahwa windows memiliki semua aplikasi. Apakah anda merasakan perlu adanya WINE (windows emulator) untuk itu.

Host Linus_Torvalds- says:

Saya kira WINE itu penting, tapi saya tidak merasa kebutuhan yang mendesak.

[17]Host Chris_MSNBC says:

me: Mohon dijelaskan skenario bagaimana Linux dapat mengambil porsinya Microsoft – tidak hanya NT tapi juga Windows desktops.

Host Linus_Torvalds- says:

Bahwasanya anda memerlukan banyak sukses dipasaran, salah satu skenario adalah seandainya Windows-2000 gagal dan memudahkan untuk Linux masuk pasar. Umumnya membutuhkan bobot yang tertentu untuk mengganti arus.

[18]Host Chris_MSNBC says:

James (CMiYC): Dorongan apa yang anda miliki untuk bertahan bekerja untuk Linux saat ini? Apakah anda masih menikmati kesenangan bekerja untuk itu, atau anda sekarang merasa bahwa itu adalah "job" atau kewajiban anda?

Host Linus_Torvalds- says:

Sebab yang utama adalah kesenangan dan menyenangkan bersama masyarakatnya (Linux). Banyak sekali orang yang telah menghandalkan itu.

[19]Host Chris_MSNBC says:

Juan Diablo: ..linus. saya kagum (seperti juga beberapa teman saya) berapa banyak ‘code’ untuk ‘kernel’ yang sebenarnya anda tulis saat ini? …apakah anda hanya mengecheck terakhir ‘code’ yang ditulis oleh yang lainnya selama ini?

Host Linus_Torvalds- says:

Saya dapat memastikan bahwa 90% ditulis oleh yang lainnya. Saya tentu tidak menghitung baris demi baris.

[20]Host Chris_MSNBC says:

Eduardo Dominguez from Mexico: Sorry untuk bahasa Inggris yang jelek. Apakah anda merencanakan program sertifikasi untuk mereka yang bekerja dan menggunakan Linux?

Host Linus_Torvalds- says:

Belum ada rencana untuk melakukan itu dan pasti bukan dalam hal mengembangkan kernel.

Beberapa menginginkan untuk membuat ijasah Certified Engineer….untuk mengatakan: "Hey, bapak ini mengerti apa yang ia lakukan"…tapi masih belum banyak yang dikerjakan untuk itu. Umumnya datang dari sektor komersial.

[21]Host Chris_MSNBC says:

Alejandro Forero: Web site apa yang umumnya anda kunjungi?

Host Linus_Torvalds- says:

Saya cenderung memulainya dengan Alta-Vista kemudian menuju apa yang saya ingin lihat.

[22] Host Chris_MSNBC says:

CyberYoda: Bagaimana kisahnya sampai Pinguin menjadi maskotnya Linux?

Host Linus_Torvalds- says:

Sebagian secara kebetulan, sebagian karena saya suka Pinguin. Sebagian disebabkan karena saya pernah digigit pinguin. Saat itu disebuah kebun binatang terbuka. Saya membentuk jari saya menyerupai ikan herring – dan ternyata disukai oleh pinguin itu. Betul-betul merupakan gigitan yang mengerikan.

<meringis>

[23]Host Chris_MSNBC says:

Tomasz: Linus: Apakah anda melihat slashdot – www.sslashdot.org "News for Nerds" secara berkala?

Host Linus_Torvalds- says:

Tidak, saya tidak pernah mealakukannya. Saya tidak punya waktu.

[24]Host Chris_MSNBC says:

Clint: Seberapa jauh pengembangan Linux kernel tergantung kepada anda? Adakah orang lain yang mengerti Linux kernel yang lebih baik dari anda?

Host Linus_Torvalds- says:

Sudah tentu ada yang tahu bagian bagian code secara khusus. Lumayan banyak yang memiliki pengetahuan yang baik, tapi sayalah orangnya yang mengetahui secara global apa yang sedang terjadi disemua bagian.

[25]Host Chris_MSNBC says:

ElvenKnight: Apakah Microsoft pernah mencoba membeli Linux atau menawarkan banyak uang untuk menghentikan itu atau sesuatu untuk mengganggu masadepan Linux?

Host Linus_Torvalds- says:

Microsoft tidak pernah menghubungi saya sama sekali. Beberapa orang yang mengembangkan Linux pergi karena kekasaran Microsoft.

[26]Host Chris_MSNBC says:

Apakah anda menggunakan X; Dan bila demikian window manger mana yang anda utamakan?

Host Linus_Torvalds- says:

Saya selamanya menggunakan X dan ingin menggunakan FVWM

[27]Host Chris_MSNBC says:

Jeff Hoskinson: Bagaimana pendapat anda apakah Linux itu lebih baik atau buruk dengan meningkatnya popularitas dan perhatian yang meningkat?

Host Linus_Torvalds- says:

Saya tidak sependapat bahwa perhatian membuat sebuah operating system jadi lebih baik atau buruk. Itu telah membuka pintu yang selama ini dipaksa tertutup.

[28]Host Chris_MSNBC says:

Darren Rowley: Pernahkah anda menghitung berapa banyak uang yang mungkin anda bisa hasilkan dengan menjual lisensi Linux?

Host Linus_Torvalds- says:

Saya belum pernah mencoba melakukan hal itu. Saya tahu benar seberapa kesenangan telah saya peroleh dari itu.

[29]Host Chris_MSNBC says:

Pat Szuta (Perly): Bagaimana caranya agar seseorang dapat ikut membantu mengembangkan Linux?

Host Linus_Torvalds- says:

Saya tidak mempunyai syarat yang tertentu untuk meminta seseorang melakukannya. ‘Just use the system’ dan anda berarti telah melakukan pengembangan Linux, karena anda sudah memiliki keinginan untuk itu.

[30]Host Chris_MSNBC says:

Tom Savage: Bila anda sangat stres, apa yang anda lakukan untuk menghilangkan itu.

Host Linus_Torvalds- says:

Saya biasanya mengambil cuti untuk beberapa hari…istirahat dan membaca buku. Saya membaca segala, kebanyakan trash, science fiction, mistery, thrillers, fantasy, kisah detektif. War and Peace pasti tidak termasuk dalam daftar bacaan saya.

[31]Host Chris_MSNBC says:

Gary Krakow: Andaikata anda tidak menemukan Linux kernel…apakah anda pikir orang lain akan membuat OS berbasis unix di tahun 1999?

Host Linus_Torvalds- says:

Saya pikir perhatian terhadap Unix akan bertambah kecil. Linux telah membangkitkan kembali vitalitas Unix.

[32]Host Chris_MSNBC says:

Jordan: Saya tahun pertama sebagai mahasiswa computer engineering. Bagaimana saya harus mengatur waktu saya?

Host Linus_Torvalds- says:

Saya menggunakan sebagian waktu saya untuk membuat program.

[33]Host Chris_MSNBC says:

Mad-Kow: Banyak populasi rupanya adalah pemain game; Bagaimana support untuk games dan support untuk perangkat keras game dalam Linux?

Host Linus_Torvalds- says:

Tergantu dari apa yang ada inginkan. Tiadak mudah menemukan games untuk Linux tapi ada.

[34]Host Chris_MSNBC says:

joseph: Apakah ada bagian dalam kernel yang anda tidak sukai?

Host Linus_Torvalds- says:
Ada banyak detail, salah satu masalah adalah industri PC yang memiliki banyak perbedaan Hardware dimana qulitasnya juga sangat bervariasi. Ini dapat menyebabkan masalah dengan 'driver' untuk hardware tersebut.

[35]Host Chris_MSNBC says:
Steve Hartzog: Mobil apa yang anda gunakan?

Host Linus_Torvalds- says:
Sekarang saya mengendarai Rav4 warna hijau dan mobil keluarga yang lain adalah green Pontiac.

[36]Host Chris_MSNBC says:

Bart Szyszka: Apakah yang anda pikir sebagai keberhasilan yang terbesar?

Host Linus_Torvalds- says:
Hidup saya, menikah dan punya 2 anak.

[37]Host Chris_MSNBC says:

Eric Murray: Bahasa program apa yang dipakai dalam Linux, apakah semua dalam C code? Apa pendapat anda tentang Java, dan mungkinkah salah satu bahasa akan menggantikan C sebagai bahasa yang dominan?

Host Linus_Torvalds- says:
Kernel nya ditulis dalam low level assembly language...begitulah penyelesaiannya.

[38]Host Chris_MSNBC says:

ian: bagaimana saran anda untuk masyarakat Linux menghadapi kesalahan konsep dimana misalnya Linux tidak memiliki GUI, atau tidak memiliki fungsi "drag and drop" seperti windows/macro?

Host Linus_Torvalds- says:
Saya pikir kita tidak usah khawatir tentang itu. Dua tahun yang lampau juga salah konsep bahwa Linux adalah sebuah 'fluke'.
