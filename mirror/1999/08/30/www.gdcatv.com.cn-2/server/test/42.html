*你 是“假洋鬼子”吗  ？

  1. 你是否会斜眼偷看他人衣服上的商标？
           是    否 
  2. 假如你在屋里穿着旧衣服干活，这时又必须上街购物，你会换件衣服再出门吗？
           是    否 
  3. 假如你与比自己更穷的人在一起，是否感到舒适与高兴？
           是    否 
  4. 假如你外出赴宴，你是否选择自己会使用的餐具而不用外国餐具，以免引起他人的笑话？
           是    否 
  5. 假如有位朋友驾驶着旧摩托车（或破旧汽车）来拜访你，你是否会感到难堪？
           是    否 
  6. 假如有一件衣服穿旧了、磨损或是样式已过时了，但尚能穿，你是否会扔了它？
           是    否 
  7. 你对自己的地方口音感到害羞吗？
           是    否 
  8. 你是否有一套最好的衣服？
           是    否 
  9. 你是否经常对自己的家庭背景感到害羞？
           是    否 
  10. 你是否期望自己最好能够受到更好的教育？
           是    否 
  11. 你是否愿意乘坐最高等级的交通工具去旅游？
           是    否 
  12. 当你打电话时，是否有意以另一种不同的、更为优雅的腔调讲话？
           是    否 
  13. 你是否欣赏与名人、富人或“大人物”相会？
           是    否 
  14. 你是否喜欢坐上轿车到处跑？
           是    否 
  15. 你是否自己家庭背景、自己父母或是自己的财富所些谎？
           是    否
