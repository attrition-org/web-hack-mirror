*你 脾气急躁  吗？

  1. 你在匆忙之中，而一切事情又都不如意时，你会咒骂吗？
           是    否 
  2. 火车站上列车误点，你向站里的人询问而得不到回答，你会对他发脾气吗？
           是    否 
  3. 服务员怠慢了你，你会坚持见经理评理吗？
           是    否 
  4. 你在排队等候，但有人插队，在你之前, 你会让他知道你心中怎样想的吗？
           是    否 
  5. 有人嘲笑你的容貌，你会生气吗？
           是    否 
  6. 有人取笑你的配偶或是亲密朋友，你会生气吗？
           是    否 
  7. 有人取笑你的母亲，你会生气吗？
           是    否 
  8. 你看见有人责打一头猫，你会制止他吗？
           是    否 
  9. 有人在排队时插在你之前，你会提出异议吗？
           是    否 
  10. 朋友们与你一起住在你家中，不慎打坏了几样东西，他们并没有道歉，你会生气吗？
           是    否 
  11. 有人让门弹回来碰在你脸上，你会让那人知道你是怎样想的吗？
           是    否 
  12. 当你看见小孩在你花园中用石块击猫，你会出门制止他们吗？
           是    否 
  13. 有位邻居经常深夜回家，且有不少喧哗，你会提出异议吗？
           是    否 
  14. 有位不太熟悉的人到你家中，点火抽烟，你是否会劝阻他不要吸烟，或是请他离开？
           是    否 
  15. 你是否常在气头上说了些什么，而事后又后悔不已？
           是    否
