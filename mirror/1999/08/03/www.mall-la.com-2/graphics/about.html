DownTown Web Graphics (DT Graphics or DTG) is a new operation, but those who make up DTG today have been working on the web for years. In 1995, Ken Sukimoto, the principal partner in DTG, became motivated to learn web design and graphics creation so he could do work on DownTown WebLines, his web based computer Bulletin Board System (BBS). Shortly thereafter, he started receiving requests for custom graphics and HTML from other Bulletin Boards systems using the same software. The progression from there was to Perl script implementation and customization. Based on recommendations from these early efforts, Ken received requests to participate on larger, more complex projects as a sub-contractor and eventually as the project supervisor in creating several government and corporate internet and intranet sites.

However, while working on these large sites was financially lucrative, something was lacking. There was no sense of accomplishment, no opportunity to try new things and express our own taste. Several others on the project expressed the same feelings and we decided that a return to our past was what we needed - having fun while working on start ups and giving face lifts to established sites. So, we created DownTown Web Graphics as the way for us to market our services.
