Welcome to the x86 archive of the BeSpecific 6 CD by Adamation Inc.!

We have assembled applications and utilities in the following categories:

Add-Ons Command Line & Unix Drivers
Emulators Games Graphics
Networking Productivity Programming
Sound & Music Utilities

The links below will show you the README files for the applications and files on the BeSpecific CD 6 and let you download the archives directly. This archive is for applications that are compatible with BeOS R4 running on CPUs from Intel's x86 processor family. We'll add an archive of binaries for the PPC processor family in the future.

If you would like your software mirrored here, or would like us to add a link to your page or replace an archive here with a link to your own download site, contact support@adamation.com and we'll be glad to do so.

Note that the links below will download the files via http. If you'd like to ftp the files, copy the URL and paste it into the URL text field. Then replace the "http" in the URL with "ftp" and you'll be good to go.

Shareware

Although you can freely copy the contents of the BeSpecific CD (subject to possible restrictions), not all of the software is "free." Some of it is "shareware," meaning that if you use it for more than some specific length of time (indicating that you find it useful in some way), you must pay its author. Shareware payments are often low, usually less than $50, but that doesn't mean the software is low quality or that the authors don't deserve your money. Just as the GNU concept encourages freedom and technical advancement for everyone's benefit, paying for shareware you use is vital to encourage individual programmers to write yet more software. Check each app's Info panel or README file for where to send your check.
