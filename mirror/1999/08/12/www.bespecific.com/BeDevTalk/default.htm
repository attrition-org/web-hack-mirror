BeDevTalk is an open discussion list covering using the BeOS and BeOS software. The list was created as a forum for discussing issues important to BeOS users, including software, helpful tips, tools, techniques, great uses of the BeOS, business issues, and the like. For more information about this and other discussion forums, please visit Be's web site.
