February 27, 1996
Added telnet link (login: library) to OVID Databases at University of Missouri Libraries on the Research and Resources page. The following databases are available: ABI-Inform (>1985), Biological Abstracts (>1985), CINAHL (>1982), Compendex (>1990), Current Contents (all editions), Dissertation Abstracts (>1861), ERIC (>1966), MEDLINE (>1966), Newspaper Abstracts (>1989), Periodical Abstracts (>1986), and PsycINFO (>1967).
February 26, 1996
Updated CBB and MBB doctoral faculty listings.
January 25, 1996
A link to an experimental WWW Bulletin Board has been added on the SBS Events page. This will allow those interested to post announcements of upcomining journal club and simmilar events.
The SBS Homepage may now be accessed by simply typing "http://sgi.bls.umkc.edu/". Please note that it is no longer necessary to specify the filename "sbshome.html" when you access the top page, and that the literal name of the host is now working.
January 08, 1996
The evaluation of the SBS Website has been completed. The server has been moved to its final location; it will no longer be necessary to type the port number (:8080). We thank to all those who contributed information and suggestions during this testing period!
December 07, 1995
Added server-side imagemap support. You should be able to use the navigation button bar found at the top of each document.
December 03, 1995
UMKC School of Biological Sciences Website is finally online, for evaluation. Please let us know if you encounter bugs, or have any suggestions. SBS Website Info Page contains more information about this site, credits, disclaimers, and describes how to contribute material for these pages.
