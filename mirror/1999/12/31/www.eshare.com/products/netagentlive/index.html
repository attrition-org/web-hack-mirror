NetAgent Live is the easiest, fastest, and smartest way to bring real-time communication and automated e-mail management services to your web site. eShare NetAgent Live hosting service provides the critical infrastructure for the industry's leading web-based customer service application, allowing companies to deliver high quality customer service without the hassles.

Benefits

Minimize Up-Front Cost
Immediately receive all the benefits of eShare's NetAgent Software Suite without the up-front hardware, software, maintenance, and IT infrastructure costs. Maximize the return on your investment!

Eliminate Cost of Ownership
All back-end support for NetAgent Live is hosted by eShare - reducing your company's administration and overhead costs. You provide the people, we'll provide the power!

Immediate Deployment
NetAgent Live eliminates your need to purchase or configure hardware and software so you can be up and running in no time. Each customer is assigned a dedicated service representative who ensures that all installation and training associated with the NetAgent Live service is managed successfully.

Network Services
NetAgent Live hardware and software is managed through the eShare Hosting Network. Leveraging a global network of Internet Data Centers, the eShare Hosting Network is equipped to provide the highest level of reliability and security with multiple redundant connections to internet backbones, redundant power, backup generators, redundant server hardware, automated backups, and pro-active network and system monitoring.

Remote Access
Using a standard web browser, customers can remotely set and modify all NetAgent Live settings, including agent configurations, agent knowledge bases, passwords, scheduling, and service line and e-mail box configurations.

Ownership Option
eShare's NetAgent Live ownership policy makes it easy for customers to move from a hosting solution to an in-house system purchase to accommodate ever-changing budget and business needs.

Multi-Channel Solution
NetAgent Live offers the same functionality as NetAgent 3.0, including:

  • Live text interaction
  • e-Mail response management (eShare NetAgent Re: Sponse)
  • Voice over IP

All interaction channels are fully integrated, sharing agent administration, authentication of agents, supervisor monitoring and reporting, and historical data from multiple channels.

Technical Requirements

Companies interested in delivering hassle free customer service solutions with NetAgent Live will need the following:

Agent and Supervisor Desktop Requirements:

  • Pentium 200 or greater
  • 64MB RAM recommended
  • 25MB free hard-disk space
  • Windows 95, Windows 98, Windows NT Workstation with service pack 4, Windows 2000
Administrator Desktop Requirements:
  • Netscape Navigator or Internet Explorer version 4 or higher
  • 64MB RAM recommended
  • 25MB free hard-disk space
  • Windows 95, Windows 98, Windows NT Workstation with service pack 4, Windows 2000
