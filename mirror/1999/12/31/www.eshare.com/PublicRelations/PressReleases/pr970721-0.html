eShare Technologies Expressions Cartridge for Oracle Web Application Server Provides Online Communications Solutions

eShare Expressions Recognized in Oracle Cartridge Solutions Competition

Commack, NY (July 21) — eShare Expressions, running on Oracle® Web Application Server, is the first powerful, easy-to-use and cost efficient live online communication solution of its kind on the market. eShare Expressions empowers corporations, organizations and individuals to easily create live chat and threaded bulletin board services ideal for corporate Intranets, corporate training, help desks, call centers, distance learning, Extranet implementation and electronic commerce. eShare Technologies Inc. is the recipient of an Engineering Award in Oracle's Cartridge Solutions Competition for eShare Expressions.

eShare Expressions enables businesses and organizations to harness the universal reach of the Web to create enduring bonds and improve relationships with business partners, associates and employees.

The eShare Expressions cartridge runs on Oracle Web Application Server, which is based on Oracle's Network Computing Architecture™. Network Computing Architecture is Oracle's open, standards-based platform for network computing. Cartridges, or application components, are a key element of network computing, allowing users to easily build, customize and extend their applications in a modular fashion while preserving existing hardware and software investments.

"Development of the eShare Expressions cartridge for Oracle Web Application Server means that customers can extend to the Web the internal and external communication processes that are so vital to companies today," said James Tito, President and CEO of eShare Technologies. "This lowers costs and improves customer service, a combination that gives companies significant competitive advantages."

"The ability to get closer to the customer by extending business systems to both internal and external customers via the Web is what network computing is really all about," said Beatriz Infante, Oracle Senior Vice President, Application Server Division. "We are pleased that the eShare Technologies cartridge has been developed for Oracle Web Application Server."

Live communications is becoming an important part of all network systems. Analysts and venture capitalists estimate that the market for live web communication software will reach $1.5 billion by the year 2000. Right now, however, the market is in its infancy with first-generation products marked by technical complexity and installation difficulties. However, the demand for interpersonal communication software, such as chat, is escalating.

eShare Expressions offers many features:

  • No browser plugs-ins or downloads
  • Operates behind firewalls — a critical feature as nearly 50% of Internet users are protected by firewalls
  • Support for 50 simultaneous users — expandable to unlimited users with upgrade
  • Choices: create public or private, password protected rooms, set up profanity filters, create buddy lists
  • Rotating banners within both Chat and BBS evironments
  • Presentation capabilities — take chat room users through html pages and PowerPoint-type presentations. Plus, enhance communication by dropping graphic images in line with text messages
