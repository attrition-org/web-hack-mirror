AMMS WEB ORDER ENTRY NOW AVAILABLE

Code 230 now has a new stores stock order entry system available. To replace the AMMS portion of ASAP which replaced the SATS system, Code 230 designed a new web-based order entry system which has been tested by several of our biggest users and we received positive responses from them. You may continue to use ASAP to process your orders but it may be discontinued in the near future. The new web based AMMS Order Entry System will streamline and expand your stores stock order entry process by allowing up 100 lines to be processed on a single customer requisition. To be able to access the AMMS Order Entry system your computer needs to meet the following requirements:

PC Requirements

Netscape Navigator 4.06 or later version or Microsoft Internet Explorer 4.01 or later version.

Macintosh Requirements

Power PC

Mac OS 7.6.1 or later

At least 32 megabytes of ram

MS Internet Explorer 4.01 or later version

Mac OS Runtime for Java (MRJ) 2.1.2 Available from www.apple.com/java

To insure that the JON information we have is correct in the new system, please e-mail a listing of all JON accounts currently being used and the name(s), code, shipping address (building and room number), and phone number of the individual(s) authorized to use these accounts to this address: bonnie.c.perry.1@gsfc.nasa.gov.

Upon receipt of user information, all data will be verified in the new system, and users will be notified via e-mail of the userid and URL to access the AMMS web-order entry system.
