Microsoft Outlook 98 Configuration
(Notes: In this page, <username> should be replaced by your user name. For example, if your user name is xyang, you should use xyang replace <username>. Don’t input "<" and ">".)

1. Open Outlook 98.   

     

2. From the Tools menu, choose Accounts.

3. Click the Add button, and choose Mail.

4. In the Display name box, type your name and click the Next button.

5. In the E-mail address box, type your e-mail address (e.g. <username>@mail.bsu.net). Click the Next button.

6. From the drop-down list, choose POP3.

In the Incoming mail (POP or IMAP) server box, type mail.bsu.net.

In the Outgoing mail (SMTP) server box, type mail.bsu.net. Click the Next button.

7. Select Log on using: POP account name, Password.

In the POP account name box, type your username.

In the Password box, type your password. Click the Next button.

8. In the Internet mail account name box, type BSU.NET Mail (Note: this can be anything you want). Click the Next button.

9. Select Connect using my local area network (LAN).  (Note: If you use a modem connect to BSU.NET, you should select Connect using my phone line.) Click the Next button.

10. Click the Finish button to save settings. Setting is done. Now you can try to dial in BSU.NET.
