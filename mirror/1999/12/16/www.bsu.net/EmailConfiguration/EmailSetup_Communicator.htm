Setup Email in Communicator

The installation of Communicator is straight forward and only requires that you fill in a few blanks. To start the installation, find the file you downloaded and double-click on it. This will start the installation immediately on Windows 95/98 and NT. Follow the on-screen prompts and provide the information below.

1. Your Full Name
2. Your E-mail Address, e.g. <username>@bsu.net
3. Profile Name, (will be filled in by the installer)
4. Outgoing mail (SMTP) server mail.bsu.net
5. Mail server username, e.g.. <username>
6. Incoming Mail Server - mail.bsu.net
7. News (NNTP) Server - leave it blank(Don't put anything here).


This should be all the information required. Once you've reached the last screen, click the Finish button and Communicator will send and receive e-mail.
