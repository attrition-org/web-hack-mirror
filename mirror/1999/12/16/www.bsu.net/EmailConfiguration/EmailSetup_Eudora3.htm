Eudora 3.x Configuration

(Notes: In this page, <username> should be replaced by your user name. For example, if your user name is xyang, you should use xyang replace <username>. Don’t input "<" and ">".)

  1. Open Eudora
  2. Click the Tools menu, choose Options and select the Getting Started category.
  3. In the POP Account text box, type your e-mail address. For example: <username>@mail.bsu.net
  4. In the Real name text box, type your name.
  5. In the Return Address text box, type your E-mail address. For example: <username>@bsu.net

         

  1. Under Category, select Hosts.
  2. In the SMTP text box, type mail.bsu.net

             

 

  1. Under Category, scroll down to Advanced Network and select it.

          

  1. Uncheck Cache network info.
  2. Set Network open timeout to 900 seconds.
  3. Set Network timeout after to 900 seconds.

             

  1. Click the OK button.
