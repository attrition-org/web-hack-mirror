Reconfiguring an Existing Account
(Notes: In this page, <username> should be replaced by your user name. For example, if your user name is xyang, you should use xyang replace <username>. Don’t input "<" and ">".)

  1. From the Tools menu, choose Account.
  2. Select the Mail tab.
  3. Select the account to be reconfigured.
  4. Click the Properties button.
  5. In the Mail account text box, BSU.NET Mail
  6. In the Name text box, type your real name.
  7. In the E-mail Address text box, type your email address.
    For example: <username>@mail.bsu.net

    8. Or  <username>@bsu.net

     

  8. Check Include this account when doing a full send and receive.
  9. Select the Servers tab.
  10. In the Outgoing Mail (SMTP) text box, type mail.bsu.net.
  11. In the Incoming Mail (POP3) text box, type mail.bsu.net.
  12. Select Log on using.
  13. In the Account name text box, type your <username>.
  14. In the Password text box, type your password of BSU.NET.
  15. Uncheck My server requires authentication.

  16. Select the Connection tab.
  17. Select Connect using my local are network (LAN).

  18. Select the Advanced tab.
  19. In the Outgoing Mail (SMTP) text box, type 25
  20. In the Incoming Mail (POP3) text box, type 110
  21. Move the Server timeouts slider to the right (toward Long) to set the Server timeouts value to 5 minutes.

  22. Click the OK button.
  23. Click the Close button.

 

Configuring a New Account

  1. From the Tools menu, choose Account.
  2. Select the Mail tab.
  3. Click the Add button.
  4. From the Add menu, choose Mail.
  1. In the Display name text box, type your name.
  1. Click the Next button to continue.
  2. In the E-mail address text box, type your email address.
  1. In the Incoming mail (POP3 or IMAP) server text box, type mail.bsu.net
  2. In the Outgoing mail (SMTP) server text box, type mail.bsu.net
  3. Click the Next button to continue.
  4. In the Password text box, type your password.
  1. Click the Next button to continue.
  2. Click the Next button.
  3. Select Connect using my local area network (LAN).
  1. Click the Next button to continue.
  2. Click the Finish button.
  3. Click the Close button.
