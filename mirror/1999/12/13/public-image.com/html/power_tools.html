Web Master Toolkit

At present, we now have Windows 3.1 and 95 and Mac resources available.


Windows95 Telnet Clients:

CRT CRT is a 32-bit terminal emulator that supports both telnet and rlogin. Emulation includes VT100, VT102, VT220, and ANSI. Zmodem upload/download over telnet is supported. CRT 1.1.2 supports a simple mechanism for automating the login process. For more info, check out the CRT Home Page.
HyperTerminal Private Edition A more powerful version of the HyperTerminal program that comes in Windows 95. For more info, check out the HyperTerminal Home Page.
WinTelnet & FTP Pro WinTelnet & FTP Pro is a telnet and ftp program that brings the ease of use of Windows to the Internet! No longer will you need to memorize arcane UNIX commands to use your host computer---WinTelnet & FTP Pro hides the complexities of UNIX with an easy to use Windows front end interface. For more info, check the WinTelnet & FTP Pro Home Page.


Windows95 FTP Clients:

WS_FTP 32-bit A terrific, easy to use, and intuitive front end to Internet file transfer. For more info, check out the WS_FTP32 Home Page.
CuteFTP The niftiest FTP client around. CuteFTP is one of very few programs that allow novice users to utilise the capabilities of FTP without having to know all the details of the protocol itself. This link is very busy! For more info, check out the CuteFTP Home Page.


Windows95 Graphics Editors:

Paint Shop Pro Paint Shop Pro is one of the easiest and most powerful image viewing, editing and converting programs you may ever use. With support for over 30 image formats, and several drawing and painting tools, this may be the only graphics program you will ever need. Check out the Paint Shop Pro Home Page.
LView Pro One of the most versatile image manipulation and view programs around. A must for anyone who works with many different graphical images and types. Check out the LView Pro Home Page.
GIF Construction Set GIF Construction Set for Windows is a powerful collection of tools to work with multiple-block GIF files. It will allow you to assemble GIF files containing image blocks, plain text blocks, comment blocks and control blocks. It includes facilities to manage palettes and merge multiple GIF files together. It will make the extensions of the GIF specification work for you. Check out the GIF Construction Set Home Page.


Other Tools for Windows95:

CSE 3310 HTML Validator Validate your HTML documents faster and easier on your own computer without having to use internet validation services. Includes support for HTML 2.0, Netscape Extensions (Mozilla), Microsoft Internet Explorer Extensions, frames, and table. For more info, check the CSE 3310 HTML Validator Home Page.
HTML Color Reference A handy utility to convert RGB colors to WWW hex codes. Great for figuring out the right background colors and text colors for your Web page.
WinZip Extraction Utility An absolute must have utility. This should be the first program downloaded from the Internet. For more info, check the WinZip Extraction Utility Home Page.
