All Virtual Domains include the following features:

       250 MEGS!! of space per domain.  
     $45.00 per month

       Unlimited email aliases - name@'yourdomain'.com .

              You may create and remove these aliases, by editing a simple text file in
your domain.

              Each alias may be forwarded to a separate individual email address.

       Unlimited Mailing Lists - This feature allows you to create a mailing list with
     a simple text file of addresses.

 

               You may have as many lists in your directory as you like, each is
password protectable.

               Ideally these lists will handle upto 5000 addresses each.

       CGI-Bin access - Each domain will have this, a variety of pre-configured
     scripts are provided:

               Guest book pre-configured ready to run.

               WWW Bulletin Board, ready to run.

               Site Database and search engine (see below).

               Full site counter - Easily added to pages, uses SSI.

               Cgi-Email - fully configurable forms send script, A sample order form is
provided in all domains.

               Form.cgi - Allows any form to be sent by mail or written to a database.

               Random text displayer - Allows a rotation of simple sayings or quotes,
Uses SSI.

               Imagemap - for click able image redirection.

               Site Search - Simple whole site search engine.

     Unlimited Auto responders - This will allow an automated response to an
     email message.

               This allows you to drop as many simple text files as you like, to respond
to specific incoming inquiries.

               You can also capture the requesting address, for use in your mailing list.

     Weekly stats of how many visitors came to your site by page, Hits and transfer
     amounts (fully graphical).
     List servers - (1 for each domain) Closed or open mailing list program,
     designed to handle thousands of names.
     Password protected Web directories - Fully configured ready to use.

                This will allow you to create a login and password file on specific web
pages.

                Visitors will not be able to view these pages unless you provide them
access.

     Redirection capability - The ability to redirect a web page to read information
     from other sources.
     Secure commerce directory for transferring credit-card information.
