#ClanUK IRC Set-up Tutorial

How to connect to the NewNet IRC Channel, by Mith 26/1/99

The ClanUK IRC chat channel has moved to a different IRC network, called NewNet, because of many problems with DALnet. The following guide will help people make a connection to the new channel, and can be started at two points:

  • If you are new to IRC, or have not used an IRC program before (for example, if you have used the Java applet to connect in the past) then please read this guide from the beginning. The Java applet will not connect you to NewNet, therefore can no longer be used.

  • If you already have mIRC or an IRC client, you can skip the first few stages of this guide which deal with downloading and installing mIRC. However, you may want to use this information to set up your own client better.

1) First, you will need to download an IRC "client" program which allows you to connect to Internet Relay Chat and join the ClanUK channel. You can download it from either of these links:

  • [ mirc55t.exe ] - from Globalnet

    • There is also a newer version from the following sites

  • [ mirc551t.exe ]
  • [ mirc551t.exe ]
  • [ mirc551t.exe ]
  • [ mirc551t.exe ]

Download and install this program - this is fairly routine. Once installed, you will be able to start mIRC by selecting it from your Start Menu.

2) When you run mIRC, it will pop up a "Setup" window, looking something like the window opposite. Make the following changes here:

  • First, enter your name (or Clan alias) in the "Full Name" box.

  • Then enter your e-mail address (it doesn't have to be your real address - anything will do) in the "E-Mail Address" box.

  • Enter the nickname that you wish to use in the "Nickname" box. There is a maximum of 8 letters on NewNet, so choose the name appropriately. If you wish, you can also enter an "Alternative" name, in case your choice is already used.

  • Tick the "Invisible mode" box - this stops mass-messages being sent to you, which are annoying, but does not affect any other functions of IRC.

Don't close this window yet!

3) You should click on the "Add" button, to add the server which will allow you to join the Clan channel. In the dialog box which appears:

  • Enter a server description in the Description box. I recommend something like NewNet IRC for ClanUK - but it is up to you what you enter here.

  • Next, enter irc.tscnet.net in the IRC Server box.

  • Enter 6665 in the Port(s) box.

  • The other fields can be left blank. Click on the Add button to add this server, and you will be taken back to the original setup window. Again, don't close it - we are not finished yet!

4) Next, click on the "Options" category in the list on the left of the window. This will reveal the dialog box shown on the right. Here you need to:

  • Tick the "Connect on startup" box

  • Tick the "Reconnect on disconnection" box

  • Clear the "Pop up connect dialog on startup" box.

Guess what - we are not finished yet - one more setting to go! So leave the window open again...

5) Click on the + next to IRC on the Category list on the left, and select the Perform item. This will open up the dialog box shown on the left. Do the following:

  • Tick the "On connect, perform these commands" box

  • In the white box, type in /join #clanuk

Once you have done this (the window should look the same as the image opposite) then congratulations - you are finished setting up mIRC!

Close the Setup window by clicking on the OK button, which will also start to connect you to the IRC server. From here on, you should automatically join the #ClanUK channel without problems.

ClanBot
The IRC ClanUK Channel Bot.

Instructions specific to ClanBot can be found at http://www.cropton.demon.co.uk/clanbot/ - this will give a brief overview of basic ClanBot features and commands, particularly the voting commands. Make sure you use ClanBot's own online help system too!

To use this, simply type /msg clanbot help whilst in the #ClanUK channel - as long as ClanBot is around.

NewNet IRC Servers
For advanced use only - in case the TSCNet server is down

The following servers will all connect to NewNet:

  • irc.tscnet.net - port 6665 (recommended)
  • eu.newnet.net - port 6667
  • irc.newnet.net - port 6667
  • irc.eskimo.com - port 6667
