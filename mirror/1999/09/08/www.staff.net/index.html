The Myth:
E-Commerce is safe. We use 128 bit encryption,
making it virtually impossible for hackers to
get your credit card number.

The Facts:
Sure we use encryption, but this is only between your
computer and the server the cart software is run on.
Anyone who gains access to the server also gains
access to your credit card number.
Seeing this message just shows how unsecure most
servers running on the net today are!

Too many people (with no computer knowledge) are in a mad rush
to make it rich on the internet.
Equally guilty (if not more so) are people
rushing to write & sell unsecure CGI scripts.

Until Security is an issue that both CGI writers and Web Admin
take seriously, the web will remain a unsafe place to do business!
