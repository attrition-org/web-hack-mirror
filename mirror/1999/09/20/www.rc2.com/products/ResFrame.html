ResFrame Interactive 3D Faulted Model Construction

Picture illustrates physical property calculated with ResMod built on a grid created with ResFrame.

ResFrame is a revolutionary application providing the user with tools to construct complete 3D models in literally minutes and hours instead of days and weeks. The grid cells are generated by intersecting an orthogonal XY grid with the fault framework and stratigraphic horizon information. The resulting 3D grid accurately defines complex geological models while interfacing efficiently with reservoir characterization applications such as geostatistics and conventional fluid-flow simulators. If you have a fast connection, a gallery of ResFrame picture is available by clicking here ResFrame Gallery.

A highly interactive user interface provides the functionality for loading well, tops, picks, horizons, fault planes and polygons and seismic data into one 3D environment.

The user can then interactively digitize fault lines along fault gaps to construct parametric fault surfaces.

The interface provides all of the functionality for users to easily toggle on and off any of the loaded data objects for quality control.

Once the fault lines are constructed, parametric fault planes are simultaneously computed in seconds. Surfaces can then be toggled on and off, and a very efficient iterative loop allows the user to interactively change the fault planes to properly intersect the loaded surface horizons.

When the user is satisfied with the fault-horizon and fault-fault intersections, an efficient fault-gapping algorithm removes the appropriate surface information near the faults and re-projects the surface into the fault planes.

After the surfaces are appropriately tied to the fault horizons, an extremely fast algorithm is used to construct the grid cell topologies that are then used for populating attributes.

The resulting 3D grid accurately defines complex geological relationships both along fault edges and at bounding geological horizons by using onlap, offlap, truncated, and conformable stratigraphic bedding styles. For example, the above picture shows secondary faults that are seamlessly generated off of the main fault shown in blue. Well trajectories are emanating from the top.

After the 3D grid framework is complete, the model can be populated with any number of ResMod's powerful geostatistical algorithms. An example porosity distribution is displayed on the slice above.

Features:

  • Cross Platform (Unix & NT)
  • Highly automated tools for interactive construction of Faulted Frameworks.
  • Handles a large number of faults efficiently.
  • Efficient and accurate horizon-fault and fault-fault intersections.
  • Full interactive modeling and lag visualization.
  • Accurate calculation of cell volumes and cell connections across faults.
  • Links to other (RC-Squared) products.
  • High performance 3D visualization of wells, logs, surfaces, 3D models, seismic data, and faults
