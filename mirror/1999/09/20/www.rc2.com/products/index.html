RC2 software applications have been designed and developed by geoscientists for use in their field studies and project work resulting in a clear and straightforward approach to 3D reservoir model building. RC2 software applications provide an integrated toolkit for facilitating the reservoir model building process. All modules have similar user friendly MotifTM interfaces and intuitive menu structures. They share the HDF data format and file structure allowing for easy data manipulation, fast access and storage, and rapid communication between modules. Results may be output in many formats for hardcopy, display, and output in several fluid-flow simulator input formats.

ResMod is (RC)2's covariance based geostatistical and object modeling package. It features a large suite of estimation and simulation techniques for generating reservoir models. ResMod models continuous (e.g., porosity), categorical (i.e., facies), and categorical-to-continuous (facies dependent) properties aw well as Boolean channels, Boolean deposits and turbidite channels.

ResSeis is specifically designed to condition seismic information for direct integration with well data. This is accomplished by supplying a set of tools for converting seismic traces to pseudo-logs constrained by the available well information. It relies heavily upon the geostatistical techniques in ResMod and ResGram to build 3D models for full wide-band 3D seismic inversion. The sample by sample correlation between the resulting pseudo-logs and the well logs is maximized through the incorporation of these high and low frequency models and the use of accurate depth-time functions. This approach provides that ability to invert quickly very large datasets for different types of data such as 2D, 3D and crosswell seismic.

Besides the described deterministic inversion approach a stochastic inversion approach again utilitizing ResMod functionality but contained within ResMod is available. Tools to accomplish these inversion steps also include a full featured wavelet estimation capability with auto-solution option. Impedance models generated through the inversion techniques have been shown to add good controlling detail to reservoir models when used as soft data in ResMod. ResMod allows the user to specify the degree of this control.

ResFrame (soon to be released) provides the user with a large set of easy to use tools to rapidly construct,edit and visualize complete faulted 3D models. These faulted frameworks can then be populated with physical properties using ResMod. ResFrame is cross platform working with Windows NT as well as Solaris and Irix operating systems.

ResFlow (soon to be released) is a very fast streamline based flow simulator. It can easily take realizations generated from ResMod and quickly rank them.

ResFrac provides a set of tools for understanding and characterizing fractured reservoirs. With its intuitive interface, you'll be able to import, analyze and integrate geologic, geophysical and engineering data into a fractured reservoir model.

ResCalc is a tool for performing model operations, connectivity analysis, constrainted volumetric calculations, constrained surface calculations, virtual well planning and uncertainty assessment. It is run on models that have been generated previously with ResMod. An example is the Model Operations Module where the user can read in any number of sessions from ResMod, and run the data cubes through a very user friendly equation parser. A simple example would be to calculate an OOIP data cube from EFFPOR and WSAT cubes.

ResGram is used to compute and model spatial variations (variography) for use in creating geostastical models. Input data types can be well log data, surface data, seismic data or marker data. Isotropic and Directional variograms are handled with ease and data can be modeled in transformed space.

ResPrep is used to convert standard data formats into the public domain HDF format (binary format developed at the Illinois Supercomputing center to handle very large scientific datasets). ResPrep is designed to get data into our software very quickly (at most a couple of hours on very large datasets involving seismic data). There are tools for reading LAS Files, General Ascii Files, SEG-Y data, OpenWorks data, etc.

This program provides upscaling tools to easily coarsen very large reservoir models to sizes acceptable to commercial reservoir simulators. Using ResScale, these large models can be manipulated quite easily.

This program provides a 3D visualization tool based on the OpenInventor graphics toolkit. It provides the capability to generate fence diagrams, rotate the model, see through certain ranges of values, display well paths and their logs, animate front displacements, etc. ResScape is provided with any ResMod license.
