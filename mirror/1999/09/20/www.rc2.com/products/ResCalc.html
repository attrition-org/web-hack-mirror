ResCalc: Model Calculation Utilities and Uncertainty Analysis


This program is provided as a complement to ResMod to visualize and perform calculations on multiple realizations. Support can be reached through email rescalc@rc2.com or through our Denver office.

ResCalc is a model operations and uncertainty assessment program that allows the user to post-process data cubes from ResMod. There are several modules in ResCalc. Most important and directly applicable to everyday operating company projects is the Model Operations Module. Here the user can read in any number of sessions from ResMod, and run the data cubes through a very user friendly equation parser. A simple example would be to calculate an OOIP data cube from EFFPOR and WSAT cubes. The operation would be entered as follows:

OOIP = EFFPOR / 100. * (1. - WSAT)

Here, effective porosity, in percent, is converted to fractional porosity and then multiplied against the OSAT (1. - WSAT). The equation parser handles the following operands:

+addition
-subtraction
*multiplication
/division
%modulus
^exponentiation
( )parentheses

The equation parser has robust error checking and catches mismatched parens and syntax errors. The user may supply different equations for different reservoir zones. After the new cube is calculated, the user may readily view the cube and use it for subsequent calculations. Boolean logic such as if-then-else or if-and if or if-or logic may be used in all of the model operation logic. The user can specify up to 10 boolean operators per reservoir zone.

The Map Maker Module allows the user to build a variety of maps such as net, gross, net-to-gross, net pore volume etc.. from 3D data cubes. The user is able to generate maps for different reservoir zones and is also allowed to planimeritize the grid with a polygon tool. The user may constrain the map calculations by using boolean operators such as "only calculate the net for a cell if the water saturation cube is below x amount." Any number of model constraints may be used for calculating the maps. The user is also allowed to bring in external surfaces, such as oil-water contacts, and calculate the maps while constrained by the external surfaces (e.g. "only calculate a cell volume if the cell centroid is below or above a given surface. All of the generated maps are readily available for viewing and appropriately labeled in our surface viewer.

Also available in ResCalc is the uncertainty assessment module. Here the user reads in any number of realizations generated from any simulation algorithm or via spatial bootstrapping. Cdf's are calculated for each cell using all of the realizations and the user may display the uncertainty as probability isoshells in our 3D viewer or as uncertainty cross sections in the interactive uncertainty viewer. All statistical parameters such as mean std, variance etc.. are calculated for each cell and are interactively shown when the cursor is dragged over a cell. The user may also view the cell cdf's interactively and pan all of the realizations at once using an interactive viewer.

A univariate statistics option is also available to compare the model results with the input data from well control. A qunatile-quantile plot is also available in a separate window to compare the two distributions.

Features:

  • Mathematical operations on models
  • Calculation of volumetrics
    • Single model gross, net, and net-to-gross calculations
    • Uncertainty using a automated jackknife procedure
  • Local uncertainty calculation using ResBatch results
  • Map maker module
  • Difference, crossplot maps between any two realizations per area and zone
  • Univariate and bivariate statistics for any set of realizations
  • Well planning capabilities (in development)
  • Drainage volume and connected pore volume calculations (in development)
  • Customer requested model calculations
