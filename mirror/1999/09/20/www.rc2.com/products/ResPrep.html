ResPrep: Data Handling Utilities

This program is used to convert standard data formats into the HDF format (binary format developed at the Illinois Supercomputing center to handle very large scientific datasets). It is designed to get data into our software very quickly (at most a couple of hours on very large datasets involving seismic data). Support or questions can be addressed through email resprep@rc2.com or through our Denver office.

Features:

  • Very user friendly
  • Handles LAS well formats
  • Handles GeoEAS well formats
  • Handles SEG-Y seismic data formats
  • Handles PROGRESS formats
  • Generic ASCII file format reader
  • Quick visualization of input data for quality control
  • Downloading of datasets in ASCII format
  • Addition of data (wells, markers, seismic data, surfaces, 3D models) to existing dataset is done on the fly
  • Direct link to OpenWorks (in development)
  • Will customize to handle specific data formats
  • Surface and Log calculator
  • Derive markers from surfaces
  • Extensive data search capabilities
