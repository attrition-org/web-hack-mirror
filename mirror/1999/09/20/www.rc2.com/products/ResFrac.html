ResFrac: Fractured Reservoir Interpretation and Modeling

ResFrac provides a set of tools for understanding and characterizing fractured reservoirs. With its intuitive interface, you'll be able to import, analyze and integrate geologic, geophysical and engineering data into a fractured reservoir model.

Features:

  • Unique RC Squared technology designed for understanding and modeling fractured reservoirs in any depositional environment.

  • Intuitive, work-flow oriented, and user-friendly interface facilitates the use of ResFrac.

  • Unique tool for integrating geologic, geophysical, and engineering data into a fractured reservoir model.

  • Uses static reservoir information and/or dynamic data to define a Fracture Indicator.

  • Provides the unique opportunity to find what geologic parameters control the fracture intensity.

  • Provides network maps that help identify IN-SITU fracture directions and types of fractures

  • Links to other RC Squared software.

  • Cross platform software (Unix and Windows NT)
