Patch Update
A patch update file has been released to fix the problem of a runtime error on some fast 200MHz+ machines trying to run the programming software.

   Download :  TPPATCH.ZIP   9k

Extract the file to your Haltech directory where the programming software   .EXE  file is located. After extraction you must go to the DOS prompt then go to the same directory as the .EXE file and type in :
TPPATCH FILE.EXE
Where FILE.EXE is the name of your Haltech software.
For instance , is you have E6A software and the file is called  E6A.EXE  , then you would type in TPPATCH E6A.EXE and the patch will run itself which takes a few seconds.
If you have any difficulty then do not hesitate to email or ICQ me about it.
 

Haltech F9 / F9-A Fuel Computer
The Haltech F-9 is a fuel only engine management system suited to engines that already have their own ignition system , such as older EFI engines that still run a distributor.

   Click HERE for more Information on the Haltech F9
 

Haltech E6-A Fuel / Ignition Computer
The Haltech E-6a is a comprehensive fuel and ignition engine management system for the professional or enthusiast. It is suited to modern multiple ignition coil engines or custom made engine conversions.

   Click HERE for more Information on the Haltech E6-A

Haltech E6-S Fuel / Ignition Computer
The new E6-S is a redesign of the existing E6-A with a few major updates. Controlling both Fuel and Ignition systems it is the complete solution for your engine management needs.

   Click HERE for more information on the Haltech E6-S

Haltech E6-GM Fuel / Ignition Computer
The Haltech GM is a comprehensive fuel and ignition engine management system with a direct plug-in setup for Delco computer equiped cars. Offering the same levels of adjustment as the E6-A it is a perfect solution for modified vehicles running the Delco. See the E6A information linked below.

   Click HERE for more Information on the Haltech E6-GM

Haltech IG5 Ignition Computer
A stand-alone ignition management system. Basically the ignition-side only of the E6 systems it is perfect for those wishing to update at a later date from a fuel-only system.

   Click HERE for more information on the Haltech IG5

Haltech HaligatorTM  *Not available yet. Still under development*
The Haligator is an electronic programmable boost controller and extra injector fueller.

Demo Software
A DEMO version of the Haltech programmable software is now available for download. Just unzip the file and run the DEMO.EXE file. Hit "Y" to go online as this demo version has a simulated running engine so you can see how it all works. To see the menus on which are on top, hold down the ALT key and then hit the corresponding first letter for that menu, eg. ALT-F to bring up the FILES menu. You can also download some of my basic sample maps that may help point you in the right direction at the bottom of this page.
