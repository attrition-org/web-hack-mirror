Haltech F-9 / F-9a
Information taken from Haltech info sheet. Minor update : 26th March, 1999.

The fuel-only Haltech F9 system is perfect for those wishing to convert from carburettor to EFI, or for those who have done an engine conversion to an imported EFI motor. The F9 system offers near unsurpassed levels of adjustability that will allow you to intuitively set up any engine to run to its best possible performance. Read below the information taken directly from the Haltech brochures. Do not hesitate at all to email me with any questions and queries about the Haltech systems.

   The Haltech F9 is a powerful "real-time" programmable fuel injection system computer designed for those seeking optimum performance. No other system in the same class is as adaptable , easy to install or program.

   The F-9 System has up to 22 adjustable fuel maps each with 32 individual bars. The F9 will run up to 16000 rpm with better resolution and greater accuracy than ever before. Up to 8 injectors ( more using the optional driver box ) can be controlled directly , fired all together , in batched , or can be staged when running high boost turbo or superchargers.

   The patented HALTECH system of programming virtually elliminates the input of numbers. You simply manipulate graphics in the form of bar graphs and by pressing arrows you increase or decrease the amount of fuel delivered aty that particular load point. The process is repeated for all load points in each rpm range.

The F9-A together with the above, will control closed loop feed-back in conjuction with an optional Oxygen Sensor, controls motor idle speed when optional idle air control motor is installed and can control any 3 of the listed output options.
