Haltech E6-S
Fuel / Ignition Engine Management System
Last Updated : 22nd March, 1999
   The newly released E6-S is now available. Now featuring an on-board reluctor adaptor meaing it will accept many more trigger input types such as magnetic reluctor pickups common in many factory cars. Also updated with more internal drivers to give the ability to run more applications with an additional 4 drivers available with the E6S-8 Option. Also now available in either a completed wiring loom, or as a "flying" loom for you to assemble to your own required length.
Basic tuning features are the same as the E6-A and F9 , along with an internal dip switch arrangement for selecting ignition input type.


The E6S-8 - as used on my own car

Currently it is programmed with the same interface as the E6-A and F9 systems , but a new Windows 95 3D style programming interface is under development.

Advanced features , unsurpassed ease in tuning , and more adjustability than anyone could want, get yourself some drivability as well as optimum power. Why waste your time with a hand-controller EFI system when for similar money you can have a system that can be tuned and then stashed away. Set once and forget about it.
