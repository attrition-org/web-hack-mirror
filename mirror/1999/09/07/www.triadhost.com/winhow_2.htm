Hiding Annoying ScreenTips in Windows 98 as seen when you point at a desktop icon or at the Close, Minimize, and restore icons in any window and a description of what your pointing at appears.  This annoying feature can be removed from the desktop by a simple checkbox, but you have to hack the Registry to hide the others.
Step 1:  Open Explorer and choose View, Folder Options.   Click the View tab and uncheck the box labeled Show Pop-up Description for Folder and Desktop Items.
Step 2: Click Start, Run, and type in REGEDIT.  Find the key HKEY_CURRENT_USER\Control Panel\desktop, and locate
UserPreferanceMask.  This Value consists of four groups of two characters.   You'll need to change the first character in the first value; if the value is AE 00 00 00. for example, the character you'll change is A.
Step 3:  Double-click the UserPreferenceMask value and replace the first two-character value.  Use the following list to re[place the first character: A=2, B=3, C=4, E=6, 8=0.  Do not change the second character.  For example, AE becomes 2E.
Step 4:  Exit REGEDIT
Renaming the Recycle Bin is not as easy as it is to rename your other desktop icons.   It will take a registry hack to do it.
Step 1: First back up the registry.  That way if you make a mistake you only need to restore the file to get your system back to normal.
Step 2:  Click Start button, select run, type regedit and hit ENTER.
Step 3:  Navigate to (find) HKEY_CLASSES_ROOT\CLSID\{645FF040-5081-101B-9F08-002F954E
Double-click on the default key and enter your new name for the Recycle Bin.
Step 4: Exit Regedit, click on the Recycle Bin and press F5 to make the name change appear on the desk top.
When you install IE4 or later, it adds a cascading menu to the Start Menu, letting you display the contents of your Favorites folder without opening a new window.  If your Favorites folder has been well stocked with shortcuts and subfolders, you'll quickly find out that the new menu is nearly unusable.  This usability blunder can be eliminated fortunately.
Step 1:  Bring up Regedit by clicking Start, Run, then type REGEDIT.
Step 2:  Navigate to the key HKEY_CURRENT_USER\Software\
Microsoft\Windows\CurrentVersion\Policies\Explorer Choose Edit, New, DWORD value.
Step 3:  Change the default name to NoFavoritesMenu and press Enter and change the contents of the Value Data to 1.
Step 4:  Click OK and close the Registry Editor.  The next time you start Windows the Favorites menu will be available from any Explorer window, but you won't see it on the start menu.
Ever notice that when you click on the Start button, then Programs how the menu takes a second or more to appear?  That's because Windows has a built-in delay before displaying the fly-out menus.  Follow these steps to reduce that delay for quicker service.
Step 1:  Back up the REGISTRY.
Step 2:  Bring up Regedit by clicking Start, Run, then type REGEDIT.
Step 3:  Find the key HKEY_CURRENT_USER\Control Panel\Desktop\.   Choose edit, New, String Value, and call the new value MenuShowDelay.   Right-click on the new key, choose Modify and enter a value of 1.
Step 4:  Exit Regedit.  Restart Windows click the Start button and any of the fly-out menus and see how they run!
Most Y2K problems happen because a program misinterprets a two-digit date you enter,  in a spreadsheet program like Excel for example.  One way to make Y2K problems less likely is to change the Windows Short Date format so it always displays the full year.  To make the change;
Step 1:  Open the Control Panel and select Regional Settings option.
Step 2:  Click the Date tab, and select the M/d/yyyy option in the Short Date section.
Step 3:  Click OK to make the change permanent.
