We have tried to make navigating this site as easy as possible.  All of the major areas can be reached simply by using the menu on the left or the bottom of every page.  If you are in the habit of using the 'forward' and 'backwards' buttons on your browser's icon bar, this will work most of the time, but not all of the time.  It would be best if you used our established menus to navigate our site.

If you have set your particular browser not to show graphics on web pages to speed up the page display, then the graphical menu system on the left will not be shown, and you will need to use the menu bar at the bottom of the pages.

If you would like to print out a particular page, you can use your brower's 'print' function to accomplish this.  If you are on a page which is using 'frames,' you must first click in the text you wish to print.   That action make that particular frame active and will be printed.  If you are not sure if the page uses frames, simply look at the scrollbar (if there is one) on the right side of the screen.  If the scrollbar does not extend to the complete side of the browser window, then the page is using frames.  Just to be on the safe side, always click on the text you wish to print.
