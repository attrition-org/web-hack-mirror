You can search the Sambar site using the Search engine below. To search the Sambar site, simply type in any word, set of words, or phrases you might use when speaking or writing, then click OK. Additional help is available for phrasing queries. The top 100 matching documents are listed in weighted order. The Search engine is provided by Sambar Technologies.

Tips: To exclude a word, add a minus sign directly in front of it:

To force a word to appear in a document, add a plus sign in front of it:
windows +Microsoft

The above illustrates how to search two different indexes. There is no practical limit to the number of indexes that the Sambar Server can support.
