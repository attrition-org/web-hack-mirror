Microsoft Internet Information Server

為 Windows NT Server 所設計的 Web Server

請趕快將最新功能新增到您的 Internet Information Server！

Microsoft Internet Information Server (IIS) 可簡化您與小至會議廳大至全世界的內外客戶之間的業務。 若要學習更多 Internet Information Server 協助您開拓您事業的方法，請瀏覽 IIS 線上說明文件。
您可從 Web 瀏覽器中管理 IIS 服務。若要啟動目前 Web 瀏覽器的管理工具，請按一下此處（請注意，您必須擁有管理員特權才能使用此工具）。
請試一下上面的超連結，以查看您可用 Microsoft Internet Information Server 發佈的內容範例。 若要學習可用來建立較美觀 Web 頁面的相關 Microsoft 產品，請探訪 Microsoft Web 站台以取得 Microsoft FrontPage 與 Internet Assistants for Microsoft Office 的相關資訊。Microsoft FrontPage 也收錄在 Windows NT Server CD-ROM 中。
開始瀏覽 Internet 的好地點是 www.microsoft.com， 您可在此處找到 Microsoft 產品的相關資料。此 Web 站台中含有討論 Microsoft Internet Information Server 的頁面。Microsoft Web 站台也提供了 Microsoft Internet Explorer 下載版。而另一個開始瀏覽的好地點是 The Microsoft Network Web 站台。
