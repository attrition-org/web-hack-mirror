Description:
Take-1 is a grouped application that features FTP, Telnet, an HTML Editor, e-mailing capabilities, TraceRoute/Ping and an Image Editor.

Features include:

  • A color-coded HTML editor that allows you to edit the HTML source code side-by-side in a double pane, so you can see how it will appear on the Web
  • The ability to add WordArt or Clip Art to your graphics
  • The ability to alter the image as you see fit while the original is present to compare changes with (this makes it very easy to see what effects your changes have)
  • The ability to generate HTML code for colors and graphics
  • Support for FTP, E-mail and Telnet to get your images where they need to go

Essentially, I believe this was made for the beginning Webmaster, but it also helps the more experienced user do just about anything relating to HTML and your images.
