如何於瀏覽器上顯示Web圖表

圖表能讓使用者方便地看到資料間的比較、和趨勢。若於瀏覽器上可以顯示圖表，使用者就可以透過網際網路或intranet內部網路，來查詢資料庫資料所製作出的Web 圖表。如何於瀏覽器製作Web圖表呢？

答案之一為使用Microsoft Office 2000 Web 元件。

Microsoft Office Web 元件

Microsoft Office Web 元件，讓使用者可以建立動態的互動式Web圖表，而這種Web圖表可以用瀏覽器在網路上檢視。如果變更圖表的資料，Web圖表也會自動更新。

若使用試算表元件控制項，當使用瀏覽器的使用者變更了試算表中的數值時，Web圖表將自動更新以反映出所作變更。

Web 圖表可以使用資料庫的資料，也可以使用由 Microsoft Office Web 元件建立的試算表或樞紐分析表清單中的資料。

如果要檢視和處理圖表，使用者必須安裝 [Microsoft Office Web 元件] ，可以透過安裝 Microsoft Office 2000 來安裝這些元件，或者，如果公司具有 Office 2000 授權允許，則可以透過公司的企業內部網下載 [Microsoft Office Web 元件] 。使用者必須具有 Microsoft Internet Explorer 4.01 版或更高版本才可以處理圖表。

Web 圖表的類型

Microsoft Office Web 元件所提供的Web 圖表的類型如下：

區域圖：

區域圖顯示繪圖值的和，以及顯示部分與整體之間的關聯。

2. 直條圖：

直條圖顯示了一段時間內的資料變化情況，或說明了各項之間的比較情況。

堆疊直條圖顯示了單項與整體之間的關聯。

3. 長條圖：

長條圖說明了各單項之間的比較情況。

堆疊橫條圖顯示了各單項與整體之間的關聯。

4. 折線圖：

折線圖等間距地顯示了資料的預測趨勢。

5. 平滑圖：

平滑圖等間距地顯示了資料的預測趨勢，並透過平滑設定弧形顯示了間距間的估計值。

6. 圓形圖：

圓形圖顯示了各項相對於項總和的比例大小。

可以建立具有多個數列的堆疊圓形圖。

7. 環圈圖：

像圓形圖一樣，環圈圖也顯示部分與整體之間的關聯，但環圈圖可以包含多個值數列，環圈圖的每個環都代表一個值數列。

8. 股票圖：

兩種股價的股票圖：開盤-盤高-盤低-收盤圖和盤高-盤低-收盤圖。

9. XY 散佈圖：

XY 散佈圖可以顯示多個值數列中數值之間的關聯，也可以將兩組數字繪圖成一系列 XY 座標。

10. 泡泡圖：

泡泡圖是一種 XY 散佈圖。

11. 雷達圖：

在雷達圖中，每個類別都有自己的值軸，此值軸由中心點向外發散。折線連接了相同數列中的所有值。

12. 極座標圖：

極座標圖可比較角度和距離之間的關聯。

如何製作Web圖表呢？

如何製作以上各種Web圖表呢？

Web圖表與資料相結合的方式如下：

1. 於陣列設定資料

2. 使用字串設定資料

3. 使用 ADO 記錄集中的資料

4. 使用 Microsoft Office 試算表中的資料

5. 使用 Microsoft Office 資料來源控制項中的資料

分別說明如下：

於陣列設定資料

於陣列中設定資料建立圖形的方法，首先將陣列設定為類別資料和值資料，然後透過 SetData 方法設定圖形資料。

於陣列中設定資料之HTML碼與說明如下：

<object id=ChartSpace1 classid=CLSID:0002E500-0000-0000-C000-000000000046 style="width:100%;height:350"></object>

 

<script language=vbs>

Sub Window_OnLoad()

Dim categories(3), values(3)

 

' 建立具有二個數列和四季的直條圖，用於顯示每季的營業額淨利

 

' 建立代表類別的字串陣列。

' 所有二個數列的類別都一樣。

categories(0) = "Q1"

categories(1) = "Q2"

categories(2) = "Q3"

categories(3) = "Q4"

 

' 清除圖形工作空間中的內容。這將移除任何既存圖形，並徹底清空圖形工作空間。

' 隨後再新增一個圖表物件。

ChartSpace1.Clear

ChartSpace1.Charts.Add

Set c = ChartSpace1.Constants

 

' 向圖形中新增二個數列。

ChartSpace1.Charts(0).SeriesCollection.Add

ChartSpace1.Charts(0).SeriesCollection.Add

   

' 第一個數列中包含每季的淨利資料。

' 設定數列的標題（圖例中顯示的文字）。

'ChartSpace1.Charts(0).SeriesCollection(0).Caption = "淨利"

 

' 設定第一個數列的類別（此集合最小為零）

'ChartSpace1.Charts(0).SeriesCollection(0).SetData c.chDimCategories, c.chDataLiteral, categories

 

values(0) = 120 ' Q1

values(1) = 150 ' Q2

values(2) = 190 ' Q3

values(3) = 250 ' Q4

 

ChartSpace1.Charts(0).SeriesCollection(0).Caption = "淨利"

ChartSpace1.Charts(0).SeriesCollection(0).SetData c.chDimCategories, c.chDataLiteral, categories

ChartSpace1.Charts(0).SeriesCollection(0).SetData c.chDimValues, c.chDataLiteral, values

 

' 第二個數列中包含每季的營業額資料。

' 更新值陣列，然後設定圖形資料。

values(0) = 420 ' Q1

values(1) = 500 ' Q2

values(2) = 630 ' Q3

values(3) = 940 ' Q4

 

ChartSpace1.Charts(0).SeriesCollection(1).Caption = "營業額"

ChartSpace1.Charts(0).SeriesCollection(1).SetData c.chDimCategories, c.chDataLiteral, categories

ChartSpace1.Charts(0).SeriesCollection(1).SetData c.chDimValues, c.chDataLiteral, values

 

' 使圖表圖例可見，將左數值座標軸的格式設定為百分比，

' 並指定數值格線的間隔為 10%。

ChartSpace1.Charts(0).HasLegend = True

ChartSpace1.Charts(0).Type = 0

ChartSpace1.Charts(0).Axes(c.chAxisPositionLeft).NumberFormat = "000"

ChartSpace1.Charts(0).Axes(c.chAxisPositionLeft).MajorUnit = 100

End Sub

 

 

Sub next1()

if ChartSpace1.Charts(0).Type = 45 then

        ChartSpace1.Charts(0).Type = 0

else

        ChartSpace1.Charts(0).Type = ChartSpace1.Charts(0).Type + 1

end if

form1.bu1.value = "Type = " & ChartSpace1.Charts(0).Type

End Sub

</script>

<form name=form1>

<input name=bu1 type=button value="Type = 0" onclick=next1()>

</form>

使用字串設定資料

除了於陣列中設定資料建立圖形的方法外，您也可以首先用Tab﹝Chr(9)﹞分隔為類別資料和值資料，然後透過 SetData 方法設定圖形資料。譬如：

categories = "Q1" & Chr(9) & "Q2" & Chr(9) & "Q3" & Chr(9) & "Q4"

使用 ADO 記錄集中的資料

您可以使用 ADO 記錄集連結到使用者電腦資料庫中的資料譬如：

<object id=ChartSpace1 classid=CLSID:0002E500-0000-0000-C000-000000000046 style="width:100%;height:350"></object>

<object id=ADOConnection1 classid=CLSID:00000514-0000-0010-8000-00AA006D2EA4></object>

 

<script language=vbs>

Sub Window_OnLoad()

    Dim rs, categories, values

   

    categories = ""

    values = ""

 

    ' 開啟連接並執行查詢。

    ADOConnection1.Open "DRIVER={Microsoft Access Driver (*.mdb)};DBQ=c:\test.mdb"

   

    Set rs = ADOConnection1.Execute("SELECT * FROM FLOWCTL")

 

    rs.MoveFirst

    While Not rs.EOF

        categories = categories & rs.Fields(0).Value & Chr(9)

        values = values & rs.Fields(1).Value & Chr(9)

        rs.MoveNext

    Wend

    rs.Close

    ADOConnection1.Close

   

    ' 刪除字元串末尾的額外定位字元。

    categories = Left(categories, Len(categories) - 1)

    values = Left(values, Len(values) - 1)

   

    ' 建立只有一個銷售數列的圖形。

    ChartSpace1.Clear

    ChartSpace1.Charts.Add

    ChartSpace1.Charts(0).SeriesCollection.Add

    ChartSpace1.Charts(0).SeriesCollection(0).Caption = "銷售"

 

    '使用根據記錄集建立的字串設定數列類別和值。

    Set c = ChartSpace1.Constants

    ChartSpace1.Charts(0).SeriesCollection(0).SetData c.chDimCategories, c.chDataLiteral, categories

    ChartSpace1.Charts(0).SeriesCollection(0).SetData c.chDimValues, c.chDataLiteral, values

   

    ' 設定圖表類型並將座標軸設定為圓為。

    ChartSpace1.Charts(0).Type = 18

    ChartSpace1.Charts(0).Axes(c.chAxisPositionBottom).NumberFormat = "$#,##0"

End Sub

</script>

使用 Microsoft Office 試算表中的資料

使用 Microsoft Office 試算表中資料的方法，首先用 DataSource 屬性將圖形與試算表相結合，然後使用 SetData 方法設定來自試算表區域中的圖形資料。

使用試算表元件控制項，當使用瀏覽器的使用者變更了試算表中的數值時，Web圖表將自動更新以反映出所作變更。

使用 Microsoft Office 試算表中資料之HTML碼與說明如下：

<object id=ChartSpace1 classid=CLSID:0002E500-0000-0000-C000-000000000046 style="width:100%;height:350"></object>

 

<object id=Spreadsheet1 classid=CLSID:0002E510-0000-0000-C000-000000000046 style="width:49%;height:350"></object>

 

<script language=vbs>

Sub Window_OnLoad()

    ' 先將資料放入 [Spreadsheet1] 中。

    Spreadsheet1.ActiveSheet.Cells.Clear

    Spreadsheet1.ActiveSheet.Cells(2, 1).Value = "Q1"

    Spreadsheet1.ActiveSheet.Cells(3, 1).Value = "Q2"

    Spreadsheet1.ActiveSheet.Cells(4, 1).Value = "Q3"

    Spreadsheet1.ActiveSheet.Cells(5, 1).Value = "Q4"

   

    Spreadsheet1.ActiveSheet.Cells(1, 2).Value = "淨利"

    Spreadsheet1.ActiveSheet.Cells(2, 2).Value = 100

    Spreadsheet1.ActiveSheet.Cells(3, 2).Value = 120

    Spreadsheet1.ActiveSheet.Cells(4, 2).Value = 134

    Spreadsheet1.ActiveSheet.Cells(5, 2).Value = 150

   

    Spreadsheet1.ActiveSheet.Cells(1, 3).Value = "營業額"

    Spreadsheet1.ActiveSheet.Cells(2, 3).Value = 540

    Spreadsheet1.ActiveSheet.Cells(3, 3).Value = 670

    Spreadsheet1.ActiveSheet.Cells(4, 3).Value = 777

    Spreadsheet1.ActiveSheet.Cells(5, 3).Value = 989

 

' 清除圖形工作空間中的內容。這將移除任何既存圖形，並徹底清空圖形工作空間。

' 隨後再新增一個圖表物件。

ChartSpace1.Clear

ChartSpace1.Charts.Add

Set c = ChartSpace1.Constants

 

    ' 將圖形的 DataSource 屬性設定為試算表。

    ' 可指定多個值源，但本範例只用一個。

    ChartSpace1.DataSource = Spreadsheet1

       

    ' 向圖形中新增三個數列。

    ChartSpace1.Charts(0).SeriesCollection.Add

    ChartSpace1.Charts(0).SeriesCollection.Add

 

 

    ' 第一個數列中包含每季的淨利資料。

    ' 請將數列名稱、類別名稱和數值結合起來。

    ChartSpace1.Charts(0).SeriesCollection(0).SetData c.chDimSeriesNames, 0, "B1"

    ChartSpace1.Charts(0).SeriesCollection(0).SetData c.chDimCategories, 0, "A2:A5"

    ChartSpace1.Charts(0).SeriesCollection(0).SetData c.chDimValues, 0, "B2:B5"

   

    ' 第二個數列中包含每季的營業額資料。

    ChartSpace1.Charts(0).SeriesCollection(1).SetData c.chDimSeriesNames, 0, "C1"

    ChartSpace1.Charts(0).SeriesCollection(1).SetData c.chDimCategories, 0, "A2:A5"

    ChartSpace1.Charts(0).SeriesCollection(1).SetData c.chDimValues, 0, "C2:C5"

 

ChartSpace1.Charts(0).HasLegend = True

ChartSpace1.Charts(0).Type = 0

ChartSpace1.Charts(0).Axes(c.chAxisPositionLeft).NumberFormat = "000"

ChartSpace1.Charts(0).Axes(c.chAxisPositionLeft).MajorUnit = 100

使用 Office 資料來源控制項中的資料

使用 Microsoft Office 資料來源控制項中的資料，資料來源控制項用於連接至資料庫並執行查詢，然後將圖形工作空間資料來源設定為資料來源控制項，建立的記錄集設定為資料成員，最後透過 SetData 方法設定圖形的類別和值。

譬如HTML碼與說明如下：

<object id=ChartSpace1 classid=CLSID:0002E500-0000-0000-C000-000000000046 style="width:100%;height:480"></object>

<object id=DataSourceControl1 classid=CLSID:0002E530-0000-0000-C000-000000000046></object>

 

<script language=vbs>

Sub window_onLoad()

    Dim rsd

 

    DataSourceControl1.ConnectionString = "DRIVER={Microsoft Access Driver (*.mdb)};DBQ=c:\TEST\test.mdb"

    Set rsd = DataSourceControl1.RecordsetDefs.AddNew("SELECT * FROM TEST", 3)

 

    Set c = ChartSpace1.Constants

    ChartSpace1.DataSource = DataSourceControl1

    ChartSpace1.DataMember = rsd.Name

    ChartSpace1.Charts.Add

    ChartSpace1.Charts(0).Type = c.chChartTypeBarClustered

    ChartSpace1.Charts(0).SetData c.chDimCategories, 0, "lname"

    ChartSpace1.Charts(0).SetData c.chDimValues, 0, "job_lvl"

End Sub

</script>
