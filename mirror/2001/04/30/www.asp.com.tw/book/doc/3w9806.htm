Dynamic HTML簡介

讓我們先看看現有的HTML，是一個「靜態」的網頁設計語言，能夠提供顯示圖文的功能，但是不易提供一些多媒體的效果，譬如二度空間的移動、動畫、混音、即時互動式等。

Netscape Communicator 4.0版本、和Microsoft Internet Explorer 4.0（簡稱IE 4.0）版本，都宣稱提供了「Dynamic HTML」（動態HTML）的功能，到底「Dynamic HTML」是什麼東西呢？

Dynamic HTML是「動態」的，讓您不須經過網站伺服器的處理，於瀏覽器中直接動態地更新網頁的內容（content）、和排版樣式（style）、動畫等。

IE 4.0、與Netscape 4.0所宣稱的Dynamic HTML內容，並不相同，以下介紹這兩種4.0版本瀏覽器的Dynamic HTML：

IE 4.0版本

Microsoft Internet Explorer 4.0版本的「Dynamic HTML」，提供了以下的功能：

  1. 動態內容（Dynamic Content）：動態地更新網頁的內容（content），可「動態」地隨時插入、修改、或刪除網頁的元件，譬如文字、標記等。

  2. 動態排版樣式（Dynamic Styles Sheets）：透過W3C的「Cascading Style Sheets」（串聯式排版樣式，簡稱CSS1或CSS），提供了設定HTML標記的字體大小、字型、粗細、字型樣式、行高度、文字顏色、加底線或加中間橫線、與邊緣距離、靠左右或置中、縮排、背景圖片或顏色等「排版」功能。而「動態排版樣式」，可以「動態」隨時地改變排版樣式。

  3. 動態定位（Dynamic Positioning）：透過W3C的 「Working Draft on Positioning HTML with Cascading Style Sheets」，提供HTML元件於X、Y、Z軸的定位功能，讓設計者可以放置影像、Controls、文字等於網頁上的任何位置上。放置於不同的Z軸上，設計者可以產生重疊的效果。

  4. 內建資料處理（Data Awareness）：不須複雜的程式，不須花伺服器太多資源，即可讓網頁設計者可以即時處理資料。

  5. 內建多媒體支援：IE 4.0結合CSS與內建的ActiveX Controls，提供多媒體支援的功能，包括轉換特效、濾鏡特效、路徑控制、順序控制、動畫、製圖、播放聲音和影像等多媒體功能。

Netscape 4.0版本

Netscape Communicator 4.0版本的「Dynamic HTML」，提供了以下功能：

  1. 動態排版樣式（Dynamic Styles Sheets）：支援W3C的「Cascading Style Sheets」（串聯式排版樣式，簡稱CSS1或CSS）、和自創的「JavaScript Style Sheets」，「JavaScript Style Sheets」可使用JavaScript控制Style Sheets。

  2. Dynamic Layers：提供圖文定位、改變圖文重疊順序、控制隱藏或顯示圖文、移動圖文的功能，讓您設計出「動畫」的效果！

  3. Dynamic Fonts：提供由伺服器下載字型的功能。

Dynamic HTML差異

雖然IE 4.0、和Netscape Communicator 4.0都提供了「Dynamic HTML」的功能，但是事實上兩者除了「Cascading Style Sheets」的規格相近外，其餘都完成相異。

IE 4.0並未支援Netscape Communicator 4.0的JavaScript Style Sheets，而使用自己的動態排版樣式，可使用VBScript、或JavaScript來控制Style Sheets。

IE 4.0並未支援Netscape Communicator 4.0的Dynamic Layers，IE 4.0有自己的多媒體支援方式，使用與CSS結合和ActiveX Controls方式。

IE 4.0下載字型的規格也與Netscape Communicator 4.0不同。

Netscape Communicator 4.0並未支援IE 4.0的動態內容、動態排版樣式、動態定位、內建資料處理、內建多媒體支援。

以下將簡介「Dynamic HTML」。

動態內容簡介

Dynamic Content（動態內容）

Dynamic Content（動態內容），提供動態地更新網頁的內容（content）的功能，載入網頁的內容後，仍可「動態」地隨時插入、修改、或刪除網頁的元件，譬如標記、文字等。

IE 4.0所謂的「動態內容」，就是可以讓VBScript、JavaScript等程式能夠改變網頁的內容。

Dynamic Content提供的功能，如下：

  • 插入、修改、或刪除文字。

  • 插入、修改、或刪除標記。

  • 文字範圍物件（text range object），提供您搜尋文字、選擇與捲動文字、比較文字等進階的功能。

動態排版樣式簡介

Dynamic Styles（動態排版樣式）

Cascading Style Sheets（串聯式排版樣式，簡稱CSS1或CSS），提供設定HTML標記的字體大小、字型、粗細、字型樣式、行高度、文字顏色、加底線或加中間橫線、與邊緣距離、靠左右或置中、縮排、背景圖片或顏色等的「排版」功能。

所謂的「動態」，就是可以讓程式能夠改變網頁的內容；而「動態排版樣式」就是程式能夠控制Style Sheets（排版樣式），即可使用VBScript、JavaScript等來控制Style Sheets。「Dynamic Styles」讓您能夠「動態」地改變網頁上「任何的HTML元件」。您可以改變文字的字體大小、字型、間隔、縮排、位置、甚至可見或暫時隱藏。

為了讓程式可以控制Style Sheets，IE 4.0新增加「Dynamic Styles」（動態排版樣式）的功能，使用VBScript、JavaScript等程式語言來「動態」地控制Style Sheets。

IE 4.0提供由伺服器下載字型的功能。

動態定位簡介

Dynamic Positioning（動態定位）

所謂的「動態」，就是可以讓程式能夠改變網頁的內容；而「動態定位」就是可使用VBScript、JavaScript等程式能夠動態地控制定位（Positioning）。

IE 4.0的Dynamic Positioning（動態定位），透過W3C的 「Working Draft on Positioning HTML with Cascading Style Sheets」，提供HTML元件於X、Y、Z軸的定位功能，讓設計者可以放置影像、Controls、文字等於網頁上的任何位置上。放置於不同的Z軸上，設計者可以產生重疊的效果。

使用JavaScript、VBScript語言，來控制HTML元件於X、Y、Z軸的放置，並配合Dynamic Styles（動態排版樣式）、與IE 4.0內建透明的ActiveX Controls的使用，設計者可以提供移動圖形文字的功能。

內建資料處理簡介

資料處理存取

若要由使用者的瀏覽器中，經過Internet或Intranet，存取和處理伺服器的資料庫資料，IE 4.0內建兩個ActiveX Control，如下：

  • 處理資料：Tabular Data Control（簡稱TDC）將一個文字（text）檔案的資料顯示出來，並可將資料過濾搜尋和排序，與其他的HTML標記結合，動態地更新網頁的內容。

  • 存取資料：Advanced Data Connector（簡稱ADC）由使用者的瀏覽器中，經過Internet或Intranet，存取伺服器的資料庫之資料，可存取各種ODBC（Open Database Connectivity）資料庫。

內建多媒體支援簡介

IE多媒體支援

IE 4.0結合CSS（Cascading Style Sheets，串聯式排版樣式），提供的多媒體支援，包括：提供由一張圖片，轉換到另一張圖片的轉換特效（Blend Transition Filter、Reveal Transition Filter）、以及產生模糊、顛倒、漸層、陰影、灰階、投射光源、陰影、罩幕、波浪、X光等特效的濾鏡特效（Visual Filter effect）。

IE 4.0提供許多內建支援多媒體的ActiveX Controls，包括：可以設定移動的路徑的路徑控制（Path Control）、設定ActiveX controls或script語言等各種動作之執行時間順序功能的順序器控制（Sequencer Control）、產生類似播放電影軟片的動畫效果的精靈控制（Sprite Control），以及提供播放聲音和影像功能的影片機控制（ActiveMovie Control）。

Dynamic Layers簡介

Netscape Communicator 4.0，提供一個<LAYER>的HTML標記，代表一段圖文，提供圖文定位、改變圖文重疊順序、控制隱藏或顯示圖文、移動圖文、改變LAYER的背景顏色和圖片等功能，讓您設計出「動畫」的效果。若配合JavaScript語言，即可隨心所慾地控制動畫圖文。
