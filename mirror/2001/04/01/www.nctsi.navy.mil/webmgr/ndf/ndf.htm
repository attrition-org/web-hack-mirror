Navy Center for Tactical Systems Interoperability

                 JTIDS/MIDS

             Network Design Facility

Purpose:The Navy Network Design Facility provides JTIDS/MIDS network designs that support TADIL J information exchange requirements of authorized network users.  In order to accomplish this mission, the NDF gathers requirements for network designs from the operating forces then validates, designs, tests, and delivers networks to support the requirements. NCTSI coordinates with other Navy, Joint, and Allied commands as required for network design requirements, validation, distribution control, and configuration management.

Network Request: Here you will find an interactive form that allows you to request a modification to an existing network or request a new new network, all on-line.

Network Documents: You have access to all the Network Documentation from the latest Networks, this includes Operational, Exercise, R&D and Test Networks. You can also find other JTIDS/Link-16 Documents here.

Network Loads: If you are EJNL capable you can download Network Load Files for Ship, E-2C, F-14D, F/A-18, and LMS-16. There are Network Load Files for Operational, Exercise, R&D and Test Networks

Network Index: This is were you will find the JNL-200 Quick Reference Guide (QRG). The QRG gives a brief description of all the networks contained in a JNL. For each Network a Connectivity Matrix and a table of the Design Options are given (NPG-7, NPG-9, NPG-19). The Network Index also includes brief descriptions of  Networks that were designed during or after the release of JNL-200.

JTIDS Links: Links to other JTIDS/MIDS sites, including Army, Airforce, and Marine Network Design Facilities.

Glossary: A glossary of JTIDS/MIDS terms and definitions

Contacts: Contains phone numbers, email addresses, fax numbers and mailing address of the Navy JTIDS/MIDS Network Design Facility.
