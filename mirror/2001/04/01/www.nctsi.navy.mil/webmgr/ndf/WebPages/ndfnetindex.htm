US Navy JTIDS Network Design Facility

JNL 200 Quick Reference Guide(QRG)

The JTIDS Network Library (JNL) is a collection of the latest approved Navy Networks.  Each network in the JNL contains the load files for each participant in the network.  In order to support the varied requirements of users, the JNL contains an ever changing number of networks.  These network designs will be maintained and updated to match changes in data link specifications, system enhancements, unit capabilities, changes in theater requirements (including pulse density restrictions on deconfliction), changes in the geopolitical environment, and changing mission expectations of Commanders.

The JNL consists of the following files for each network:

  • Network Identification File
  • Mapping files map J series messages to NPGs
  • Platform files define the transmit, receive, and relay assignments for network participants (one per participant).
  • Design files define transmit assignments for network participants in NPGs (Surveillance, Air Control Backlink, Fighter-to Fighter Targets) where a variety of options have been provided.

The QRG gives a brief description of all networks contained in the JNL.   For each network a Connectivity Matrix, Time Line, and a table of the Design options are given.  The QRG also contains sections on the Network Management Process and a Link-16 overview.

  • The Connectivity Matrix is the form used to specify a network's transmissions.
  • The Network Time Line allows one to see at a glance how network capacity has been divided among operational/technical functions.
  • The information presented in the table of design options indicates the options available for NPG 7 (Surveillance), NPG 9 (Air Control Backlink), and NPG 19 (Fighter-to-Fighter Targeting).

 

*To use the QRG as a planning tool, click on the network number that you want to view.  This will give you a description of the Connectivity Matrix, design options for Surveillance, Air Control Backlink, Fighter-to-Fighter Targeting, and a brief description of the network.

*If after viewing the networks available on JNL 200 none meet your requirements, go to the NDF web page to get the latest list of networks that were distributed after JNL 200.  If the list of networks does not meet your requirements, call the NDF or complete the Network Design Request page.  Follow the instructions for requesting a modified or new network.

NOTES:

Surv = Surveillance track capacity (approximate) for entire Network and is based on CDS Model 4 Protocols.  CDS Model 5 units will have an approximately 20% greater throughput (assumes a 12 second update rate).

A/C Back = Air Control Backlink target capacity per net.

FTR-FTR = Fighter to Fighter target capacity per net.

Data Fwd = Number of simultaneous Data Forwarding Units.

Voice = 16 Kbps unless otherwise noted, /R is relayed.
