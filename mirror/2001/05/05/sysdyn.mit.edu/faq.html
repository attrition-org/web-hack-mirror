FAQ about Problems related to sysdyn.mit.edu

During the years we have received a number of requests for help with problems concerning our Website, or files originated from it.  Here is a list of the most commonly asked questions.
 

Q:  I have been trying to log on to the System Dynamics in Education Project, but I keep getting the message "Error 403 Forbidden by rule". 
A:  To solve this problem we need more information from you. If it is possible, please E-mail webmasters@sysdyn.mit.edu with the following information: 
  1. Name of web browser 
  2. IP Address of computer used to contact us 
  3. Address of web pages that gave you this error 
  4. Time of day the error occurred 
  5. Name of documents you were unable to download 
  6. Where you got the addresses used to contact us i.e. a link from another web page? 
  7. Are you able to connect to the main page at "http://sysdyn.mit.edu/"? 
 
Q:  Adobe Acrobat does not open when I open a PDF document from this web site using my AOL account.
A:  If you are using Netscape via AOL then set up Adobe Acrobat as the default pdf reader by choosing the "General Preferences" sub menu via the "Options" menu that is located on the main menu bar at the top of the Netscape program window.  This action will bring up the preferences window,  which has with a tab column at the top.  Choose the "Helpers" tab.  Under this tab will be a window with a scroll bar.  Scroll down this window until you see row entitled "application/pdf".  Highlight this row with the mouse button.  Below this window you 
will see radio buttons entitled "Action:"  Choose the radio button entitled "Launch the Application:".  Beneath the radio buttons you will see a command button entitled "Browse".  Click onto this button; a window of your computer's hard drive directories will pop up. 
Use this directory window to search for the acrobat executable file (most likely located in the directory "c:\acrobat".  Highlight the Acrobat  executable file with your mouse then click the "OK" command button.  Then click  the "OK" button of the "Preferences" windows.  After this procedure Netscape will load acrobat automatically to view the documents. 

If you are using another browser besides Netscape then please inform me of the browser type.  Any questions regarding the operation of AOL or any of its configuration files needs be directed to an AOL consultant. 
 
 
Q:  When I try printing a paper that was downloaded from your site, the diagrams do not print and in their place there is only a square.
A:  This may not be much help, but I may be able to help you narrow down the source of the problem.  If the causal loop diagrams show up on your computer screen, then the problem 
should be between printing from Acrobat and the printer itself.  I suggest playing with the print settings in Acrobat.  On my version, there are two Postscript options.  Mine is set to "Level 2 only."  If that does not help, then the problem may be with the way that your printer is set up. 
 
 
Q:  I am having problems printing graphics in Road Maps papers.
A:  Solutions to this problem could be found in the Adobe "Read Me" file. There is information in that file about printing anomalies with different kinds of printers.
