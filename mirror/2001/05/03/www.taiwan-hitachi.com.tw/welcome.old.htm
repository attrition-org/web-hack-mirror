歡迎使用 Microsoft(R) Personal Web Server 4.0

此首頁是置於 Microsoft Personal Web Server (PWS) 4.0 上。PWS 會將執行 Windows(R) 95 或 Windows 98 的任何電腦轉換成 Web 伺服器，並啟動個人網頁的立即發行功能。因為容易安裝與管理，所以 PWS 可簡化公司 intranet 或 Internet 上所有使用者的資訊共用功能。PWS 最適合用於容量小、點對點的 Web 發行。使用新的 PWS 發行精靈，您可快速發行文件。

此為預設首頁。執行 Personal Web Server 首頁精靈，即可建立屬於自己的自訂首頁。請啟動 [Personal Web 管理員]，並按一下 [Web 站台] 圖示以啟動首頁精靈。

PWS 已將互動式工作站使用最佳化，且不像 Microsoft Internet Information Server 有系統方面的需求。若要取得您桌面電腦適用的 Personal Web Server，請探訪 http://backoffice.microsoft.com/downtrial/optionpack.asp。
