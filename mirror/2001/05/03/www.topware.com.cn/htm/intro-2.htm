彤云证券业务管理系统
 


　　彤云证券业务管理系统采用三层或多层客户/服务器技术。前端应用与中间件服务之间交互是建立在功能协议、安全控制协议、事务处理协议和运行环境管理的基础上（本系统依赖于网络通讯协议、消息队列管理和密钥管理协议），这四个软件构成了可以灵活扩展应用功能、增强处理能力、提高安全级别、保证可靠性处理和协调运作负荷的功能，充分发挥了三层应用结构的优势，真正实现了证券系统安全防范三分离。
　　系统包括七大子系统，为各证券公司提供了丰富的管理功能。除证券交易业务子系统、业务统计与分析子系统、银行转帐子系统、业务管理子系统、法人清算子系统、系统管理分析子系统外，还设计了严密的交易业务风险监控子系统，针对不同风险采用不同的应对策略，彻底提高证券经营机构的抗风险能力。
　　本系统采用大型数据库，支持NOVELL、NT、UNIX三种系统平台，系统功能强大，界面友好且操作简单。就应用平台而言，现有PC平台和小型机平台之分：
PC平台
硬件平台：HP、康柏、DELL、IBM等各种品牌的PC Server
操作平台：Windows NT
数据库：MS SQL Server
安全保障：NCR Lifekeeper、VINCA StandbyServer、MS Wolfpack
小型机平台
硬件平台：IBM RS/6000
操作平台：IBM AIX (UNIX的IBM版本）
数据库：Sybase
安全保障：小型机高并行的处理能力、极高的可靠性、灵活的伸缩性； RS/6000独有的HACMP(灾难备份系统）。
