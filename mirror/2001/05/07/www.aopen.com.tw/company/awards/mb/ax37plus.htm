  • 建碁 AOpen DDR 主機板 AX37Plus 獲科技網站 Maximum3D 四顆半星超高評價
    建碁     AOpen 之 AX37Plus 建立了一個偉大的作業平台.  如果你想昇級到 DDR, VIA 的 Apollo Pro266 是最簡易的方案, 而建碁 AOpen 之 AX37Plus 則提供了應有盡有的內容.  AX37Plus 內建之 Promise 磁碟陣列是相當好的加值功能, 0與1 之簡易功能讓大家都會使用.  AX37Plus 之BIOS 簡單易用, 而 1Mhz 外頻微調加上調電壓功能, 更使得 CPU 表現臻於極致.  整體來說, AX37Plus 對於想要昇級到DDR 的使用者, 是個很實在的選擇.


    摘錄自     :     Maximum3D, 5/4/2001

  • 狂賀!!! 建碁 AOpen DDR 主機板 AX37Plus 榮獲美國知名科技網站 Active Hardware 評為 "最高等獎" (Top Grade Hardware Award )
    AOpen     的 AX37Plus 主機板是我們測試過支援 VIA Pro266主機板中之最優秀者.  它的設計概念極佳, 在我們的分析測試中表現十分優異
