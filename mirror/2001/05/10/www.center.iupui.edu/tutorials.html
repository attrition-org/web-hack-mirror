HTML Basics Tutorial

The purpose of this online tutorial is to give you a working knowledge of basic HTML (primarily version 3.2 standards) so that you can create home pages manually, if you so choose, or understand how to "debug" HTML editor "confusions" when necessary. It covers tag structures, how to locate and use images, and contains a complete HTML file that you can refer to (or copy to practice!)

Beyond HTML Basics

Setting up a basic home page is a relatively easy task. It's even fairly easy to get some simple graphics in place on your pages. Sooner or later, however, you will want to go beyond the basics. This online tutorial covers using options with basic HTML tags, creating ISMAPS, designing tables, and using color effectively.

Introduction to Windows 95 Tutorial

This online tutorial introduces you the operating system called Windows 95. It discusses the Windows work environment, file, directory and windows manipulation, color and arrangement set-ups, as well as primary Windows utilities and features. Also is provides an introduction to computers and disks, in general.

How to Manage your LISTSERV List Effectively

This online tutorial discusses general discussion list issues, walks you through the online list application and explains the available options, discusses managing subscribers, and explains how to change list options.

Tips for On- camera Presenters

These tips were developed from various I.U. sources and faculty who have presented successfully in the Virtual Indiana Classroom (VIC)—the two-way video/audio broadcast rooms. The hope is that they will serve as a guide to effective preparation and presentation practices for the novice VIC user, as well as a reminder for seasoned VIC presenters. The general practices contained here are generally effective for any on-camera presenter.

Creating Effective Visual Aids

The purpose of this online tutorial (created using PowerPoint's web option) is to introduce you to basic color, typography, and design principles so that you design more effective presentations. It offers tips and techniques for developing and delivering presentations, both general and electronic.

PowerPoint 97 Basics

This online tutorial introduces you to the basic features of PowerPoint 97, including Masters, animation, layers, and saving as HTML.

Power Point 2000 Basics

This online tutorial introduces you to the basic features of PowerPoint 2000, including Masters, animation, layers, and saving as HTML, in addition to introducing the new or changed features in 2000.

Using WS_FTP

This online tutorial teaches you how to download and install WS_FTP and use that application to transfer files, primarily between your local workstation and your designated location on a web server.

Creating Web Pages with Word 97 HTML Editor

This online tutorial teaches you to use the HTML editor that is built into Word 97. It uses screen captures to visually explain toolbars and dialog boxes.

Creating Web Forms

Building upon your knowledge of HTML and web design, this online tutorial teaches you to create a web form.

Using Transform to Process Web Forms

Transform is a utility developed to help you make the forms a functioning tool in your technology toolbox. (NOTE: Transform is only available to those whose web pages are on the IUPUI champion.iupui.edu web server.)

UNIX Basics

UNIX Basics covers the basic UNIX commands necessary to set up and work with web pages and files on the IUPUI CHAMPION and CORD central web servers.
