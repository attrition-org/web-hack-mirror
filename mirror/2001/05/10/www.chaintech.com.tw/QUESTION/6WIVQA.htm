6WIV Q&A
Q1. Why can't power off system after Win98 software shutdown ?
A1. Because driver conflict issue , don't install driver IDE 1&2 of enclose CD under Win98 SE. And just need to install security driver.
Q2. Why doesn't it have SPP item in BIOS setup ?
A2. The point is NS I/O chipset not support SPP function. User can set as ECP mode to work with more popular printer.
