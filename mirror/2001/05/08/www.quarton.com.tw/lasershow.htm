Controlling the Laser Device by using LaserShop

The main purpose of INFINITER LaserShop is to transmit graphical objects and animations created by you to the laser device attached to the parallel port (Printer Port) so it's easy to operate the software with PC or NoteBook. The laser device will then project your work on a wall. Understanding the basic operation of this laser device is important to a successful use of LaserShop.The galvo-based optical scanner in the laser device generates a single light beam (or laser beam) which initially is sent to a certain position on the wall. This position, called point, equals one pixel in your draw area, and is the start point of an object consisting of one or several lines which are made up of points (pixels). These lines, called paths, are projected by the laser beam pixel by pixel from the start point to the end point of each single line. The time the laser beam projects one point after the other on the wall, passes so quickly that the human eye perceives continuous lines. The speed the laser device is scanning paths of objects is adjustable .Using the LaserShop function, the most of difficult digitizing task of laser pattern created are now made easy.The line drawings and solid objects with bitmap (BMP) format can transfer to laser patterns automatically.

Operation Mode

PC Master: Activates the hardware by connecting the laser device to the parallel port of your
                  computer. This button enables you to control the position of laser beams using
                  Laser Show or any other ILDA-compliant laser device in real-time.

Transmit: Uploads the data created using LaserShop (draw file, Marquee file or animation file) to
                the flash memory on the built-in DA control card.

Memory Display: Immediately displays the laser pattern previously uploaded.
