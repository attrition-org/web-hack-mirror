Applications

  • Serbase est un ensemble complet de gestion commerciale et de partage des informations pour P.M.E. Le client fonctionne indifféremment sous environnement Microsoft Windows ou Unix/X11. Il permet les opérations usuelles de gestion des stocks, livraison, facturation, suivi des clients et fournisseurs, mais permet également une gestion centralisée du courrier et de manière générale le partage de documents électroniques. L'installation de nouveaux postes de travail est particulièrement simple, grâce à une interface client distribuée "à la demande" lors d'une nouvelle connexion.
  • Semafor est un environnement de programmation Fortran, intégrant des outils originaux de calcul formel. Il vous permet d'introduire dans vos codes numériques une sémantique mathématique de niveau supérieur, afin de traduire plus facilement les modèles mathématiques sous-jacents.

Systèmes embarqués

  • noyau Linux
  • librairie C compacte
  • support Tcl intégré

Le composants Linux (noyau) et Tcl sont des versions complètes, amputées d'aucune fonctionnalités. La librairie C, le support pour librairies dynamiques, et le système de fenétrage utilisé sont eux propriétaires. La distribution occupe moins de 1,44Mo. Une version d'évaluation sera prochainement disponible pour téléchargement sur notre site.

Librairies

  • Serik permet à vote code d'effectuer aisément des opérations de manipulation d'images. Il offre en standard le support pour la majorité des formats graphiques usuels, aussi bien en lecture qu'en écriture.

Extensions Tcl/Tk

 Tcl/Tk est un langage interprété, créé par John  Ousterout. Pour obtenir plus d'information sur le langage Tcl/Tk, ou télécharger gratuitement la dernière version de Tcl/Tk, vous pouvez consulter la page officielle de la société Scriptics.

Nous développons des extensions Tcl/Tk, dont certaines sont disponibles pour téléchargement, en version complète ou d'évaluation. Nous sommes également spécialisés dans le développement d'extensions et de logiciels multi-plateformes basés sur le langage Tcl/Tk et notre jeux d'extensions. N'hesitez pas à prendre contact avec notre équipe Recherche & Développement, pour nous faire de vos besoins spécifiques.
