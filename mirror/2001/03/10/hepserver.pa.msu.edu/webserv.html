MSUHEP Web Server

Yet another place to show off your artistic talents and create websites!

To use the web services you have to create a subdirectory in the wwwroot folder on the server. You should also set the permissions on the directory that you create. If you need help setting permissions follow this link: setting permissions. In that subdirectory create a HTML document named default.htm (yes, leave off the "L" on the extension). That document then becomes your homepage. to access it through the browser use the address:

http://hepserver.pa.msu.edu/username/

In the above example substitute the name of the subdirectory that you created.
