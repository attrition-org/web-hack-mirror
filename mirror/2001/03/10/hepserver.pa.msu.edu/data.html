Data Storage on the High Energy Physics Server

Each user is given a directory (folder) on the server to store data. Below is an example of the user directory on the server.

By default all data stored in a particular user's directory can only be accessed by that user. That means User1 cannot access the files in User2's directory. Although any user can modify the permissons on his/her directory in order to give others access.

Files kept in the user directories will be backed up to tape each weekday at 11:30PM. Reading or writing to server disks during this time will be slow. It is recommended that only data files be stored on the server. This means files such as Word documents, spreadsheets, PostScript documents and the like. The server is not meant to serve as a backup device for your entire local PC. A plan is being developed that may allow workstations connected to the server to be backed up from the server.
