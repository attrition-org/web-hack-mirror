HEPSERVER FTP Services

Share files with colleagues via anonymous FTP

Domain users can copy files into the "ftproot" directory on the server and have them available to anyone through anonymous FTP. Domain users may also create subdirectories to the "ftproot" directory.

At this time only anonymous logons are allowed. FTP clients logging on to the system must use the username "anonymous" and use their Email address as a password. For now clients will only be able to GET (download) files from the server. Files can not be PUT (uploaded) to the server for security reasons.

To use the FTP services point your browser or FTP utility to: ftp://hepserver.pa.msu.edu

If you experience any difficulties or have any questions Email Mike Nila or phone 5-3523.
