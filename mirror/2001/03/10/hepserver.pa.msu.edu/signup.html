MSUHEP Server Account Request

To get an account on the HEP-SERVER please fill out the form below. When you select send request the information will be Emailed to the Administrator and an account will be established for you as soon as possible. If you experience problems sending the request you may not have your browser set up to send mail. You can send the information to nila@pa.msu.edu. You will be notified by Email once your account has been activated.

Request an account:

Full name
Email address
Requested username*
Requested password**
Confirm password
Computer Type Mac PC

*For Windows 9x and NT users this should be the same user name that you use to log on to your local machine, preferably NOT the administrator user name. This will make connecting to the server seamless. If you choose to use a different user name on the server you will be prompted for your server user name and password when you wish to connect to the server.

**Once your account has been established you will be able to change this password. You might want to supply a "temporary" password on the request and then change it to the password that you use to log on to your local machine to keep from being prompted when connecting to the server.

The server Administrator must make the initial connection from your local machine to the server. Once that has been done all you need to do is log on to your local machine and the connection to the server is automatic, if you use the same user name and password on both machines.
