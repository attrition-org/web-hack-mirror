Using the MSUHEP Server Printers

Three HEP Department Hewlett Packard printers are currently setup on the server:

The LaserJet in room 250
The LaserJet in room 250A
The DeskJet in room 250A

To access these printers double click on Network Neighborhood, then double click Hepserver, then double click Printers. There is no need to have printers set up on your local machine. If files are sent to the printers through the server the user is notified when print job is completed. Print documents through the server in one of two ways:

Drag and drop a document onto the server printer icon in Network Neighborhood

OR

Drag and drop the printer icon from Network Neighborhood onto your desktop.

The first time you print through the server you will get a message saying that the printer must be set up. Click "YES" and you'll never have to do that again.
