Sharing Files on the HEP Server

Share files easily with other users. Simply drag and drop a file into a public directory on the server and it's immediately ready for use by other users.

By default all files stored in the "Public Documents" folder can be read, edited, even deleted by any domain user. You can change this setting by setting permissions on a file placed in this folder.
