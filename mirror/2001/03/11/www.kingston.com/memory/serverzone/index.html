Kingston Server Memory

When you choose Kingston server memory, you get memory that is designed and tested to meet the original system manufacturer's requirements. Kingston's expert engineers and investment in the most advanced testing systems ensures that you get high-quality, 100% compatible server memory products at a fraction of OEM memory prices.

Kingston is licensed by Sun and SGI to manufacture certain proprietary workstation memory products under the Kingston brand name. Additionally, Kingston is a trusted memory source for many system and semiconductor manufacturers.

Kingston produces memory that is 100% tested to be reliable and compatible. For your assurance, all Kingston memory modules are backed by a lifetime warranty and supported by 24-7 free technical support.
