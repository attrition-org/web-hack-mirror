中晨Lotus授权培训中心（简称LAEC），是由美国Lotus公司授权的高级专业技术培训机构，提供始终与Lotus技术发展同步的培训内容,由经过Lotus严格认证的CLI（Lotus认证教师）指导授课，使用Lotus全球统一编写的教材，教学设施优越。经培训并参加CLP（Lotus认证专家）考试合格者，将获得由Lotus总部直接寄发的全球公认的专业证书。
    中晨Lotus授权培训中心针对每个学员的要求，提供最周到完整的教学，一切目的是使您掌握世界上最新、最先进的软件技术，让您由普通技术人员成为高级技术专家。
    中晨公司已成为Sylvan公司的授权考试中心，在学员学完培训课程后，可参加CLS ( Certified Lotus Single ) 或 CLP ( Certified Lotus Professional ) 认证考试。还可参加Lotus的CLE ( Certified Lotus Engineer ) 考试。
