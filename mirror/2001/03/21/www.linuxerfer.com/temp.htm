  Bash 에서의 Prompt
- 변경 방법
   자신의 홈디렉토리의 .bashrc 또는 .bash_profile 에 PS1 설정을 넣어주거나,
   /etc/bashrc 의 PS1 설정을 변경하면 된다.

- 옵션
   \\ 백슬래쉬 출력
   \d 현재의 날짜 출력
   \h 현재 쉘이 실행되고 있는 호스트의 이름 출력
   \n 한 라인을 공백 출력
   \s 실행되고 있는 쉘의 이름 출력
   \t 시간 출력
   \u 현재 사용자의 이름 출력
   \W 현재 작업 디렉토리의 베이스 이름 출력
      (예 : /usr/lib/netscape 일때 netscape 만을 출력)
   \w 현재 작업 디렉토리의 풀패스 출력

- Default
   PS1="[\u@\h \W]\\$"

  xhost 사용법에 대해...
   xhost는 자신의 X server의 접근 권한을 설정하는 프로그램이다.
    localhost : 123.123.123.2
    remote host : 123.123.123.3 이라면,

    먼저 localhost 에서 X windows 를 실행한 후
    localhost$ xhost +123.123.123.3
    localhost$ telnet 123.123.123.3

    remote host$ export DISPLAY=123.123.123.2:0.0 (bash 를 사용할 경우)
    remote host$ setenv DISPLAY=123.123.123.2:0.0 (csh 를 사용할 경우)
    remote host$ I_X_Client_program
