포토샵 강좌

file 메뉴

 

Transparency & Gamut

포토샵의 투명도에 사용되는 일정한 패턴을 지정한다.

Grid Size : 패턴의 사이즈를 지정

Guid Colors : 패턴의 색을 지정한다.

Gamut Warning : Gamut Warning에서 사용하는 색을 지정한다.  Color를 선택하면 포토샵에서 쓰는 컬러 팔레트가 나온다. 여기에서 색을 지정해 주면 된다.

 

Units & Rulers

포토샵에서 사용하는 단위를 지정한다. 값을 지정하면, 포토샵에서 사용하는 모든 대화 박스에 적용된다.

Rulers : 포토샵에서 사용하는 모든 단위를 지정한다.

Column Size : 가로와 컬럼(Gutter)의 값을 지정한다.

Point/Pica Size : 인치당 인쇄될 점의 수를 지정한다.

 

Guides & Grid

가이드 선의 색상을 변경시킬 수 있으며, 스타일에서는 Dashed Line등 두가지 형태로 변경시킬 수 있다.(나중에 다 배우고나면 아실 거예요)

 

Plug-Ins & Scratch Disks

플러그인 프로그램을 포한하고 있는 폴터를 지정한다.

 

Memory & Image Cache

포토샵에서 사용할 수 있는 메모리를 조정한다.(고급사용자가 아니면 손대지 말 것을 권하다.)

Cache Settings : 포토샵에서 사용할 수 있는 메로리를 레벨로 지정한다.

레벨이 높을수록 포토샵에서 사용할 수 있는 가상 메모리는 많아지지만, 다른 응용프로그램 실행시 메모리 부족 현상이 나타날 수도 있다.

가상메모리를 사용해서 메모리 문제를 해결할 수 있지만, 하드디스크 메모리를 사용하기 때문에 램에서 사용하는 속도보다 느리다.

이런한 속도 문제를 해결하기 위해서 램 드라이브에 해당하는 디스크를 Primary로 사용하는 것이 효과적이다.

Physical Memory Usage : 포토샵에서 사용 가능한 메로리의 전체 용량을 백분율로 나타내며, 메로리의 효율성을 나타낸다.

 

Color Settings

모니터와 출력기의 출력 환경을 설정할 수 있다.

 

RGB Setup

모니터에 나타나는 색상은 출력 색상과 차이가 있다. 이러한 모니터의 문제점을 해결하기 위한 모니터 환경설정 명령이다.

 

CMYK Setup

4도 분판 출력을 위해 색분해에 관한 설정을 하는 옵션이다.

 Edit 메뉴

Edit 메뉴에서는 포토샵에서 가장 많이 사용되는 이미지 복사, 제거, 삽입, 패턴 설정등의 모든 편집 작업을 할 수 있다.

 

Undo/Redo

Undo : 포토샵에서 이미지 작업시 실수할 때가 있는데, 그럴 때 Undo를 실행시켜 바로 취소해 버리면 된다. 그러나 마지막으로 작업한 결과만 취소할 수 있다는 단점이 있다..

Redo : Undo적용을 다시 원상태로 돌려 보낸다.

 

Cut

선택둘로 선택된 이미지의 한 부분을 잘라내어 임시 기억장소에 보관하는 명령이다. 잘려진 부분은 배경색으로  채워진다.

완전히 삭제되는 것이 아니고, Paste명령으로 새로운 위치에 붙여넣을 수 있다.

copy

선택툴로 선택한 이미지 부분을 복사해두는 명령이다. Cut과 다른 점이 있다면, 도큐멘트 상에서 잘라내는 것이 아니라 복사만 한다.

Copy Merged

화면상에 나타나는 모든 영역을 복사할 수 있다. 이미지 복사하는 것은 Copy와 같지만 Copy가 활성화된 부분만 복사하는 반면 Copy Merged는 레이어에 상관없이 현재 보이는 부분 모두 복사 한다.

 

Paste

Cut 또는 Copy한 이미지를 현재 작업중인 이미지위로 덧붙이는 기능이다.

 

Paste Into

Cut나 Copy로 복사된 이미지를 윈도우의 영역 안쪽에 붙인다.

복사한 이미지

Paste를 적용한 결과

원형태의 영역을 선택해서 Paste Into를 적용한 결과

 

Clear

이미지의 선택한 영역을 배경색으로 채운다. Cut과 같은 시각적인 효과를 얻을 수 있지만, 클립보드에서 내용이 완전히 삭제 되고 배경색으로 채워 진다.
