포토샵 강좌

툴 설명3

 

Pan Tool(펜 툴)

일러스트와 비슷한 기능을 가지고 있다 하지만,

포토샵을 그림을 전문적으로 그리는 툴이 아니라서 많이 축소된 것이다.

펜툴의 자세한 사용법은 일러스트 강좌를 보고 공부하기 바란다.

 

 

Magnetic Tool

포토샵 5.0에서 새로 추가된 툴이다.

마우스가 드래그 하는 방향으로 꺽이는 각도에 따라 점이 결정된다.

예전에는 펜 툴 하나만 있어서 사용하기 불 편한 점이 많았다.

 

 

 

Freeform Pan Toll

Pancil Tool 과 거의 같은 기능이다 단지 다른 점이 있다면 펜툴의 특성인 점이 생긴다는 점이다.

드래그 하면 선이 부드럽게 만들어 진다는 특징이 있다.

 

 

Add Anchor Piont Tool(점 추가 툴)

펜으로 그린 그림의 클릭점을 추가하는 기능이다.

그림의 보기와 같이 빨간 원에 클릭하면 아래 그림 같이 점이 생겨서

모양을 변화 시킬 수 있다.

 

 

Delete Anchor Point Tool(점 삭제 툴)

점 추가 툴과 반대로 현재의 있는 점을 없애 준다.

점 삭제 툴을 선택한 다음 그림의 보는 바와 같이 한쪽을 점을

클릭하면 점이 사라 지면서 아래와 같은 그림이 된다.

 

 

Direct Selection Tool

펜으로 그려진 그림의 점을 단수나 또는 복수로 선택 할 수 있는 툴

 

 

Convert Point Tool

펜으로 그려진 그림의 점에서 방향들을 뽑아 내주는 기능

그림과 같이 점에서 드래그 하면 방향 틀이 나오게 된다.

나온 방향틀을 선택을 해서 드래그 해주면 곡선이 원하는 데로

변한다.

 

 

Type Tool(문자 툴)

도큐멘트 상에 문자를 넣은 주는 툴이다.

문자툴을 선택한다음 도큐멘트 위에

클릭하면 그림과 같이 대화 상자가 나온다.

Font : 문자의 타입을 결정한다.

Size : 문자의 크기를 결정한다.

Leading : 글꼴의 행 간격(문장과 문장 사이의 간격)

             을 지정할 수 있다.

Color : 쓰여질 문자의 색을 설정할 수 있다.

 

 

Type Mask Tool

일반 타입 툴과 똑같은 기능이다. 다른 점이 있다면,

그리과 같이 선택된 상태로 문자가 그려진다는 것이다.

마스크 툴을 문자 안에 그림이나 색을 마음대로 넣을 수 있다.

Type Tool적용

Type Mask Tool적용

 

 

Vertical Type Tool

타입 툴과 사용법은 같다 하지만 글자가 위에서 아래로

써진다는 점이 다른 점이다.

포토샵 4.0에서는 글을 횡으로 쓸 경우 무척이나 많은 작업을

해야 했지만, 5.0에서는 그런 복잡한 일이 없어졌다.

 

 

Vertical Type Mask Tool

문자를 위에서 아래로 써주며, 마스크 기능이 들어간다.

 

 

Measure Tool

도큐멘트 오브젝트의 크기와 각도 등을 체크한다.

먼저 클릭한다음 드래그하면 위의 그림과 같이

선이 나오고 그 선의 길이 만큼의 수치가 대화상자에 표시된다.

(각도도 같이 표시 된다.)

드래그 한 상태

대화 상자

그라데이션 툴(Gradient Tool)

그라데이션 툴은 색을 점차적으로 부드러운 효과를 주는 페인팅 툴이다. 이미지의 일부 영역에만 효과를 주려면 선택 출을 이용하여 미리 선택해 둔다.

기본적으로 지원되는 그라데이션의 종류는 16가지 지만, 사용자의 지정 등록에 따라 만들 수 있기 때문에 실질적으로 무한정하다고 할 수 있다.

그라데이션의 색을 사용자의 정의로 지정하는 방법 

그라디언트 옵셥 중에서 먼저 Edit를 클릭해 준다.

그러면 아래그림의 대화 상자가 나온다.

그라디언트 대화상자

1번은 기본적으로 제공해 주는 그라디언트의 종류이다.

Color : 일반적인 컬러가 들어간다.

Transparency : 투명한 이미지를 원할 때 사용하다.

                       (%에 따라 투명도가 다르다)

2 : 시작 포인트이다 여기를 클릭하면 색을 지정할 수 있는

    대화 상자가 나온다. 거기에서 색을 지정해 주면 된다.

3 : 마지막 포인트이다 시작 포인트와 기능은 같으며

    단지 그라디언트의 끝 색깔을 지정한다.

4 : 전면색을 적용한다.

5 : 배경색을 적용한다.

6 : 색을 적용시키기 위해서 팔레트를 부른다.

 

 

여러 색으로 그라디언트를 적용시킬 때

오른쪽의 그림의 그림과 같이 중간 부분을 클릭해 주면

아래그림과 같이 중간에 선택점이 생긴다.

그러면 여러 가지 의 색으로 그라디언트를 줄 수 있다.

 

 

Linear Gradient Tool

가장 기본적인 그라디언트이다.

드래그 하는 순으로부터 번져 나간다.

 

 

Radial Gradient Tool

원을 그리며 안에서부터 번져 나가는 형식이다

 

 

Angle Gradient Tool

그라디언트를 원의 형식으로 둘러 주는 형식이다.

 

 

Raglected Gradient Tool

드래그 해서 그라디언트를 주면 중간부터 외각으로

색이 번진다.

 

 

Diamond Gradient Tool

선택된 오브젝트 안에서부터 4방향으로 그라디언트가

퍼져 나간다.

 

 

Paint Bucket Tool

페인트 버킷 툴은 배경색이나 전경색에 지정되어 있는 색을 선택된 영역에 채워주는 툴이다.

 

 

Eyedropper Tool(스포이트 툴)

지정한 픽셀의 고유색상을 전면색으로 옮기는

역할을 한다.

Point Sample : 클릭한 지점의 픽셀의 색으로 전경색으로

                      옮긴다.

3 by 3 Average : 선택한 픽셀을 중심으로 사방 3×3픽셀의

                         평균값으로 전경색으로 옮긴다.

5 by 5 Average : 선택한 픽셀을 중심으로 사방 5×5픽셀의

                         평균값으로 전경색으로 옮긴다.

 

 

Color Sempler Tool

지정한 곳의 색상 값을 알 수 있다.

오른쪽의 그림같이 최대 4개 까지 가능하며, 각의 색상

값이 Info에 나온다.(Alt를 누르고 클릭하면 선택된 곳이

사라진다.)

 

 

Hend Tool

모니터 화면이 너무 작거나 이미지가 너무 커서 원하는 부분을 볼 수가 없을 때

드래그 하면 도큐멘트의 전체가 원하는 곳으로 딸려 온다.

 

 

Zoom Tool

화면을 확대 하거나 줄일 때 쓴다.

 

 

색상 아이콘

위로 나온 보라색이 전경색이고, 뒤에 숨어 있는 녹색이  배경색이다.

전경색(Foreground Color)

연필, 붓 에어브러시 등으로 이비지를 그릴 경우에 사용되는 현재 사용중인 색상이다.

배경색(Background Color)

Clear 또는 지우개로 이미지를 지우는 경우에  사용되는 색상이다.

빨간색으로 테두리 쳐진 것은 색 초기화 버튼이다.

 

 

기본 모드와 퀵 마스크 모드

현재의 토큐멘트 창의 이미지에 마스크된 구역을 제외한 이미지에 지정된 색상으로 채우거나 연필 도구, 붓 도구, 에어브러시 도구 등으로 색을 칠하게 되면 마스크된 구역은 색이 칠해지지 않으며, 마스크가 안된 구역은 색이 칠해지게 된다.

기본 모드와 퀵 마스크 모드를 더블클릭하면 다음과 같은 마스크 옵션 대화 박스가 나타난다.

Masked Areas : 마스크 구역에 지정한 색상이 투명하게 나타난다.

Selected Areas : 선택한 구역에 지정한 색상이 투명하게 나타난다.

Color : 사각형은 마스크에 사용될 색상이며, 이 부분을 더블클릭하면 컬러 픽커 대화 박스가 나타난다.
