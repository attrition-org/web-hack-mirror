메뉴 명령

File 메뉴

작업에 꼭 필요한 기본적인 기능들이 모여 있다. 그림을 그리기 위해 새로운 도큐멘트를 열고, 작업 내용을 저장하고, 자신만의 환경 설정을 하여 남들과 차별된 여건에서 독특하게 작업할 수 있다.

 

New

새로운 이미지를 만들 경우 그려넣을 새로운 도규멘트 창이 필요한데, 이때 사용하는 명령어다.

Name : 새로운 도큐멘트창의 이름

Image Size :

 Width : 도큐멘트의 폭의 설정 값

 Height : 도큐멘트의 높이 설정 값

 Resoltion : 해상도의 값을 설정

 Mode : 포토샵에서는 여러 가지 형식의 Color를 지원하는데. 여러분들이 RGB, CMYK등은 많이 들어 보았을 것이다. Mode에서는 그러한 것들을 지정한다.

Contents

 White : 도큐멘트가 열릴 때 하얀색으로 열린다.

Background Color : 현재의 색상 아이콘에서 배경색으로 도큐멘트를 연다.

Transparent : 새로운 도큐멘트를 어떠한 색도 넣지 않고 연다.

 

Open

이미 작업한 도큐멘트를 연다.

위치 : 자신의 하드 디스크의 위치

파일 이름 : 자신이 찾고자 하는 파일의 이름

 

파일 형식

Photoshop(*.PDS,*PDD) : 포토샵 고유의 포맷 방식이다. 디스크에 쓰기/읽기 속도가 빠른 반면 저장 용량이 다른 포맷 방식 보다 크다

BMP(*.BMP,*.RLE) : 윈도우즈에서 제공 되는 비트맵 이미지 포맷 방식이다.

Photoshop EPS(*.EPS) : 이 포맷 방식은 비트맵 이미지 방식이 아니고 어도비 일어스트레이터 또는 코렐드로우 등과 같은 아웃라인 프로그램에서 제작한 벡터 포맷 방식이다.

EPS TIFF Preview(*.EPS) : 일러스트레이션과 페이지 레이아웃 프로그램에서 지원하는 포맷 방식이다.

JPEG(*.JPG,*.JPE) : 비트맵 방식의 컬러 이미지 압축시 압축률이 가장 우수한 포맷 방식이다.

PCX(*.PCX) : 페인트 브러시와 디럭스 페인트에서 일반적으로 사용하는 파일 포맷 방식이다.

PDF(*.PDF) : 어도비 에크로벳에서 지원하는 파일 포맷 방식이다.

PICT File(*.PCT,*.PIC) : 매킨토시의 그래픽 프로그램에서 일반적으로 사용하는 파일 포맷 방식이다.

Raw(*.RAW) : 응용 프로그램과 시스템 플랫폼의 문서를 교환하는 과정에서 지원하는 파일 포맷 방식이다.

 

Open As

디스크에 저장된 이미지 파일을 불러온다. Open명령으로 잘 불러 들이지 못하는 매킨토시에서 만든 파일인 TIff 이미지 파일을 불러온다. 또한 Open As에서는 포맷 방식이 다른 이미지 파일을 불러올 수 있다.

 

Close

현재 작업중인 도큐멘트 창을 닫는다.

 

Save

현재 작업중인 도큐멘트 창의 이미지를 디스크에 저장한다.

 

Save As

현재 작업중인 도규멘트 창에 새로운 이름을 지정하여 저장한다.

 

Save A Copy

현재 작업중인 도큐멘트 창의 이미지 복사본을 만든다.

 

Revert

현재 작업중인 도큐멘트 창의 이미지를 지우고 최근에 저장된 상태를 불러온다.

 

Import

포토샵에서 직접 스켄이나 디지털 카메라등을 제어할 수 있고 플러그인 파일을 추가 시킬 수 있다.

 Quick Edit : 도큐멘트 창에 큰 용량의 이미지를 불러올 경우 이미지의 크기다 커서 처리 하지 못할 경우가 있다. 이때 Quick Edit명령어를 사용하면, 불러올 이미지에서 일정한 영역을 선택할 수 있으며 TIFF와 ScitexCT파일 형식만 지원한다.

Select TWAIN Source : 스캐너 공급업체에서 제공하는 스캐너 드라이버를 설치하면 대화 박스의 메뉴에서 드라이버가 추가 되며, 사용할 드라이버를 선택할 수 있다.

TWAIN : 스캐너가 선택된 경우 이 명령어를 실행시켜 스캐너를 실행 시킨다.

 

Export

현재 작업중인 도큐멘트 창의 이미지를 다른 프로그램에서 사용하는 포맷으로 파일을 저장할 수 있다.

GIF89a Export : 현재 작업중인 도큐멘트창의 이미지를 GIF파일 형식으로 저장한다.

Paths to Illustrator : 펜 경로 박스에서 작업한 경로를 코렐드로우나 일러스트에서 사용할 수 있는 EPS파일로 저장한다.

Quick Edit Save : Import의 Quick Edit 명령으로 작성된 파일을 저장 시키는 명령이다.

 

File Info

작업한 파일의 다양한 정보(캡션 추가, 제목 지정 등)를 나타낸다.

 

Page Setup

현재 작업된 도큐멘트 창의 이미지를 출력하기 위해 종이의 크기, 방향 등을 설정하는 명령이다.

 

Preterences

포토샵에서 기본적인 환경을 지정할 수 있다. 포토샵에서 파일 저장 방식, 디스플레이 상태, 커서 모양, 투명 상태, 감마값, 눈금 단위 ,그리드 단위, 플러그인, 메로리등을 조정할 수 있는 설정중 가장 중요한 환경 메뉴이다.

 

Color Picker

포토샵에서 제공하는 팔레트 기능으로 24비트를 지원하지 못하는 그래픽 카드인 경우에도 간접적으로 24비트 컬러로 나타낼 수 있는 컬러 테이블을 제배치 해준다. 보통 Photoshop으로 지정하고 쓴다.(지원되지 않는 그래픽 카드인 경우에 Windows를 선택)

Interpolation

이미지를 비틀거나 늘리거나 크기 조절 및 투시, 왜곡, 회전시키는 등의 작업을 할 경우 이러한 효과를 인터폴레이션에 설정된 값으로 묶어서 이미지의 상태를 최적화시킬 수 있으며 실행 속도에 영향을 미친다.

Nearest Neighbor

속도는 빠르지만 이미지의 질이 떨어진다. 이미지 크기 조절시에 사용하면 유용하다.

Bilinear

중간 정도의 이미지 질과 속도를 나타낸다.

Bicubic(Better)

이미지 질이 가장 우수 하지만 속도는 떨어진다.

 Options(옵션)

Anti-alias PostScript

포스트 스크립트에 안티알리아싱 효과를 주는 옵션이다.

Export Clipboard

포토샵이 종료될 경우 클립보드에 있는 이미지도 함께 저장 된다.

Short PANTONE Names

포토샵의 펜톤 컬러를 다른 응용 프로그램에서 사용할 수 있게 한다.

Show Tool Tips

도구 박스 또는 팔레트에 마우스를 위치시키면 각 목록에 대한 사용 설명과 단축키 이름등을 나타내는 옵션이다.

Beep When Done

이미지에 옵션 적용이 끝날 경우 경고음을 울리게 한다.

Dynamic Color Sliders

컬러 팔레트 또는 슬라이더를 제공하는 컬러 조정 도구에서 슬라이드를 드래그하게 되면 즉시 프리뷰 할 수 있게 한다. 단, 프리뷰를 하게 되면 속도가 약간 느려진다.

Save Palette Locations

현재 상태를 저장할 결우 Preferences파일에 포토샵의 환경 상태를 기억시킨다.

 

 

Saving Files

이미지 프리뷰 기능을 Save As메뉴를 실행할 때마다 자동으로 지정한다.

 

Display & Cursors

오브젝트상에 작업을 할 때 포인트나 페인팅할 경우 마우스에 따라서 움직이는 그림을 변화 시키는 기능이다.
