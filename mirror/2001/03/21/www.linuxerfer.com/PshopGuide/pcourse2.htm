포토샵 강좌 2

선택 도구2

Airbrush Tool

이미지 리터칭 효과를 주는 툴이다.

오른쪽 그림은 브러쉬 툴을 사용하는 옵션들이다.

Pressure : 그려지는 브러쉬의 넓이를 설정하는 옵셥이다.

Fade : 입력하는 숫자만큼 사라지게 하는 효과를 준다.

Fade to : Fade를 어디자 줄 것인지를 설정하는 옵션이다.

Stylus Pressure : 압력을 감지하는 옵션이다.(전문가가 아닌이상 별 소용이 없는 것)

옵션아래 Brushes로 넓이와 명도등을 조절할 수 있다.

 

Paintbrush Tool

브러쉬툴과 비슷한 기능이나 미세하게 차이가 난다.

Opacity : 투명도를 결정하는 옵션이다.

Fade : 입력하는 숫자만큼 사라지게 하는 효과를 준다.

Wet Edges : 마치 물을 묻혀 페인팅 한 것과 같이 불투명도를 100%로

하여도 약가의 투명도를 가지며 붓의 테두리는 진하고 중간 부분은 엷게

나타납니다.

 

 

Pan Tool

펜으로 그림을 그리는 것같이 표현할 수 있다.

옵션은 브러쉬 툴과 별다른 것은 없다.(비슷한 기능이다.)

 

 

포토샵 5.0에서 새로생긴 툴로 선택되어있는 부분을 원본 상태로 살려주는

브러쉬 툴이다. 투명도도 같이 옵션으로 설정해 줄 수 있다.

어떻게 보면 예술적인면을 나타낼 수 있는 툴이다.

한번 직접 해보면 이해가 쉬울 것이다.

 

 

Rubber Stamp Tool

보통 도장 툴이라고 부른다

스템프 툴을 선택한 다음 복사하고자 하는 지점을(Alt를 누르고)찍고

도장모양의 그림을 복사해 놓을 곳을지정해서 지정을 하면서 드래그 하면 처음

찍은 장소의 그림이 현재 도장모양이 있는 곳에 그려진다.

 

 

Eraser Tool(지우개 툴)

말 그대로 지우개로 지워주는 툴이다.

지우는 기능 말고도 여러 가지 효과가 있는데, 페인팅도 가능하다.

포토샵 5.0에서 메뉴가 무척이나 다양해졌다.

빨간 줄이 그어진 옵션은 지워지는 형태를 나타내는 것이다.

Opacity : 투명도를 나타낸다.(눈치가 빠른 사람은 알겠지만 옵션에서 Opacity가

나오면 무조건 투명도 조절을 할 수 있는거죠.)

Erase to History : 현재 작업중인 그림이 저장 되어있다면, 다시 살려주는

기능을 한다.하지만, 저장되어 있지 않다면 Erase to History가 비활성화 된다.

 

 

 

Blur Tool(블러 툴)

초점을 흐리게 하는데 사용한다.

옵션에서 Pressure는 블러 효과의 강도를 조절해 준다.

 

Sharpen Tool(샤픈 툴)

이미지를 날카 롭게 해주는 툴이다.

이미지가 너무 흐린 경우 샤픈 도구를 이용하여 경계선과 같은

부분을 선명하게 나타내준다.

원 본

샤픈 적용

 

Smudge Tool(손가락 툴)

 아직 마르지 않은 그림을 손가락으로 문지른 것 처럼 번지는 효과를 준다.

그림을 보면 금방 이해가 갈 것이다.

그림 합성을 했을 때, 경계선을 부드럽게 해주는 것도 손가락 툴의 한가지 이용법이다.

 

 

Smudge Tool의 옵션에 보면 Finger Painting에 체크해 주면

Foreground Color로 문지른 효과를 준다.

 

 

Pancil(연필도구)

브러시 팔레트에서 선택한 굵기로 자유로운 선을 그릴 때 사용하는 도구이다.

포토샵에서 유일하게 자오 안티알리아싱 처리가 되지 않는 드로잉도구이다.

 

 

Line(선 도구)

도규멘트상에 선을 그려 넣을 수 있으며, 양쪽 또는 한쪽 방향에

화살표를 넣을 수 있다.

사용방법은 마우스를 클릭하고 드래그해서 원하는 점까지 그려주면 된다.

선툴을 잘 사용하면 편집이나 그림을 합성할 때 유용하게 쓰인다.

선툴을 자세히 설명하기 위해서 아래 선 도구 옵션을 보며, 설명하겠다.

 

 

Weight : 선의 굵기를 지정하는 옵션이다.

Arrowheads

  Start : 이 체크 박스에 체크를 하면 아래 그림과 같이

            드래그 시작점부터 화살표가 생긴다.

           

 End : 이 체크 박스에 체크하면 아래 그림과 같이

          드래그가 끝나는 부분에 화살표가 생긴다.

            

※ Start나 End 둘중에 아무것도 체크가 없으면 그냥 선만 나오게된다.

옵션 대화상자의 Shape는 아래 그림과 같이 설명하겠습니다.

 

 

Width : 화살표의 높이를 결정, 아래의 그림은 Width를 800%로 값을 주었다.

Length : 화살표의 넓이를 결정, 아래의 그림은 Length를 300%로 값을 주었다.

Concavlty : 화살표의 모양이 변한다.(그림을 보면 필이 팍!!)

(-50 ~ 50까지 값을 지정할 수 있다.)
