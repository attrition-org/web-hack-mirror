Affichage
Ce site est optimisé pour un affichage 800 pixels x 600 pixels
Pour régler l'affichage de votre ordinateur voici la démarche à suivre :

Windows 95/98/NT et plus
- Cliquer sur le bouton "Démarrer" puis dans le menu "Paramètres"
- Sélectionner "Panneau de configuration"
- Double-cliquer sur "Affichage"
- Une boite de dialogue apparaît : cliquer sur l'onglet "Configuration"
- Régler le curseur "Espace du bureau" sur 800x600 ou plus
- Confirmer en appuyant sur le bouton OK
- Suivant votre carte graphique, il vous faudra redémarrer Windows;
- Une boite de dialogue apparaîtra alors pour vous le proposer.

Mac OS
- Accéder aux "Tableaux de Bord" par le menu "Pomme" et ouvrir "Moniteur" ou "Moniteur et son"
- Dans la fenêtre qui apparaît cliquer sur "Option" ou "Moniteur" et sélectionner la résolution 800x600 ou plus
- Confirmer en cliquant sur "OK" dans les boites de dialogues qui apparaissent.

Navigateur (version conseillée) Pour utiliser pleinement ce site, vous devez utiliser au minimum Microsoft Internet Explorer 4.0 et Netscape Navigator 4.0.
